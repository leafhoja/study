Motivation

なぜ抽象的なベクトル空間を考えるか

Motivation
  1. 接空間の抽象化：曲面の接面と $\mathbb{R}^3$ を近似的に同一視したい。 座標を入れるためには原点・基本単位ベクトルが必要だが、これらは恣意的であり、どう選んでも同じ幾何的内容が得られるようにしたい。 結論：「どう定式化すればよいか？」→ 以下のように定める。
    • 接空間とは、曲面上の点 $p$ でその曲面に「接する」平面のこと。局所的には $\mathbb{R}^2$ に見えるが、$p$ をどの「原点」とし、どの「方向」を基本とするかは自由に選べる。どの選び方でも「同じ幾何的構造」が得られることを保証するために抽象的なベクトル空間を使う。
  2. より一般の対象を扱いたい：$\mathbb{R}^n$ よりも広い対象（関数空間など）に対しても線形代数の枠組みを使いたい。
    • 例：微分方程式 $y''+y=0$ の解の全体 $\{\sin x, \cos x, \ldots\}$ は関数の空間だが、「足す」「スカラー倍する」という操作が自然に定まり、ベクトル空間の公理を満たす。$\mathbb{R}^n$ とは全く異なる対象だが、同じ理論が使える。
  3. 線形代数の主役は線形写像：線形代数における主役はベクトルそのものではなく線形写像である。 線形写像が定義されるために必要なものは加法とスカラー倍の 2 つだけであった。
    • 「ベクトル空間の公理」の最小限の要件がまさに「線形写像を定義するために必要な構造」であり、これが公理の選び方の根拠になっている。
補足 — 「抽象化」の威力

$\mathbb{R}^n$ の場合に成り立つことを証明するのに「成分」を使うと、$n$ ごとに別々の証明が必要になる。 抽象的なベクトル空間の公理だけから証明すれば、$\mathbb{R}^n$ でも $\mathbb{C}^n$ でも関数空間でも、同時に成立する。 これが抽象化の最大の恩恵である。

Setup

基本設定・記法

設定 以降は実数だけでなく複素数も考える。
  • $\mathbb{C}^n = \left\{\left.\begin{pmatrix}a_1\\\vdots\\a_n\end{pmatrix}\right| a_1,\ldots,a_n\in\mathbb{C}\right\}$
  • $M_{m,n}(\mathbb{C}) = \left\{\left.\begin{pmatrix}a_{11}&\cdots&a_{1n}\\\vdots&&\vdots\\a_{m1}&\cdots&a_{mn}\end{pmatrix}\right| a_{ij}\in\mathbb{C},\ 1\le i\le m,\ 1\le j\le n\right\}$（$m\times n$ 複素行列全体）
  • $\mathbb{K}$ は $\mathbb{R}$ か $\mathbb{C}$ のいずれかとする。$\mathbb{K}^n$, $M_{m,n}(\mathbb{K})$ も同様。
  • $\mathbb{K}$ 上のベクトル空間を $\mathbb{K}$-ベクトル空間 という。$\mathbb{R}$-ベクトル空間 ＝ 実ベクトル空間、$\mathbb{C}$-ベクトル空間 ＝ 複素ベクトル空間。

Definition 1.1

ベクトル空間（vector space）

定義 1.1 — $\mathbb{K}$-ベクトル空間 $\mathbb{K} = \mathbb{R}$ または $\mathbb{C}$。 組 $(V, +, 0, \cdot)$ が $\mathbb{K}$-ベクトル空間（vector space）であるとは、以下の (V1)〜(V11) をすべて満たすことをいう：
  1. (V1) $V$ は集合である
  2. (V2) $0$ は $V$ の元である（零元、ゼロベクトル）
  3. (V3) $+$ および $\cdot$ はそれぞれ以下の写像である：
    • $+: V\times V\to V,\quad (x,y)\mapsto x+y$ （$V$ の加法）
    • $\cdot: \mathbb{K}\times V\to V,\quad (a,x)\mapsto ax$ （スカラー倍）
  4. (V4)（可換律）任意の $x,y\in V$ に対し $x+y = y+x$
  5. (V5)（結合律）任意の $x,y,z\in V$ に対し $(x+y)+z = x+(y+z)$
  6. (V6)（零元の性質）任意の $x\in V$ に対し $x+0 = 0+x = x$
  7. (V7)（逆元の存在）任意の $x\in V$ に対し、ある $y\in V$ が存在して $x+y = y+x = 0$
  8. (V8)（スカラー倍の分配・ベクトル側）任意の $a\in\mathbb{K},\ x,y\in V$ に対し $a(x+y)=ax+ay$
  9. (V9)（スカラー倍の分配・スカラー側）任意の $a,b\in\mathbb{K},\ x\in V$ に対し $(a+b)x=ax+bx$
  10. (V10)（スカラー倍の結合性）任意の $a,b\in\mathbb{K},\ x\in V$ に対し $(ab)x=a(bx)$
  11. (V11)（単位元）任意の $x\in V$ に対し $1\cdot x=x$
補足 — 11 個の条件の意味
グループ公理意味
構造の定義 (V1)–(V3) 「何が与えられているか」を宣言。$V$ は集合、$0\in V$、$+$ と $\cdot$ という 2 種類の演算が存在する。
加法群の公理 (V4)–(V7) $(V, +, 0)$ が可換群（アーベル群）をなすことを保証。加法に関して「合理的な計算」ができる。
スカラー倍の公理 (V8)–(V11) スカラー倍と加法が「矛盾なく共存」できることを保証。(V8)(V9) は分配法則、(V10) は結合性、(V11) は $1$ 倍恒等性。
  • (V3) の $\cdot$ はスカラー倍：$\mathbb{K}$ の元（スカラー）を $V$ の元に掛ける演算。ベクトル同士の「内積」とは全く別物。
  • なぜ $\mathbb{K}=\mathbb{R}$ か $\mathbb{C}$ か：より一般の「体（field）」上でも同じ理論が展開できるが、この講義では実数・複素数に限る。
  • $(V,+,0)$ はアーベル群：群とは「結合律・単位元・逆元」を持つ代数的構造。さらに可換律が加わったものをアーベル群という。

各公理が「なぜ必要か」をもう少し掘り下げる

  • (V4) 可換律 $x+y=y+x$： この条件がないと、$x+y$ と $y+x$ は原理上「別物」になりうる（実際、行列の積は $AB\ne BA$ が一般的）。 ベクトルの加法には特別に「順序によらない」性質を課すことで、自由に式変形できるようにする。
  • (V5) 結合律 $(x+y)+z=x+(y+z)$： 「カッコの位置によらず結果が同じ」という保証。 これがあって初めて $x_1+x_2+\cdots+x_n$ というカッコなしの和が意味を持つ（定例 1.3 参照）。
  • (V6) 零元 $x+0=x$： 「何も足さない」ことを表す特別なベクトル $0$ の存在を宣言する。 ここで言う $0$ はスカラーの $0$（$\mathbb{K}$ の零元）とは別物で、$V$ の元（ゼロベクトル）であることに注意。 両者が同じ記号を使うのは慣習で、文脈から区別する。
  • (V7) 逆元 $x+(-x)=0$： 各ベクトル $x$ に対して、足すとゼロベクトルになる元 $-x$ の存在を保証する。 これがあって初めて「$x$ を右辺に移項して $-x$ を左辺に持ってくる」操作が正当化され、 引き算 $x-y \overset{\mathrm{def}}{=} x+(-y)$ も定義できる。 ただし (V7) は存在のみを言い、一意性は系 1.2(1) で証明して初めて「$-x$」と一意に書ける。
  • (V8) 分配法則（ベクトル側）$a(x+y)=ax+ay$： 「先に足してからスカラー倍」＝「別々にスカラー倍して足す」という自明に見えて必要な性質。 たとえば $2(x+y) = 2x+2y$ は中学数学で自明に使うが、 抽象的なベクトル空間でそれを言うにはこの公理が必要。
  • (V9) 分配法則（スカラー側）$(a+b)x=ax+bx$： 「スカラーを先に足してからベクトルに掛ける」＝「別々に掛けて足す」。 (V8) とセットで「スカラー倍と加法が矛盾なく共存する」ことを保証する。
  • (V10) 結合性 $(ab)x=a(bx)$： 「$x$ を $b$ 倍してから $a$ 倍する」のと「$x$ を $ab$ 倍する」のが同じ結果になる。 これがないと $\mathbb{K}$ の乗法と $V$ のスカラー倍の「連鎖」が整合しなくなる。
  • (V11) 単位元 $1\cdot x=x$： $\mathbb{K}$ の乗法単位元 $1$ でスカラー倍しても変わらない。 スカラー倍の演算が $\mathbb{K}$ の乗法と「食い違わない」最低限の要請。

ベクトル空間の例

例 1.1 — 標準的なベクトル空間
  • $\mathbb{C}^n$：$\mathbb{C}$-ベクトル空間（複素 $n$ 次元縦ベクトル空間）
  • $\mathbb{R}^n$：$\mathbb{R}$-ベクトル空間（実 $n$ 次元縦ベクトル空間）
  • $M_{m,n}(\mathbb{K})$：$\mathbb{K}$-ベクトル空間（$m\times n$ 行列空間）
  • $M_{1,n}(\mathbb{C})$：$n$ 次元複素横ベクトル空間

関数空間（$-\infty < a < b \le +\infty$）：

  • $C^\infty((a,b)) = \{\varphi:(a,b)\to\mathbb{C} \mid \varphi \text{ は } C^\infty \text{ 級（何回でも微分可能）}\}$
  • $C((a,b)) = \{\varphi:(a,b)\to\mathbb{C} \mid \varphi \text{ は連続}\}$

いずれも $(f+g)(x) \overset{\text{def}}{=} f(x)+g(x)$、$(af)(x) \overset{\text{def}}{=} a\cdot f(x)$ により $\mathbb{C}$-ベクトル空間となる。

注：$\{0\} = \{$零元のみからなる集合$\}$ も $\mathbb{K}$-ベクトル空間（零ベクトル空間）。最小のベクトル空間である。

具体的な確認：$\mathbb{R}^2$ はベクトル空間か

例 — $\mathbb{R}^2$ が $\mathbb{R}$-ベクトル空間であることの確認

$V = \mathbb{R}^2 = \left\{\begin{pmatrix}x\\y\end{pmatrix}\middle| x,y\in\mathbb{R}\right\}$, $0 = \begin{pmatrix}0\\0\end{pmatrix}$, $\begin{pmatrix}x_1\\y_1\end{pmatrix}+\begin{pmatrix}x_2\\y_2\end{pmatrix} = \begin{pmatrix}x_1+x_2\\y_1+y_2\end{pmatrix}$, $a\begin{pmatrix}x\\y\end{pmatrix} = \begin{pmatrix}ax\\ay\end{pmatrix}$ と定める。

公理確認成立
(V1) $\mathbb{R}^2$ は集合である ✓
(V2) $\begin{pmatrix}0\\0\end{pmatrix}\in\mathbb{R}^2$ ✓
(V3) $+$ は $\mathbb{R}^2\times\mathbb{R}^2\to\mathbb{R}^2$、$\cdot$ は $\mathbb{R}\times\mathbb{R}^2\to\mathbb{R}^2$（成分の演算は $\mathbb{R}$ に閉じている） ✓
(V4) $\begin{pmatrix}x_1+x_2\\y_1+y_2\end{pmatrix} = \begin{pmatrix}x_2+x_1\\y_2+y_1\end{pmatrix}$（実数の可換律より） ✓
(V5) $\begin{pmatrix}(x_1{+}x_2){+}x_3\\(y_1{+}y_2){+}y_3\end{pmatrix} = \begin{pmatrix}x_1{+}(x_2{+}x_3)\\y_1{+}(y_2{+}y_3)\end{pmatrix}$（実数の結合律より） ✓
(V6) $\begin{pmatrix}x+0\\y+0\end{pmatrix} = \begin{pmatrix}x\\y\end{pmatrix}$ ✓
(V7) $\begin{pmatrix}x\\y\end{pmatrix}$ の逆元は $\begin{pmatrix}-x\\-y\end{pmatrix}$（$\mathbb{R}$ で $x+(-x)=0$） ✓
(V8) $a\begin{pmatrix}x_1+x_2\\y_1+y_2\end{pmatrix} = \begin{pmatrix}a(x_1+x_2)\\a(y_1+y_2)\end{pmatrix} = \begin{pmatrix}ax_1+ax_2\\ay_1+ay_2\end{pmatrix}$ ✓
(V9) $(a+b)\begin{pmatrix}x\\y\end{pmatrix} = \begin{pmatrix}(a+b)x\\(a+b)y\end{pmatrix} = \begin{pmatrix}ax+bx\\ay+by\end{pmatrix}$ ✓
(V10) $(ab)\begin{pmatrix}x\\y\end{pmatrix} = \begin{pmatrix}abx\\aby\end{pmatrix} = a\begin{pmatrix}bx\\by\end{pmatrix}$ ✓
(V11) $1\cdot\begin{pmatrix}x\\y\end{pmatrix} = \begin{pmatrix}1\cdot x\\1\cdot y\end{pmatrix} = \begin{pmatrix}x\\y\end{pmatrix}$ ✓

11 個すべてを満たすので $\mathbb{R}^2$ は $\mathbb{R}$-ベクトル空間。
同様に $\mathbb{K}^n$ も $\mathbb{K}$-ベクトル空間であることが成分ごとに確認できる。

注意 — ベクトル空間でない例

$\mathbb{Z}^2$（整数の格子点）は $\mathbb{R}$-ベクトル空間でない。

$\mathbb{Z}^2 = \left\{\begin{pmatrix}m\\n\end{pmatrix}\middle| m,n\in\mathbb{Z}\right\}$ に普通の加法・スカラー倍を入れると (V3) が破れる。 たとえば $\frac{1}{2}\in\mathbb{R}$ に対して $\frac{1}{2}\cdot\begin{pmatrix}1\\0\end{pmatrix} = \begin{pmatrix}1/2\\0\end{pmatrix} \notin \mathbb{Z}^2$。 スカラー倍の結果が $\mathbb{Z}^2$ に入らないため、$\cdot$ が $\mathbb{R}\times\mathbb{Z}^2\to\mathbb{Z}^2$ という写像にならない。

Corollary 1.2

系 1.2 — ベクトル空間の基本性質

系 1.2 — すぐわかること $V$ を $\mathbb{K}$-ベクトル空間とする。
  1. （逆元の一意性）$x+z=0$ かつ $x+y=0$ ならば $z=y$。よってこのような元を $-x$ と書く。
  2. $0\cdot x = 0 \quad (x\in V)$　 （スカラーの $0$ で掛けるとゼロベクトルになる）
  3. $(-1)\cdot x = -x$　 （$-1$ 倍すると逆元になる）
また、$x+(-y)$ のことを $x-y$ と書く。
系 1.2 の証明（クリックで展開）
(1) 逆元の一意性：
$x+z=0$ かつ $x+y=0$ と仮定する（どちらも $x$ の「逆元」の候補）。 $z=y$ を示す。各等号に使う公理を【】で示す： $$\begin{aligned} z &\overset{[\text{V6}]}{=} z + 0 \\ &\overset{[\text{V7}]}{=} z + (x+y) \quad (x+y=0\text{ を使用}) \\ &\overset{[\text{V5}]}{=} (z+x)+y \quad (\text{結合律}) \\ &\overset{[\text{V4}]}{=} (x+z)+y \quad (\text{可換律}) \\ &\overset{[\text{仮定}]}{=} 0+y \quad (x+z=0\text{ を使用}) \\ &\overset{[\text{V6}]}{=} y \end{aligned}$$ ポイント：「$x+y=0$（一方の逆元条件）」と「$x+z=0$（もう一方の逆元条件）」を うまく組み合わせて $z$ と $y$ が等しいことを導く。証明の出発点は $z = z+0$ という恒等式から始めること。

(2) $0\cdot x = 0$：
スカラーの $0$（$\mathbb{K}$ の零元）でベクトル $x$ を掛けるとゼロベクトルになることを示す。 各等号の右上に使った公理を示す： $$\begin{aligned} 0\cdot x &\overset{(\text{V6})}{=} 0\cdot x + 0 \\ &\overset{(\text{V7})}{=} 0\cdot x + \bigl(x + (-x)\bigr) \\ &\overset{(\text{V5})}{=} \bigl(0\cdot x + x\bigr) + (-x) \\ &\overset{(\text{V11})}{=} \bigl(0\cdot x + 1\cdot x\bigr) + (-x) \\ &\overset{(\text{V9})}{=} (0+1)\cdot x + (-x) \\ &\phantom{{}={}} = 1\cdot x + (-x) \quad (\mathbb{K}\text{ での計算：}0+1=1) \\ &\overset{(\text{V11})}{=} x + (-x) \\ &\overset{(\text{V7})}{=} 0 \end{aligned}$$

ポイント：$0$（ゼロベクトル）を $x+(-x)$ に書き換えて巧みに利用するのが鍵。 「スカラー $0$ を掛ければ当然 $0$」と直感的には明らかに見えるが、 それを公理だけから厳密に導くにはこのような迂回が必要。 これが公理的アプローチの特徴であり、「当たり前に見えることほど証明が要る」という事例である。


(3) $(-1)\cdot x = -x$：
$x + (-1)x = 1x + (-1)x = (1+(-1))x = 0\cdot x = 0$ より、$(-1)x$ は $x$ の逆元。(1) の一意性より $(-1)x = -x$。
$\blacksquare$
補足 — 系 1.2 の読み方
  • (1) 逆元の一意性：(V7) は「逆元が存在する」と言っているが、一意性は保証していない。系 1.2(1) でそれが証明される。だから $-x$ という記法が一意に定まる。
  • (2) ふたつの「$0$」の区別：$0\cdot x = 0$ の左辺の $0$ はスカラー（$\mathbb{K}$ の元）、右辺の $0$ はゼロベクトル（$V$ の元）。同じ記号だが意味が違う。
  • (3) 符号の整合性：「$-1$ 倍が逆元になる」は自明でなく、公理から証明が必要な事実。
  • 同様に $a\cdot 0 = 0$（任意の $a\in\mathbb{K}$、$0\in V$）も公理から示せる。
Remark 1.3

定例 1.3 — $n$ 個の元の和と一次結合

定例 1.3 — $n \ge 3$ 個の和

$n$ 個の元 $x_1,\ldots,x_n\in V$ に対して $x_1+\cdots+x_n$ を、「2 個の元の和」の繰り返しとして定義する。 どのようなカッコのつけ方をしても結果は同じ（$n$ についての帰納法で示せる）ので、カッコを外して書いてよい。

$n=4$ のときの例（カッコの付け方は 5 通り）

$x_1, x_2, x_3, x_4 \in V$ に対して、以下の 5 通りはすべて等しい：

$$\begin{aligned} &((x_1+x_2)+x_3)+x_4 \\ &(x_1+x_2)+(x_3+x_4) \\ &x_1+(x_2+(x_3+x_4)) \\ &(x_1+(x_2+x_3))+x_4 \\ &x_1+((x_2+x_3)+x_4) \end{aligned}$$

これはみな結合律 (V5) を組み合わせて示せる。一般の $n$ については $n$ についての帰納法で証明（オンデマンドで配信予定）。

これにより、$a_1,\ldots,a_n\in\mathbb{K}$、$x_1,\ldots,x_n\in V$ に対して $$a_1 x_1 + a_2 x_2 + \cdots + a_n x_n$$ という一次結合（線形結合、linear combination）が意味を持つ。

例 — $\mathbb{R}^3$ における一次結合 $v_1 = \begin{pmatrix}1\\0\\0\end{pmatrix}$, $v_2 = \begin{pmatrix}0\\1\\0\end{pmatrix}$, $v_3 = \begin{pmatrix}0\\0\\1\end{pmatrix}$ として、スカラー $a_1=2, a_2=-1, a_3=3$ による一次結合は： $$2v_1 + (-1)v_2 + 3v_3 = \begin{pmatrix}2\\0\\0\end{pmatrix} + \begin{pmatrix}0\\-1\\0\end{pmatrix} + \begin{pmatrix}0\\0\\3\end{pmatrix} = \begin{pmatrix}2\\-1\\3\end{pmatrix}$$ 実は $\mathbb{R}^3$ の任意のベクトルはこの $v_1, v_2, v_3$（基本単位ベクトル）の一次結合で表せる。
補足 — 一次結合の幾何的イメージ

一次結合 $a_1v_1+\cdots+a_kv_k$ は「$v_1$ を $a_1$ 倍し、$v_2$ を $a_2$ 倍し、…すべて足し合わせてできるベクトル」。 スカラー $a_i$ を動かすことで、どんなベクトルが作れるかが変わる。

  • 1 本のベクトル $v$ のスカラー倍全体： $\{av \mid a\in\mathbb{R}\}$（$= \langle v\rangle_{\mathbb{R}}$）は、 $v$ を延長・縮小・反転させた直線（原点を通る）。 $a=2$ は「$v$ の 2 倍の長さ」、$a=-1$ は「逆向き」、$a=0$ は「原点」。
  • 2 本のベクトル $v_1, v_2$（平行でない）の一次結合： $\{a_1v_1+a_2v_2\mid a_1,a_2\in\mathbb{R}\}$ は平面全体に広がる。 $v_1, v_2$ が「2 方向の基準」となり、平面内の任意の点に到達できる。
  • 2 本のベクトル $v_1, v_2$（平行）の一次結合： $v_2 = cv_1$（$c$ は定数）のとき、$a_1v_1+a_2v_2 = (a_1+ca_2)v_1$。 結局 $v_1$ の実数倍全体と同じで、直線になるにとどまる。

「$v_1,\ldots,v_k$ の一次結合全体」が §2 で学ぶ span $\langle v_1,\ldots,v_k\rangle$（生成された部分空間）であり、 どの方向が「$v_1,\ldots,v_k$ で到達可能か」を表す。

Definition 1.4

線形写像（linear map）

準備 — 「写像」とは何か

「写像（map）」とは、集合 $X$ の各元 $x$ に対して集合 $Y$ の元をちょうど 1 つ対応させる規則のこと。 $f: X\to Y$，$x\mapsto f(x)$ と書く。高校数学の「関数 $f(x)$」を、実数以外の集合に一般化した概念。

名称記号意味例
定義域（domain） $X$ 入力の集合。$f$ に「入れてよい値」の全体。 $f:\mathbb{R}^2\to\mathbb{R}$のとき $X=\mathbb{R}^2$
値域（codomain） $Y$ 出力の集合として宣言した集合。 同上のとき $Y=\mathbb{R}$
像（image） $f(X)$ 実際に出力として現れる $Y$ の部分集合 $\{f(x)\mid x\in X\}$。 $f(x,y)=x^2+y^2$ なら $f(\mathbb{R}^2)=[0,\infty)$

「写像である」ことの確認：$f:X\to Y$ が写像であるための必要十分条件は、 $X$ のすべての元 $x$ に対して $f(x)\in Y$ がちょうど 1 つ定まること（多値でも未定義でもない）。 $V$ の元への作用を「式で与える」場合、式が $V$ の元から $W$ の元に一意に対応しているか確認すれば十分。

特に (V3) の「$+: V\times V\to V$」とは、2 つのベクトル $(x,y)$ の組に対して $x+y$ というベクトルをちょうど 1 つ返す規則がある、という宣言である。 この写像が存在することは「ベクトルを足すと必ずまたベクトルが出てくる（$V$ の外に飛び出さない）」 ことを意味する。これを $V$ が加法に関して閉じている ともいう。

定義 1.4 — $\mathbb{K}$-線形写像 $V, W$: $\mathbb{K}$-ベクトル空間。写像 $T: V\to W$ が $\mathbb{K}$-線形写像 であるとは、以下の (L1), (L2) をともに満たすことをいう：
  • (L1) 任意の $x,y\in V$ に対して $T(x+y) = T(x)+T(y)$　（加法を保つ）
  • (L2) 任意の $a\in\mathbb{K},\ x\in V$ に対して $T(ax) = aT(x)$　（スカラー倍を保つ）
補足 — 線形写像の直感
  • (L1) 加法を保つ：「先に足してから $T$ を適用」しても「先に $T$ を適用してから足す」しても、結果が同じ。
    図式で書くと： $$x, y \xrightarrow{\text{足す}} x+y \xrightarrow{T} T(x+y) \quad\text{と}\quad x, y \xrightarrow{T} T(x), T(y) \xrightarrow{\text{足す}} T(x)+T(y)$$ が等しい、ということ。「$T$ を適用する前後で足し算の結果が変わらない」。
  • (L2) スカラー倍を保つ：「先にスカラー倍してから $T$」しても「先に $T$ してからスカラー倍」しても、結果が同じ。
    すなわち、$ax \xrightarrow{T} T(ax)$ と $x \xrightarrow{T} T(x) \xrightarrow{a\text{ 倍}} aT(x)$ が等しい。
  • 合わせると：$T(ax+by) = aT(x)+bT(y)$（一次結合を保つ）。これが「線形」の本質。
    さらに一般に $T(a_1x_1+\cdots+a_nx_n) = a_1T(x_1)+\cdots+a_nT(x_n)$（系 1.5(3) で証明）。
  • なぜ「線形」という名前か： $\mathbb{R}\to\mathbb{R}$ の線形写像は $T(x)=cx$（原点を通る直線）の形のみ。 「定数 $c$ でスケーリングする」こと以外できない。 これを高次元・抽象的な設定に一般化したものが「線形写像」であり、 行列によるベクトルの変換がその典型例。
  • $\mathbb{K}^n\to\mathbb{K}^m$ の場合：線形写像は行列で完全に記述できる（例 1 参照）。 逆に言えば、$\mathbb{K}^n\to\mathbb{K}^m$ の線形写像と $m\times n$ 行列は 1 対 1 に対応する。

非線形写像との対比：$f(x)=x^2$, $g(x)=\sin x$, $h(x)=x+1$ はどれも非線形。 例えば $h(x+y)=(x+y)+1$ だが $h(x)+h(y)=(x+1)+(y+1)=x+y+2\ne h(x+y)$。 「$+1$」というシフトが線形性を壊す。線形写像は必ず $T(0)=0$（系 1.5(1)）を満たすから、 「原点を原点に移さない写像」はすべて非線形。

線形写像の確認例

例題 A — $T$ が線形写像であることの確認

$T: \mathbb{R}^2\to\mathbb{R}^2$, $T\begin{pmatrix}x\\y\end{pmatrix} = \begin{pmatrix}2x+y\\x-3y\end{pmatrix}$ は線形写像か？

解答 (L1) の確認： $$T\!\left(\begin{pmatrix}x_1\\y_1\end{pmatrix}+\begin{pmatrix}x_2\\y_2\end{pmatrix}\right) = T\begin{pmatrix}x_1+x_2\\y_1+y_2\end{pmatrix} = \begin{pmatrix}2(x_1+x_2)+(y_1+y_2)\\(x_1+x_2)-3(y_1+y_2)\end{pmatrix} = \begin{pmatrix}(2x_1+y_1)+(2x_2+y_2)\\(x_1-3y_1)+(x_2-3y_2)\end{pmatrix}$$ $$T\begin{pmatrix}x_1\\y_1\end{pmatrix}+T\begin{pmatrix}x_2\\y_2\end{pmatrix} = \begin{pmatrix}2x_1+y_1\\x_1-3y_1\end{pmatrix}+\begin{pmatrix}2x_2+y_2\\x_2-3y_2\end{pmatrix} = \begin{pmatrix}(2x_1+y_1)+(2x_2+y_2)\\(x_1-3y_1)+(x_2-3y_2)\end{pmatrix}$$ 両者が一致するので (L1) が成立。

(L2) の確認： $$T\!\left(a\begin{pmatrix}x\\y\end{pmatrix}\right) = T\begin{pmatrix}ax\\ay\end{pmatrix} = \begin{pmatrix}2ax+ay\\ax-3ay\end{pmatrix} = a\begin{pmatrix}2x+y\\x-3y\end{pmatrix} = aT\begin{pmatrix}x\\y\end{pmatrix}$$ (L2) も成立。よって $T$ は線形写像。

行列表示：$T = T_A$ （例 1 参照）で、$A = \begin{pmatrix}2&1\\1&-3\end{pmatrix}$。
注意 — 線形写像でない例

例 B：$f: \mathbb{R}\to\mathbb{R}$, $f(x) = x+1$ は線形写像でない。

(L1) を確認する：$f(x+y) = (x+y)+1$, 一方 $f(x)+f(y) = (x+1)+(y+1) = x+y+2$。 一般に $f(x+y)\ne f(x)+f(y)$（$1\ne 2$）。また系 1.5(1) より線形写像は $T(0)=0$ を満たすが、$f(0)=1\ne 0$。

例 C：$g: \mathbb{R}\to\mathbb{R}$, $g(x) = x^2$ は線形写像でない。

$g(1+1) = 4$ だが $g(1)+g(1) = 1+1 = 2\ne 4$。

グラフが原点を通る直線でなければ（$\mathbb{R}\to\mathbb{R}$ の場合）線形でないことが多い。

Corollary 1.5

系 1.5 — 線形写像のすぐわかること

系 1.5 $T: V\to W$ が線形写像ならば：
  1. $T(0) = 0$
  2. $T(-x) = -T(x) \quad (x\in V)$
  3. $T(a_1 x_1+\cdots+a_n x_n) = a_1 T(x_1)+\cdots+a_n T(x_n)$　（一次結合を保つ）
系 1.5 の証明（クリックで展開）
(1)：$T(0) = T(0\cdot 0) = 0\cdot T(0) = 0$（L2 と 系1.2(2) より）

(2)：$T(-x) = T((-1)x) = (-1)T(x) = -T(x)$（L2 と 系1.2(3) より）

(3)：$n$ についての帰納法。
$n=1$ のとき：$T(a_1 x_1)=a_1 T(x_1)$（(L2) そのもの）。✓
$n=2$ のとき：$T(a_1 x_1 + a_2 x_2)=T(a_1 x_1)+T(a_2 x_2)$（(L1)）$= a_1 T(x_1)+a_2 T(x_2)$（(L2) を各項に適用）。✓
$n-1$ のとき成立すると仮定して $n$ のとき： $$T\!\left(\sum_{i=1}^n a_i x_i\right) = T\!\left(\sum_{i=1}^{n-1}a_i x_i + a_n x_n\right) \overset{(\text{L1})}{=} T\!\left(\sum_{i=1}^{n-1}a_i x_i\right) + T(a_n x_n) \overset{\text{帰納法仮定}}{=} \sum_{i=1}^{n-1}a_i T(x_i) + a_n T(x_n) = \sum_{i=1}^n a_i T(x_i)$$
$\blacksquare$
補足 — 系 1.5 の使い方と意義

(1) $T(0)=0$ の意義： 線形写像は必ず原点を原点に写す。これを逆用すると、$T(0)\ne 0$ であれば $T$ は線形写像でないと即座に判定できる。 例えば $T(x)=x+1$（$T(0)=1\ne 0$）は線形写像でない。 「まず $T(0)$ を計算する」のは線形性確認の最初のチェックとして有効。

(2) $T(-x)=-T(x)$ の意義： 線形写像は符号（逆元）を保つ。$-x$ を写すと必ず $T(x)$ の逆元になる。 これも線形性チェックの補助として使える。

(3) 一次結合の保存（最重要）： $T$ の値は $T(x_1),\ldots,T(x_n)$ を知れば、その一次結合 $a_1x_1+\cdots+a_nx_n$ の値がすべて決まる。 すなわち $V$ の「適切な有限個のベクトル（後に学ぶ「基底」）」における $T$ の値を 指定するだけで、$T$ 全体が決まる。

たとえば $T:\mathbb{R}^n\to\mathbb{K}^m$ の場合、基本単位ベクトル $e_1,\ldots,e_n$ に対する $T(e_1),\ldots,T(e_n)$ を決めるだけで $T$ 全体が決まり、 それが行列 $A=(T(e_1)\mid\cdots\mid T(e_n))$ による表示に相当する（例 1 参照）。

Examples

線形写像の例

例 1 — 行列による線形写像（最重要例）

$A\in M_{m,n}(\mathbb{K})$ とする。

$$T_A: \mathbb{K}^n \to \mathbb{K}^m, \quad v \mapsto Av$$

は $\mathbb{K}$-線形写像である（行列の演算規則から容易に確認できる）。

逆方向（重要）：任意の線形写像 $T:\mathbb{K}^n\to\mathbb{K}^m$ に対して

$$T = T_{(T(e_1)\mid T(e_2)\mid\cdots\mid T(e_n))}$$

と行列で表示できる。ここで $e_1,\ldots,e_n$ は $\mathbb{K}^n$ の基本単位ベクトル（$e_j$ は $j$ 番目成分が 1、他は 0）。 この行列 $A = (T(e_1)\mid\cdots\mid T(e_n))$ を $T$ の行列表示という。

理由：任意の $v = \begin{pmatrix}v_1\\\vdots\\v_n\end{pmatrix} = v_1 e_1+\cdots+v_n e_n$ に対して

$$T(v) = v_1 T(e_1)+\cdots+v_n T(e_n) = (T(e_1)\mid\cdots\mid T(e_n))\begin{pmatrix}v_1\\\vdots\\v_n\end{pmatrix} = Av$$ （系 1.5(3) を使用）
例題 — 行列表示を使った計算

$T:\mathbb{R}^3\to\mathbb{R}^2$ を線形写像で、基本単位ベクトルへの作用が

$$T(e_1) = \begin{pmatrix}1\\2\end{pmatrix},\quad T(e_2) = \begin{pmatrix}3\\-1\end{pmatrix},\quad T(e_3) = \begin{pmatrix}0\\4\end{pmatrix}$$

で与えられているとき、$T\begin{pmatrix}2\\-1\\3\end{pmatrix}$ を求めよ。

解答 行列表示は $A = \begin{pmatrix}1&3&0\\2&-1&4\end{pmatrix}$。 \begin{align} T\begin{pmatrix}2\\-1\\3\end{pmatrix} &= 2T(e_1) - T(e_2) + 3T(e_3) \\ &= 2\begin{pmatrix}1\\2\end{pmatrix} - \begin{pmatrix}3\\-1\end{pmatrix} + 3\begin{pmatrix}0\\4\end{pmatrix} = \begin{pmatrix}2-3+0\\4+1+12\end{pmatrix} = \begin{pmatrix}-1\\17\end{pmatrix} \end{align} 行列計算でも確認：$A\begin{pmatrix}2\\-1\\3\end{pmatrix} = \begin{pmatrix}1(2)+3(-1)+0(3)\\2(2)+(-1)(-1)+4(3)\end{pmatrix} = \begin{pmatrix}-1\\17\end{pmatrix}$。
例 2 — 左積による線形写像 $A\in M_{m,n}(\mathbb{K})$ とする。 $$M_{n,\ell}(\mathbb{K}) \to M_{m,\ell}(\mathbb{K}), \quad X \mapsto AX$$ は $\mathbb{K}$-線形写像である。（行列の分配法則 $A(X+Y)=AX+AY$, $A(\lambda X)=\lambda AX$ より。）
例 3 — 微分作用素（differential operator）

$-\infty < a < b < +\infty$, $f_0(x),\ldots,f_k(x)\in C^\infty((a,b))$ として、微分作用素

$$P = f_k(x)\frac{d^k}{dx^k} + \cdots + f_1(x)\frac{d}{dx} + f_0(x)$$

を次の写像として定める：

$$P: C^\infty((a,b)) \to C^\infty((a,b)), \quad \varphi \mapsto \sum_{r=1}^k f_r(x)\frac{d^r\varphi}{dx^r} + f_0(x)\varphi(x)$$

これは $\mathbb{C}$-線形写像である。

補足 — なぜ線形か 微分は線形演算である：$(f+g)' = f'+g'$（和の微分）、$(cf)' = cf'$（定数倍の微分）。 これがまさに (L1)(L2) に対応する。よって $P$ の線形性は微分の線形性から従う。
具体例：$P = \frac{d}{dx} - 2x$ のとき $P\varphi = \varphi' - 2x\varphi$。これが線形写像であることは、常微分方程式の理論において重要。
Preparation

全射・単射・全単射

用語の整理 — 全射・単射・全単射

次節の同型写像の定義に「全単射」という概念が使われる。ここで整理しておく。 写像 $f: X\to Y$ に対して：

名称定義直感逆写像
単射（injective） $x_1\ne x_2 \Rightarrow f(x_1)\ne f(x_2)$
同値：$f(x_1)=f(x_2)\Rightarrow x_1=x_2$		 異なる入力は必ず異なる出力になる。「情報が失われない」。 全単射のときのみ存在
全射（surjective） 任意の $y\in Y$ に対し、$f(x)=y$ となる $x\in X$ が存在する $Y$ の元すべてに「到達できる」。行き先のない $y$ がない。$f(X)=Y$。
全単射（bijective） 単射かつ全射 $X$ と $Y$ の元が完全に 1 対 1 に対応する。 逆写像 $f^{-1}:Y\to X$ が存在（$f\circ f^{-1}=\mathrm{id}_Y$，$f^{-1}\circ f=\mathrm{id}_X$）

具体例（$f:\mathbb{R}\to\mathbb{R}$）

  • $f(x)=2x+1$：単射かつ全射 → 全単射。逆写像 $f^{-1}(y)=\frac{y-1}{2}$ が存在。
  • $f(x)=x^2$：全射でも単射でもない。$f(1)=f(-1)=1$（単射でない）、$y=-1$ に届かない（全射でない）。
  • $f(x)=e^x$：単射だが全射でない。$e^x>0$ なので $y\le 0$ に届かない。
  • $f(x)=x^3$：全単射。$f^{-1}(y)=y^{1/3}$。
補足 — 単射性の言い換えがなぜ成り立つか

単射の定義 「$x_1\ne x_2\Rightarrow f(x_1)\ne f(x_2)$」 の対偶 をとると 「$f(x_1)=f(x_2)\Rightarrow x_1=x_2$」になる。 論理的に対偶は元の命題と同値なので、どちらを確認してもよい。 実際の証明では対偶形式 「$f(x_1)=f(x_2)$ と仮定して $x_1=x_2$ を示す」方が計算しやすいことが多い。

線形写像 $T:V\to W$ の単射性判定には、さらに便利な特徴付けがある（定理 3.3）： $$T\text{ が単射} \iff \operatorname{Ker}(T)=\{0\}$$ 一般の写像では「すべての $y$ に対して $T^{-1}(y)$ が高々 1 元」を調べる必要があるが、 線形写像では $y=0$ の場合だけ確認すれば十分というのが驚くべき事実である。

Definition 1.6 / 1.7

同型写像・同型

定義 1.6 $T: V\to W$ が $\mathbb{K}$-線形写像かつ全単射であるならば、逆写像 $T^{-1}:W\to V$ も線形写像になる。
定義 1.7 — 同型写像・同型 $V, W$: $\mathbb{K}$-ベクトル空間。
  1. $T: V\to W$ が全単射な線形写像のことを 同型写像（isomorphism） といい、$T: V\xrightarrow{\sim} W$ と書く。
  2. $V$ と $W$ が同型（isomorphic）であるとは、ある同型写像 $T: V\xrightarrow{\sim} W$ が存在することをいい、$V\cong W$ と書く。
同型の基本性質（同値関係）
  • $V \cong V$　（反射律：$\operatorname{id}_V$ が同型写像）
  • $V\cong W$ ならば $W\cong V$　（対称律：$T^{-1}$ が同型写像）
  • $V\cong W$ かつ $W\cong U$ ならば $V\cong U$　（推移律：$S\circ T$ が同型写像）

「$\cong$」はベクトル空間の間の同値関係をなす。

補足 — 同型の直感
  • 「同型 ＝ ラベルの貼り替え」：$V\cong W$ のとき、$T$ は $V$ の各元と $W$ の各元を 1 対 1 に対応させる「辞書」。ベクトル空間としての加法・スカラー倍の構造は完全に同じで、元の「名前（ラベル）」だけが異なる。
  • 全単射が必要な理由：単射でないと複数の元が同じ行き先になり逆写像が存在しない。全射でないと対応のない元が生じて「辞書」が不完全になる。
  • 定義 1.6 の注意：「全単射な線形写像の逆写像が自動的に線形」であることは自明でない。これは証明が必要（定理 3.6 や次の講義で扱う）。
補足 — 「逆写像が自動的に線形」は本当に自明でないのか

「$T:V\xrightarrow{\sim}W$ が線形かつ全単射なら $T^{-1}:W\to V$ も線形」という事実は、 一見「当然」に見えるが実際に確認が必要。以下で確認してみる。

$w_1, w_2\in W$ とする。$T$ が全単射なので $T(v_1)=w_1$, $T(v_2)=w_2$ となる $v_1,v_2\in V$ が一意に存在する。 このとき $T^{-1}(w_i)=v_i$。
(L1) の確認： $$T^{-1}(w_1+w_2) = T^{-1}(T(v_1)+T(v_2)) \overset{(L1)}{=} T^{-1}(T(v_1+v_2)) = v_1+v_2 = T^{-1}(w_1)+T^{-1}(w_2)$$ (L2) の確認：任意の $a\in\mathbb{K}$ に対して $$T^{-1}(aw_1) = T^{-1}(aT(v_1)) \overset{(L2)}{=} T^{-1}(T(av_1)) = av_1 = aT^{-1}(w_1)$$ いずれも $T$ の線形性（(L1)(L2)）を $T^{-1}$ の線形性に「転嫁」することで示せる。 見かけ上は「当然」だが、$T$ が線形であることを具体的に使っているため、 一般の（非線形な）全単射写像の逆写像は線形にならない場合がある。

例 — $\mathbb{C}^4 \cong M_2(\mathbb{C})$

$\mathbb{C}^4$ と $M_2(\mathbb{C})$（$2\times2$ 複素行列）は同型。以下の写像はいずれも線形写像で、$\cong$ かどうかが異なる。

写像 $T:\mathbb{C}^4\to M_2(\mathbb{C})$ 単射 全射 同型
$\begin{pmatrix}a\\b\\c\\d\end{pmatrix}\mapsto\begin{pmatrix}a&b\\c&d\end{pmatrix}$ ✓ ✓ ✓
$\begin{pmatrix}a\\b\\c\\d\end{pmatrix}\mapsto\begin{pmatrix}a&d\\b&c\end{pmatrix}$（行と列を入れ替え） ✓ ✓ ✓
$\begin{pmatrix}a\\b\\c\\d\end{pmatrix}\mapsto\begin{pmatrix}a&b\\c&a\end{pmatrix}$（$a$ が (1,1) と (2,2) 両方に登場） ✗ ✗ ✗

3 番目が単射でない理由： $\begin{pmatrix}1\\0\\0\\0\end{pmatrix}$ と $\begin{pmatrix}1\\0\\0\\1\end{pmatrix}$ はどちらも $\begin{pmatrix}1&0\\0&1\end{pmatrix}$ に送られる。 $d$ の情報が $(1,2)$ 成分に現れないため、$d$ を変えても出力が変わらない場合が生じる。
