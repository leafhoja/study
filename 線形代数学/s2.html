Definition 2.1

部分空間（subspace）

定義 2.1 — 部分空間 $V$: $\mathbb{K}$-ベクトル空間，$W\subseteq V$（$W$ は $V$ の部分集合）とする。 $W$ が $V$ の 部分空間（subspace） であるとは、以下の (S1)〜(S3) をすべて満たすことをいう：
  1. (S1) $0\in W$　（零元を含む）
  2. (S2) 任意の $x,y\in W$ に対し $x+y\in W$　（加法について閉じている）
  3. (S3) 任意の $a\in\mathbb{K},\ x\in W$ に対し $ax\in W$　（スカラー倍について閉じている）

重要：$W$ が $V$ の部分空間ならば、$(W, +|_W, 0, \cdot|_W)$ はそれ自身が $\mathbb{K}$-ベクトル空間の構造をもつ。 また、包含写像 $W\hookrightarrow V$（$w\mapsto w$）は $\mathbb{K}$-線形写像である。 さらに $V$ 自身も $V$ の部分空間（最大の部分空間）、$\{0\}$ も $V$ の部分空間（最小の部分空間）。

補足 — 部分空間の確認は 3 条件だけ

$W\subseteq V$ が部分空間かどうかを確かめるには、(S1)〜(S3) の 3 条件だけを示せばよい。 ベクトル空間の残りの公理 (V4)〜(V11)（可換律・結合律など）は $V$ での成立が $W$ にも自動的に引き継がれるからである。

条件何を確認するかポイント
(S1) $0\in W$ $W$ が空でないことの確認。零ベクトルが $W$ に属するか？ 最もよく忘れられる条件。まず確認せよ。
(S2) 加法に閉じている $W$ の元を 2 つ足しても $W$ に留まるか？ 「$W$ の元同士の和が $W$ の条件を満たすか」を確認。
(S3) スカラー倍に閉じている $W$ の元を任意のスカラーで掛けても $W$ に留まるか？ 「$0$ 倍」「負のスカラー」も含めて任意のスカラーで。
補足 — なぜ (S1)〜(S3) の 3 条件だけで十分か

ベクトル空間の定義 (V1)〜(V11) は 11 条件あった。 部分空間を確認するのに (S1)〜(S3) の 3 つだけで十分な理由を明確にしておく。

鍵は「$W\subseteq V$」、すなわち $W$ の元はすべて $V$ の元でもある、という事実。 $V$ の元として成り立つ性質は、$W$ の元に限定して眺めても当然成り立つ。

  • (V4) 可換律 $x+y=y+x$： $x,y\in W$ であれば $x,y\in V$（$W\subseteq V$ なので）。 $V$ では可換律が成り立つから、$W$ の元 $x,y$ に対しても $x+y=y+x$ は自動的に成立する。
  • (V5) 結合律, (V8)〜(V11)： 同様に、$V$ での成立が $W$ にも引き継がれる。
  • (V7) 逆元の存在： $x\in W$ に対して逆元 $-x$ が $W$ に属することは、(S3) から導ける。 スカラー $a=-1$ を選べば $(-1)\cdot x = -x \in W$（(S3) より $W$ はスカラー倍に閉じているから）。 よって (V7) も (S3) の特別な場合として保証される。

したがって「(S2) で和が $W$ に入る（加法に閉じている）」「(S3) でスカラー倍が $W$ に入る（スカラー倍に閉じている）」 ことを示しさえすれば、演算が $W$ 内で完結し、$W$ 自身が $\mathbb{K}$-ベクトル空間の構造を持つ。

補足 — 「閉じている」の意味を具体的に

「加法に閉じている」とは、「$W$ の元を 2 つ足した結果が必ず $W$ に属する」ということ。 「$W$ の外に飛び出さない」というイメージ。

  • 閉じている例：$W=\{(x,y)\in\mathbb{R}^2\mid y=2x\}$（原点を通る直線）。 $\begin{pmatrix}1\\2\end{pmatrix}\in W$ かつ $\begin{pmatrix}3\\6\end{pmatrix}\in W$ のとき、 $\begin{pmatrix}1\\2\end{pmatrix}+\begin{pmatrix}3\\6\end{pmatrix}=\begin{pmatrix}4\\8\end{pmatrix}$。 $8=2\times 4$ なので確かに $\in W$。
  • 閉じていない例：$W=\{(x,y)\in\mathbb{R}^2\mid x^2+y^2=1\}$（単位円）。 $\begin{pmatrix}1\\0\end{pmatrix}\in W$ かつ $\begin{pmatrix}0\\1\end{pmatrix}\in W$ だが、 $\begin{pmatrix}1\\0\end{pmatrix}+\begin{pmatrix}0\\1\end{pmatrix}=\begin{pmatrix}1\\1\end{pmatrix}$ は $1^2+1^2=2\ne 1$ なので $\notin W$。加法に閉じていない（だから部分空間でもない）。

スカラー倍も同様：「$W$ の元を任意のスカラーで掛けた結果が必ず $W$ に属する」。 特に $a=0$ のとき $(0)\cdot x = 0\in W$ となり (S1) が自動的に満たされるため、 実は (S1) を (S2) と (S3) から分離して示すのは「$W$ が空でないこと」を保証するためである （空集合 $\emptyset$ は (S2)(S3) を満たすが部分空間ではない）。

Examples

部分空間の具体例

例 A — 原点を通る直線（$\mathbb{R}^2$ の部分空間）

例 A $W_1 = \left\{\begin{pmatrix}x\\y\end{pmatrix}\in\mathbb{R}^2\ \middle|\ y = 2x\right\}$ （$y = 2x$ 上の点全体）は $\mathbb{R}^2$ の部分空間である。
確認 (S1)：$x=0$ とすると $y=2\cdot 0=0$。よって $\begin{pmatrix}0\\0\end{pmatrix}\in W_1$。✓

(S2)：$\begin{pmatrix}x_1\\2x_1\end{pmatrix},\begin{pmatrix}x_2\\2x_2\end{pmatrix}\in W_1$ とすると $$\begin{pmatrix}x_1\\2x_1\end{pmatrix}+\begin{pmatrix}x_2\\2x_2\end{pmatrix} = \begin{pmatrix}x_1+x_2\\2x_1+2x_2\end{pmatrix} = \begin{pmatrix}x_1+x_2\\2(x_1+x_2)\end{pmatrix}\in W_1$$ 第 2 成分が第 1 成分の 2 倍になっているので $W_1$ の条件を満たす。✓

(S3)：$\begin{pmatrix}x\\2x\end{pmatrix}\in W_1$, $a\in\mathbb{R}$ とすると $$a\begin{pmatrix}x\\2x\end{pmatrix} = \begin{pmatrix}ax\\2ax\end{pmatrix}\in W_1$$ 第 2 成分が第 1 成分の 2 倍。✓

よって $W_1$ は $\mathbb{R}^2$ の部分空間。 幾何的には原点を通る傾き 2 の直線。

例 B — 原点を通る平面（$\mathbb{R}^3$ の部分空間）

例 B $W_2 = \left\{\begin{pmatrix}x\\y\\z\end{pmatrix}\in\mathbb{R}^3\ \middle|\ x+y+z=0\right\}$ は $\mathbb{R}^3$ の部分空間である。
確認 (S1)：$0+0+0=0$ なので $\begin{pmatrix}0\\0\\0\end{pmatrix}\in W_2$。✓

(S2)：$x_1+y_1+z_1=0$ かつ $x_2+y_2+z_2=0$ とすると、 $(x_1+x_2)+(y_1+y_2)+(z_1+z_2) = (x_1+y_1+z_1)+(x_2+y_2+z_2) = 0+0 = 0$。✓

(S3)：$x+y+z=0$ かつ $a\in\mathbb{R}$ とすると、$ax+ay+az = a(x+y+z) = a\cdot 0 = 0$。✓

幾何的には $\mathbb{R}^3$ の中の原点を通る平面（法線ベクトル $(1,1,1)^T$）。

例 C — 連立一次方程式の解空間

例 C — 斉次連立方程式の解全体は部分空間

$A\in M_{m,n}(\mathbb{K})$ とする。斉次連立一次方程式の解全体

$$W = \{x\in\mathbb{K}^n \mid Ax = 0\}$$

は $\mathbb{K}^n$ の部分空間である。

確認 (S1)：$A\cdot 0 = 0$ なので $0\in W$。✓

(S2)：$Ax=0$ かつ $Ay=0$ とすると $A(x+y)=Ax+Ay=0+0=0$。✓

(S3)：$Ax=0$ かつ $a\in\mathbb{K}$ とすると $A(ax)=a(Ax)=a\cdot 0=0$。✓

この $W$ は §3 で学ぶ $\operatorname{Ker}(T_A)$（$T_A(v)=Av$ の核）に他ならない。 「斉次方程式の解空間は部分空間」という事実はこれで証明される。

具体例：$A = \begin{pmatrix}1&2&3\\0&1&2\end{pmatrix}$

$Ax=0$、すなわち $\begin{cases}x_1+2x_2+3x_3=0\\x_2+2x_3=0\end{cases}$ を解くと：$x_2=-2x_3$, $x_1=-2x_2-3x_3=4x_3-3x_3=x_3$。 自由変数 $x_3=t$ とおくと解は

$$W = \left\{t\begin{pmatrix}1\\-2\\1\end{pmatrix}\ \middle|\ t\in\mathbb{R}\right\}$$

これは $\mathbb{R}^3$ の中の原点を通る直線（1次元の部分空間）である。

Non-examples

部分空間でない例

注意 — (S1) が破れる例

$W_3 = \left\{\begin{pmatrix}x\\y\end{pmatrix}\in\mathbb{R}^2\ \middle|\ y = 2x+1\right\}$（$y = 2x+1$ 上の点全体）は部分空間でない。

理由：(S1) 違反。$\begin{pmatrix}0\\0\end{pmatrix}$ が $W_3$ に属するには $0 = 2\cdot 0 + 1 = 1$ が必要だが、これは偽。 つまり $0\notin W_3$。

幾何的には：原点を通らない直線は部分空間にならない。

注意 — (S2) が破れる例

$W_4 = \left\{\begin{pmatrix}x\\y\end{pmatrix}\in\mathbb{R}^2\ \middle|\ xy\ge 0\right\}$（第 1・3 象限＋座標軸の点全体）は部分空間でない。

理由：(S2) 違反。$\begin{pmatrix}1\\0\end{pmatrix},\begin{pmatrix}0\\-1\end{pmatrix}\in W_4$（それぞれ $xy=0\ge 0$）だが、 $\begin{pmatrix}1\\0\end{pmatrix}+\begin{pmatrix}0\\-1\end{pmatrix}=\begin{pmatrix}1\\-1\end{pmatrix}$ の $xy=-1<0$ なので $\notin W_4$。

注意 — (S3) が破れる例

$W_5 = \left\{\begin{pmatrix}x\\y\end{pmatrix}\in\mathbb{R}^2\ \middle|\ x,y\in\mathbb{Z}\right\}$（整数格子点全体）は $\mathbb{R}$-ベクトル空間 $\mathbb{R}^2$ の部分空間でない。

理由：(S3) 違反。$\begin{pmatrix}1\\0\end{pmatrix}\in W_5$ だが $\frac{1}{2}\begin{pmatrix}1\\0\end{pmatrix}=\begin{pmatrix}1/2\\0\end{pmatrix}\notin W_5$。

（(S1)(S2) は満たすが (S3) が満たされない。）

まとめ — 部分空間のイメージ

$\mathbb{R}^2$ の部分空間は以下の 3 種類しかない：

  • $\{0\}$（原点だけ）
  • 原点を通る直線（1次元）
  • $\mathbb{R}^2$ 全体（2次元）

$\mathbb{R}^3$ の部分空間：$\{0\}$, 原点を通る直線, 原点を通る平面, $\mathbb{R}^3$ 全体。

キーワード：「原点を通る」かつ「線形演算に閉じている」ものが部分空間。

Definition 2.2

生成される部分空間（span）

定義 2.2 — 生成される部分空間 $V$: $\mathbb{K}$-ベクトル空間，$v_1,\ldots,v_k\in V$ とする。 $$\langle v_1,\ldots,v_k\rangle_{\mathbb{K}} \overset{\text{def}}{=} \mathbb{K}v_1+\cdots+\mathbb{K}v_k = \{a_1 v_1+\cdots+a_k v_k \mid a_1,\ldots,a_k\in\mathbb{K}\}$$ とおくと、これは $V$ の部分空間になる。このことを「$v_1,\ldots,v_k$ が生成する（generate / span）部分空間」または 「a subspace generated by $v_1,\ldots,v_k$」という。
補足 — span の意味
  • $\langle v_1,\ldots,v_k\rangle_{\mathbb{K}}$ とは「$v_1,\ldots,v_k$ の一次結合全体の集合」。
  • $v_1,\ldots,v_k$ を含む最小の部分空間が span である（$v_1,\ldots,v_k$ を含む部分空間の共通部分に一致する）。
  • 英語では $\operatorname{span}\{v_1,\ldots,v_k\}$ とも書く。
補足 — 記法 $\mathbb{K}v_1$ の意味

定義に現れる $\mathbb{K}v_1$ とは、$v_1$ のスカラー倍全体の集合のこと： $$\mathbb{K}v_1 = \{av_1 \mid a\in\mathbb{K}\}$$ 同様に $\mathbb{K}v_2 = \{av_2\mid a\in\mathbb{K}\}$ など。 したがって $\mathbb{K}v_1+\mathbb{K}v_2+\cdots+\mathbb{K}v_k$ は 「$v_1$ のスカラー倍」＋「$v_2$ のスカラー倍」＋ … の和の集合、 すなわち $\{a_1v_1+a_2v_2+\cdots+a_kv_k\mid a_1,\ldots,a_k\in\mathbb{K}\}$ に等しい。

補足 — span が「最小の部分空間」である理由

$v_1,\ldots,v_k\in V$ を含む部分空間 $U$（すなわち $v_1,\ldots,v_k\in U$）は、 (S2)(S3) より必ず $v_1,\ldots,v_k$ の一次結合をすべて含む。 なぜなら：

  • $v_i\in U$ かつ $a_i\in\mathbb{K}$ ならば (S3) より $a_iv_i\in U$
  • $a_1v_1,\ldots,a_kv_k\in U$ ならば (S2) を繰り返して $a_1v_1+\cdots+a_kv_k\in U$

つまり $\langle v_1,\ldots,v_k\rangle_{\mathbb{K}} \subseteq U$。 一方 $\langle v_1,\ldots,v_k\rangle_{\mathbb{K}}$ 自身は $v_1,\ldots,v_k$ を含む部分空間（上の証明より）。

よって $\langle v_1,\ldots,v_k\rangle_{\mathbb{K}}$ は「$v_1,\ldots,v_k$ を含む部分空間の中で最も小さいもの（他の部分空間に含まれるもの）」となる。 これが「span は最小の部分空間」という言葉の意味である。

$\langle v_1,\ldots,v_k\rangle_{\mathbb{K}}$ が部分空間であることの証明（クリックで展開）
(S1)：$0 = 0\cdot v_1+\cdots+0\cdot v_k \in \langle v_1,\ldots,v_k\rangle_{\mathbb{K}}$。

(S2)：$(a_1 v_1+\cdots+a_k v_k) + (b_1 v_1+\cdots+b_k v_k) = (a_1+b_1)v_1+\cdots+(a_k+b_k)v_k \in \langle v_1,\ldots,v_k\rangle_{\mathbb{K}}$。

(S3)：$\lambda(a_1 v_1+\cdots+a_k v_k) = (\lambda a_1)v_1+\cdots+(\lambda a_k)v_k \in \langle v_1,\ldots,v_k\rangle_{\mathbb{K}}$。
$\blacksquare$

span の具体例

例 D — $\mathbb{R}^2$ の span
  • $\left\langle\begin{pmatrix}1\\2\end{pmatrix}\right\rangle_{\mathbb{R}} = \left\{t\begin{pmatrix}1\\2\end{pmatrix}\ \middle|\ t\in\mathbb{R}\right\} = \left\{\begin{pmatrix}t\\2t\end{pmatrix}\ \middle|\ t\in\mathbb{R}\right\}$ は $\mathbb{R}^2$ の部分空間（原点を通る傾き 2 の直線）。
  • $\left\langle\begin{pmatrix}1\\0\end{pmatrix},\begin{pmatrix}0\\1\end{pmatrix}\right\rangle_{\mathbb{R}} = \left\{a\begin{pmatrix}1\\0\end{pmatrix}+b\begin{pmatrix}0\\1\end{pmatrix}\ \middle|\ a,b\in\mathbb{R}\right\} = \left\{\begin{pmatrix}a\\b\end{pmatrix}\ \middle|\ a,b\in\mathbb{R}\right\} = \mathbb{R}^2$ （全空間を生成する）。
  • $\left\langle\begin{pmatrix}1\\2\end{pmatrix},\begin{pmatrix}2\\4\end{pmatrix}\right\rangle_{\mathbb{R}}$： $\begin{pmatrix}2\\4\end{pmatrix} = 2\begin{pmatrix}1\\2\end{pmatrix}$ なので、これは単に $\left\langle\begin{pmatrix}1\\2\end{pmatrix}\right\rangle_{\mathbb{R}}$ に等しい（直線）。 2 つのベクトルが平行なので span は 1 次元のまま。
例 E — $\mathbb{R}^3$ の span $v_1=\begin{pmatrix}1\\0\\0\end{pmatrix}$, $v_2=\begin{pmatrix}0\\1\\0\end{pmatrix}$, $v_3=\begin{pmatrix}0\\0\\1\end{pmatrix}$ として：
  • $\langle v_1\rangle = \{(t,0,0)^T\mid t\in\mathbb{R}\}$：$x$ 軸（1 次元）
  • $\langle v_1, v_2\rangle = \{(a,b,0)^T\mid a,b\in\mathbb{R}\}$：$xy$ 平面（2 次元）
  • $\langle v_1, v_2, v_3\rangle = \mathbb{R}^3$（3 次元、全空間）
補足 — span と線形写像の像

線形写像 $T:\mathbb{K}^n\to\mathbb{K}^m$（行列表示 $A$）について：

$$\operatorname{Im}(T) = \{Av\mid v\in\mathbb{K}^n\} = \langle T(e_1),\ldots,T(e_n)\rangle_{\mathbb{K}} = \langle A\text{ の列ベクトル全体}\rangle_{\mathbb{K}}$$

つまり $\operatorname{Im}(T)$ は $A$ の列空間（column space）に等しい。 §3 でさらに詳しく扱う。
