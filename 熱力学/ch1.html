第 1 章

熱力学の基本概念

熱力学の土台となる概念 ―― 温度・熱平衡・状態量 ―― を順に導入する。 同時に，状態方程式の微小変化を記述するための数学的道具（偏微分・全微分・テイラー展開）を整備し， 実験で測定可能な係数 $\beta$（定圧膨張率），$\kappa$（等温圧縮率），$\alpha_P$（圧力係数）の関係を導く。

1.1 温度と経験温度

日常的な意味での温度は，「熱い・冷たい」という感覚を定量化したものであり， 物質固有の性質を観測することで間接的に測定される。温度そのものを直接「見る」ことはできない。

温度計の種類利用する物性
水銀温度計・アルコール温度計液体の体積（膨張）
電子体温計・白金抵抗温度計電気抵抗（電気伝導度）
熱電対温度計熱起電力
定義経験温度（摂氏温度）

1 気圧のもとで，水の融点を $0\,^\circ\mathrm{C}$，沸点を $100\,^\circ\mathrm{C}$ と定義し， その間を 100 等分したものを 1℃ とする。 この「目盛りの付け方（約束）」を経験温度という。

摂氏温度 $t\,[^\circ\mathrm{C}]$ と熱力学温度 $T\,[\mathrm{K}]$ の変換： $$T\,[\mathrm{K}] = t\,[^\circ\mathrm{C}] + 273.15$$

ここで次の重要な問いが生じる： 「水銀で測った温度と気体で測った温度が「同じ」というのは，本当に普遍的なことか？」 これへの答えが熱力学温度（第3章）である。

偏微分の定義

定義偏微分
2変数関数 $z = f(x, y)$ の $x$ による偏微分を： $$\left(\frac{\partial f}{\partial x}\right)_y := \lim_{\Delta x \to 0} \frac{f(x + \Delta x,\; y) - f(x,\; y)}{\Delta x}$$ と定義する。添字 $y$ は「$y$ を固定したまま $x$ だけを変化させる」という意味。

熱力学では，固定する量が物理的に重大な意味をもつ。 例えば $\left(\dfrac{\partial V}{\partial T}\right)_P$ と $\left(\dfrac{\partial V}{\partial T}\right)_S$ では， 固定するものが圧力 $P$ かエントロピー $S$ かで値が異なり，物理的意味も全く違う。 添字を省略してはならない。

定圧膨張率 $\beta$

定義定圧膨張率（熱膨張率） $\beta$
$$\beta = \frac{1}{V}\left(\frac{\partial V}{\partial T}\right)_P$$

圧力 $P$ を一定に保ちながら温度を 1 K 上昇させたとき， 体積が相対的にどれだけ変化するかを表す係数。単位は $\mathrm{K}^{-1}$。 「体積を $V$ で割る」のは，系の大きさに依存しない（示強性の）係数にするため。

例理想気体の $\beta$

理想気体の状態方程式 $PV = nRT$ から $V = nRT/P$（圧力 $P$ 一定）。よって：

$$\beta = \frac{1}{V}\left(\frac{\partial V}{\partial T}\right)_P = \frac{1}{V} \cdot \frac{nR}{P} = \frac{1}{V} \cdot \frac{PV}{RT} = \frac{1}{T} \;\ge 0$$

$\beta = 1/T > 0$ は「温度を上げると体積が増える」ことに対応する。 これは $T > 0$（熱力学温度が非負）と整合している。

補足水の異常：$4^\circ\mathrm{C}$ で密度が最大

ほとんどの物質では $\beta > 0$ だが，水は $0^\circ\mathrm{C}$ から $4^\circ\mathrm{C}$ の間で $\beta < 0$。 すなわち，温めると縮む（密度が増す）。$4^\circ\mathrm{C}$ で密度が最大となり， 湖の底は冬でも約 $4^\circ\mathrm{C}$ に保たれる（$\Rightarrow$ 湖が完全凍結せず水生生物が冬を越せる）。

この異常は水分子の水素結合ネットワーク構造に起因するもので，理想気体とは本質的に異なる。

★ 温度の本質についての注意

重要温度計は使えるが，温度の本質はまだ解明されていない

温度は，温度計の目盛りを一度約束してしまえば，誰でも簡単に測定できる量である。 しかし「温度とは本当は何を測っているのか」という問いに答えるためには：

  • 熱力学第二法則（Carnotサイクル）から「熱力学温度」を定義するか（第3章），
  • 分子運動論・統計力学で「温度は分子の平均運動エネルギーに比例する量」として理解するか，

のどちらかが必要となる。この第1章では，その答えを保留にしたまま進む。

1.2 熱平衡

熱平衡の定義と例

定義熱平衡状態
「熱」の出入りが釣り合って，マクロな観測量が時間的に変化しない状態を 熱平衡状態という。
  • マクロには静止している（観測量 = 一定）
  • ミクロには常に揺らいでいる（各分子は熱運動している）
例食塩水の熱平衡

容器に入った食塩水は，マクロに見れば「濃度・体積」が時間的に一定 → 熱平衡状態。 ミクロに見れば，個々の Na$^+$ や Cl$^-$ は絶えず熱運動しており， 局所的な濃度は場所ごとに微妙に異なる。 熱力学が扱うのは，あくまでもこの「マクロに見た平均的な状態」である。

重要熱力学が扱う対象
熱力学は，熱平衡状態と，熱平衡状態の間の遷移過程を扱う学問である。 平衡でない状態（例：乱流・化学反応の途中・光が当たり続けている系）は 通常の熱力学の枠組みの外にある。

熱力学第0法則（熱平衡の推移律）

第ゼロ法則熱平衡の存在の保証（経験則）

物体 A と B が熱平衡にあり，かつ A と C も熱平衡にあるとき， B と C も熱平衡にある。

「A を温度計とみなす」と：「同じ温度計で同じ値を示す物体は互いに熱平衡にある」。 これが「温度」という共通尺度の存在を論理的に保証する。

歴史なぜ「第ゼロ法則」か

この法則の内容は第一・第二法則よりも論理的に先立つ（温度概念の前提）が， R. H. Fowler（1889–1944）がその重要性を指摘したのは， 第一・第二法則がすでに確立された後のことだった（1930年代）。 「後から付け加えた序章」であるため「第ゼロ法則」と命名された。

第ゼロ法則から，温度という概念が一貫して定義できる。 さらに，温度の間の関係から熱容量の非負性が導かれる。

性質温度の単調性と熱容量の非負性

温度 $T_1$ の物体と温度 $T_2$（$T_1 < T_2$）の物体を熱接触させると， 平衡温度 $T^*$ は： $$T_1 < T^* < T_2$$ を満たす（中間温度）。

これより，物体の熱容量は非負（$C \ge 0$）であることが示される： 熱を加えると温度は下がらないということ。

熱容量の定義

定義熱容量 $C(T)$
$$C(T) = \lim_{\Delta T \to 0} \frac{\Delta Q}{\Delta T}$$

温度 $T$ の物体の温度を微小量 $\Delta T$ だけ上昇させるのに必要な熱量 $\Delta Q$ の比の極限値。 単位は $\mathrm{J/K}$。 $C \ge 0$（熱容量の非負性）は上記の温度の単調性から保証される。

何を一定に保つかによって熱容量の値は変わる：定積熱容量 $C_V$，定圧熱容量 $C_P$（第2章で詳述）。

1.3 状態量と状態方程式

状態量の定義

定義状態量（state variable）
状態量とは，熱平衡状態に対して一意に定まる物理量である。 過去の履歴や変化の経路によらず，現在の熱平衡状態だけで決まる。
  • 状態量の例：温度 $T$，体積 $V$，圧力 $P$，内部エネルギー $U$，エントロピー $S$ など
  • 状態量でない例：熱 $Q$，仕事 $W$（これらは「過程量」で経路に依存する）

示量性と示強性

状態量は「系の大きさ」に対してどのように変化するかによって2種類に分類される。 温度 $T$ 一定のまま系のサイズを2倍にする（粒子数 $N$・体積 $V$ を2倍にする）ことを想定する。

種類定義例
示量性（extensive） 系を2倍にすると2倍になる
$X(T, 2V, 2N) = 2X(T,V,N)$		 体積 $V$，内部エネルギー $U$，エントロピー $S$，物質量 $n$，磁化 $M$
示強性（intensive） 系を2倍にしても変わらない
$Y(T, 2V, 2N) = Y(T,V,N)$		 温度 $T$，圧力 $P$，化学ポテンシャル $\mu$，磁場 $H$
補足共役ペアとエネルギー

示量性と示強性は「共役なペア」として現れることが多い： $(S, T)$，$(V, P)$，$(M, H)$，$(n, \mu)$ など。 これらのペアの積は「エネルギー」の次元をもつ（例：$P \cdot dV$ は仕事の次元）。 この対応は，熱力学関数（Helmholtz自由エネルギー $F$，Gibbs自由エネルギー $G$ など）を 理解する上で重要（第4章）。

状態量の自由度と状態方程式

経験則気体の状態は2つの変数で決まる

単純な気体（1成分・均一系）では， 熱平衡状態を指定するために必要な独立な状態変数の数は2つで十分である。

これはGibbsの相律$f = C - P_\text{相} + 2$（$C$：成分数，$P_\text{相}$：相の数）から 1成分・1相の場合 $f = 1 - 1 + 2 = 2$ として導かれる。

変数の選び方（組み合わせの自由度）も2つある。 例えば $(T, V)$，$(T, P)$，$(P, V)$ のいずれかを独立変数に取れる。

定義状態方程式
状態量の間に成立する関係式を状態方程式という。 状態変数が2つなので，それ以外の状態量はこの2つの関数として表される。

例えば $P$ を $(T,V)$ の関数として表したもの：

  • 理想気体：$P = nRT/V$（ボイル–シャルルの法則）
  • ファンデルワールス気体：$\displaystyle\left(P + \frac{n^2 a}{V^2}\right)(V - nb) = nRT$（分子間力と体積を補正）

状態方程式は実験事実から得られるもの（理論からだけでは導けない）。 ただし，状態方程式の具体的な形は実験で測定できる係数（$\beta$, $\kappa$ など）から 組み立てることができる。

1変数テイラー展開の復習

状態方程式の微小変化を扱うために，まずテイラー展開を復習する。

公式1変数テイラー展開

$y = f(x)$ が滑らかな関数であれば，$x = x_0$ のまわりで：

$$f(x) = f(x_0) + f'(x_0)(x - x_0) + \frac{1}{2!}f''(x_0)(x-x_0)^2 + \cdots$$

$x = x_0 + \Delta x$ と置いて1次の項まで取ると：

$$\Delta f = f(x_0 + \Delta x) - f(x_0) \approx f'(x_0)\,\Delta x \quad (\Delta x \to 0)$$

これが「$x$ の微小変化 $\Delta x$ に対する $f$ の変化量の1次近似」。 $\Delta x \to 0$ の極限で $dx$，$df$ という微分の記号を使う。

2変数テイラー展開と全微分

状態方程式は2変数関数 $z = f(x, y)$ の形をもつ（例：$P = P(T,V)$）。 2点 $(x_0, y_0)$ と $(x_0 + \Delta x, y_0 + \Delta y)$ の間での変化を展開する。

導出2変数テイラー展開 → 全微分

$\Delta z = f(x_0+\Delta x,\; y_0 + \Delta y) - f(x_0, y_0)$ を計算する。

Step 1：$y$ を $y_0 + \Delta y$ に固定して $x$ を $x_0 \to x_0 + \Delta x$ へ動かす：

$$f(x_0+\Delta x,\; y_0+\Delta y) = f(x_0,\; y_0+\Delta y) + \left(\frac{\partial f}{\partial x}\right)_{y_0+\Delta y}\!\!\Delta x + O(\Delta x^2) \quad \cdots \circled{1}$$

Step 2：$f(x_0,\; y_0+\Delta y)$ を $y_0$ まわりで展開：

$$f(x_0,\; y_0+\Delta y) = f(x_0, y_0) + \left(\frac{\partial f}{\partial y}\right)_{x_0}\!\!\Delta y + O(\Delta y^2) \quad \cdots \circled{2}$$

Step 3：$\circled{1}$ の偏微分係数を $y_0$ での値で置き換え：

$$\left(\frac{\partial f}{\partial x}\right)_{y_0+\Delta y} = \left(\frac{\partial f}{\partial x}\right)_{y_0} + O(\Delta y)$$

$\Delta x \cdot O(\Delta y)$ は高次微小なので無視。

Step 4：$\circled{1}$ に $\circled{2}$ と Step 3 を代入し，$\Delta x, \Delta y \to 0$ の極限：

$$\Delta z = \left(\frac{\partial f}{\partial x}\right)_{y_0}\Delta x + \left(\frac{\partial f}{\partial y}\right)_{x_0}\Delta y + O(2次以上)$$

$\Delta x \to dx$，$\Delta y \to dy$，$\Delta z \to dz$ として，次の全微分を得る：

公式全微分（2変数関数の1次近似）
$z = f(x, y)$ が滑らかな関数であれば，任意の $(x_0, y_0)$ の近傍で： $$\boxed{dz = \left(\frac{\partial f}{\partial x}\right)_y dx + \left(\frac{\partial f}{\partial y}\right)_x dy}$$ これは「$x$ が $dx$，$y$ が $dy$ だけ変化したときの $z$ の変化量 $dz$」の1次近似。

$P(T,V)$ の全微分形

気体の圧力 $P$ を $T$（温度）と $V$（体積）の関数 $P = P(T, V)$ と見ると，全微分は：

$$dP = \left(\frac{\partial P}{\partial T}\right)_V dT + \left(\frac{\partial P}{\partial V}\right)_T dV$$

$(T,V)$ が $(T+\Delta T, V + \Delta V)$ に変化したとき， $P$ の変化は $dP = \left(\partial P/\partial T\right)_V \Delta T + \left(\partial P/\partial V\right)_T \Delta V + O(2次)$ で近似される。

等温変化への適用$dP = 0$（等温変化）のとき

等温変化 $dT = 0$ かつ $dP = 0$（圧力一定という条件ではなく，等温線に沿った等圧状態を考える） の場合，全微分から：

$$0 = \left(\frac{\partial P}{\partial T}\right)_V + \left(\frac{\partial P}{\partial V}\right)_T \frac{dV}{dT}$$

これは次の「偏微分の $-1$ 関係」の導出に使われる。

3変数の偏微分値の $-1$ 関係

公式偏微分の三重積 $= -1$
3つの状態量 $x, y, z$ が1つの方程式 $f(x,y,z) = 0$ で結ばれているとき： $$\boxed{\left(\frac{\partial x}{\partial y}\right)_z \left(\frac{\partial y}{\partial z}\right)_x \left(\frac{\partial z}{\partial x}\right)_y = -1}$$ 例えば $P, V, T$（状態方程式で1本の拘束がある場合）： $$\left(\frac{\partial P}{\partial V}\right)_T \left(\frac{\partial V}{\partial T}\right)_P \left(\frac{\partial T}{\partial P}\right)_V = -1$$
証明三重積 $= -1$ の証明（一般の 3 変数）

$x, y, z$ が 3 変数だが 1 本の拘束式で結ばれているとする（独立変数は2つ）。

$x$ を $y, z$ の関数として全微分する：

$$dx = \left(\frac{\partial x}{\partial y}\right)_z dy + \left(\frac{\partial x}{\partial z}\right)_y dz \quad \cdots \circled{1}$$

$y$ を $x, z$ の関数として全微分する：

$$dy = \left(\frac{\partial y}{\partial x}\right)_z dx + \left(\frac{\partial y}{\partial z}\right)_x dz \quad \cdots \circled{2}$$

$\circled{2}$ を $\circled{1}$ に代入する：

$$dx = \left(\frac{\partial x}{\partial y}\right)_z \left[\left(\frac{\partial y}{\partial x}\right)_z dx + \left(\frac{\partial y}{\partial z}\right)_x dz\right] + \left(\frac{\partial x}{\partial z}\right)_y dz$$ $$= \left(\frac{\partial x}{\partial y}\right)_z\left(\frac{\partial y}{\partial x}\right)_z dx + \left[\left(\frac{\partial x}{\partial y}\right)_z\left(\frac{\partial y}{\partial z}\right)_x + \left(\frac{\partial x}{\partial z}\right)_y\right] dz$$

$x$ と $z$ は独立変数なので，$dx$ と $dz$ の係数をそれぞれ比較できる：

  • $dx$ の係数：$1 = \left(\dfrac{\partial x}{\partial y}\right)_z\left(\dfrac{\partial y}{\partial x}\right)_z$ → 逆数の関係（偏微分の逆関係）
  • $dz$ の係数：$0 = \left(\dfrac{\partial x}{\partial y}\right)_z\left(\dfrac{\partial y}{\partial z}\right)_x + \left(\dfrac{\partial x}{\partial z}\right)_y$

$dz$ の係数から：

$$\left(\frac{\partial x}{\partial y}\right)_z\left(\frac{\partial y}{\partial z}\right)_x = -\left(\frac{\partial x}{\partial z}\right)_y$$

両辺に $\left(\dfrac{\partial z}{\partial x}\right)_y$ を掛けると（逆数の関係から）：

$$\left(\frac{\partial x}{\partial y}\right)_z \left(\frac{\partial y}{\partial z}\right)_x \left(\frac{\partial z}{\partial x}\right)_y = -1 \quad \blacksquare$$
検証理想気体で三重積を確認

$PV = nRT$ から各偏微分を計算：

$$\left(\frac{\partial P}{\partial V}\right)_T = -\frac{nRT}{V^2} = -\frac{P}{V}, \quad \left(\frac{\partial V}{\partial T}\right)_P = \frac{nR}{P}, \quad \left(\frac{\partial T}{\partial P}\right)_V = \frac{V}{nR}$$

積を計算：

$$\left(-\frac{P}{V}\right)\cdot\frac{nR}{P}\cdot\frac{V}{nR} = -1 \quad \checkmark$$

状態方程式の微分形と $\beta$，$\kappa$，$\alpha_P$

今度は $V$ を独立変数 $(T, P)$ の関数として表す：$V = V(T, P)$。 全微分を取ると：

$$dV = \left(\frac{\partial V}{\partial T}\right)_P dT + \left(\frac{\partial V}{\partial P}\right)_T dP$$

ここで登場する2つの偏微分係数が実験的に測定可能な重要な物理量である。

定義定圧膨張率 $\beta$ と等温圧縮率 $\kappa$
$$\beta = \frac{1}{V}\left(\frac{\partial V}{\partial T}\right)_P \quad\text{（定圧膨張率，熱膨張率）}$$ $$\kappa = -\frac{1}{V}\left(\frac{\partial V}{\partial P}\right)_T \quad\text{（等温圧縮率）}$$

$\kappa$ の符号：$T$ 一定で圧力 $P$ を増やすと体積 $V$ は減る（$\partial V/\partial P < 0$）。 $\kappa$ が正の値になるよう，定義の前に $-1$ をつける。 単位はどちらも $\mathrm{Pa}^{-1}$ または $\mathrm{K}^{-1}$（それぞれ）。

これらを使うと，全微分式を書き直せる：

$$\boxed{dV = V\beta\,dT - V\kappa\,dP}$$
例理想気体の $\kappa$

$V = nRT/P$ から：

$$\kappa = -\frac{1}{V}\left(\frac{\partial V}{\partial P}\right)_T = -\frac{1}{V}\cdot\left(-\frac{nRT}{P^2}\right) = \frac{nRT}{PV \cdot P} = \frac{1}{P} > 0$$

$\kappa = 1/P$ は「圧力が高いほど圧縮しにくい」ことを意味する。 また，$\beta/\kappa = (1/T)/(1/P) = P/T$ は後述の圧力係数 $\alpha_P$ に等しい。

公式圧力係数 $\alpha_P = \beta/\kappa$

定積変化 $dV = 0$ の条件を $dV = V\beta\,dT - V\kappa\,dP$ に代入すると：

$$0 = V\beta\,dT - V\kappa\,dP \implies \frac{dP}{dT}\bigg|_{V=\text{const}} = \frac{\beta}{\kappa}$$

すなわち：

$$\boxed{\alpha_P := \left(\frac{\partial P}{\partial T}\right)_V = \frac{\beta}{\kappa}}$$

$\alpha_P$（圧力係数）は「体積一定のまま温度を上げたときの圧力変化率」。 $\beta$（膨張しやすさ）と $\kappa$（圧縮のしにくさ）で表せることがわかる。

三重積との整合性$\alpha_P = \beta/\kappa$ の別導出

三重積 $= -1$ の公式に $\beta$ と $\kappa$ の定義を代入しても同じ結果が得られる：

$$\underbrace{\left(\frac{\partial P}{\partial V}\right)_T}_{-1/(V\kappa)} \cdot \underbrace{\left(\frac{\partial V}{\partial T}\right)_P}_{V\beta} \cdot \left(\frac{\partial T}{\partial P}\right)_V = -1$$ $$\frac{-V\beta}{V\kappa}\cdot\left(\frac{\partial T}{\partial P}\right)_V = -1 \implies \left(\frac{\partial T}{\partial P}\right)_V = \frac{\kappa}{\beta}$$

逆数をとって $\alpha_P = (\partial P/\partial T)_V = \beta/\kappa$。2つの導出が一致する。

応用3つの係数のうち2つで1つが決まる

$\alpha_P = \beta/\kappa$ より，$\beta$（定圧膨張率），$\kappa$（等温圧縮率），$\alpha_P$（圧力係数）の うち2つを測定すれば残りの1つが求まる。 体積一定 $V = \text{const}$ を厳密に保ちながら圧力を測る実験 $\alpha_P$ は難しいが， $\beta$ と $\kappa$ から間接的に計算できる。

さらに，第2章で示す一般式 $C_P - C_V = TV\beta^2/\kappa = TV\beta\alpha_P$ に代入すると， $\beta$ と $\kappa$ だけで比熱差も求められる。

完全微分の条件（補足）

2変数1形式 $\omega = A(x,y)\,dx + B(x,y)\,dy$ が「ある関数 $f$ の全微分 $df$」と書けるための条件を補足する。 これは第2章以降（熱の不完全微分・Maxwell関係式）で使う。

定理完全微分の条件（Schwarz の定理）

$\omega = A\,dx + B\,dy$ が完全微分（$\omega = df$ となる $f$ が存在する）であるための必要十分条件：

$$\left(\frac{\partial A}{\partial y}\right)_x = \left(\frac{\partial B}{\partial x}\right)_y$$

これは混合偏微分の交換可能性（Schwarz の定理） $\dfrac{\partial^2 f}{\partial x\partial y} = \dfrac{\partial^2 f}{\partial y\partial x}$ と同値。

熱力学への応用：$dU = T\,dS - P\,dV$ に対して $A = T$，$B = -P$，$x = S$，$y = V$ とすると： $$\left(\frac{\partial T}{\partial V}\right)_S = -\left(\frac{\partial P}{\partial S}\right)_V$$ これが Maxwell 関係式の一つ（第4章）。

例$\delta Q$ は完全微分でない（理想気体）

理想気体の準静的過程での熱：$\delta Q = C_V\,dT + (nRT/V)\,dV$（第2章で導出）

$A = C_V$，$B = nRT/V$ として条件を確認：

$$\left(\frac{\partial A}{\partial V}\right)_T = 0, \quad \left(\frac{\partial B}{\partial T}\right)_V = \frac{nR}{V} \ne 0$$

条件を満たさないので，$\delta Q$ は不完全微分 ─ 経路によって $\int \delta Q$ の値が変わる。

一方，$dU = C_V\,dT$（理想気体）は明らかに $U = C_V T + \text{const}$ の全微分で，完全微分。

記法$\delta$ と $d$ の使い分け
  • $dU$，$dS$，$dT$，$dV$，$dP$ など：完全微分（状態量の変化）
  • $\delta Q$，$\delta W$：不完全微分（過程量；経路依存）

第一法則 $dU = \delta Q + \delta W$ では，右辺は各々経路依存だが その和だけが経路によらない（$dU$ は完全微分）。
