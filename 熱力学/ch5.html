第 5 章

応用例

前章までに構築した熱力学の理論を具体的な系に適用する。 2相共存系では Clausius-Clapeyron の関係式を導き、 ゴムの熱力学では気体とは異なる仕事の形 $\delta W = X\,dl$ を扱い、 Gough-Joule 効果（温めると縮む）を熱力学的に説明する。

5.1 2相共存系

相転移の概観

液体と気体が共存するとき、温度 $T$ を固定した P-V 図は次のようになる：

  • $V < V_0$（液体のみ）：体積を変えても圧力がほぼ変化しない（液体は非圧縮性）
  • $V_0 \le V \le V_1$（液体＋気体の共存）：圧力は飽和蒸気圧 $P_S(T)$ で一定
  • $V > V_1$（気体のみ）：boyle-Charles 的な振る舞い

圧力を $P_S$ 以下に下げると（または体積を $V_1$ 以上に増やすと）液体→気体の相転移が起きる。 固体⇔液体の相転移にも類似した議論が成立する。

豆知識相図と三重点

P-T 図（相図）では固体・液体・気体の境界が3本の線として現れ、 3相が共存する三重点（水では $T = 273.16\,\mathrm{K}$，$P = 611.7\,\mathrm{Pa}$）が存在する。 臨界点より高温・高圧では液体と気体の区別がなくなる（超臨界流体）。 CO₂ の超臨界流体はカフェインの抽出（デカフェコーヒー）や半導体洗浄に使われる。

Clausius-Clapeyronの関係式

定理Clausius-Clapeyronの関係式
飽和蒸気圧 $P_S(T)$ の温度依存性： $$\frac{dP_S}{dT} = \frac{S(T,V_1) - S(T,V_0)}{V_1 - V_0} = \frac{Q_{0\to1}}{T(V_1 - V_0)}$$

状態0：相0（例：液体），体積 $V_0$，圧力 $P_S$
状態1：相1（例：気体），体積 $V_1$，圧力 $P_S$
$Q_{0\to1} > 0$：相転移に必要な熱量（気化熱・融解熱など）
$V_1 > V_0$（液体→気体）なので $dP_S/dT > 0$（飽和蒸気圧は温度とともに増大）

歴史Clapeyron（1799–1864）とClausius

Émile Clapeyron はフランスの工学者で、1834 年に Carnot の結果をグラフィカルに表現し 相平衡の方程式を最初に導いた。 Clausius が 1850 年代に熱力学を再定式化する際にこれを厳密化して現在の形になった。 この式は「Clausius-Clapeyron の関係式」と両者の名を冠する。

Clausius-Clapeyron の導出

導出Helmholtz自由エネルギーを使った導出

等温過程 $(V_0, P_S) \to (V_1, P_S)$（液体→気体）で $dF = -P\,dV$，$P = P_S(T)$（一定）より：

$$F(T, V_1) - F(T, V_0) = -P_S(T)(V_1 - V_0) \quad \cdots(*)$$

温度を $T \to T + \Delta T$ にずらした同じ相転移を考える：

$$F(T+\Delta T, V_1) - F(T+\Delta T, V_0) = -(P_S + \tfrac{dP_S}{dT}\Delta T)(V_1 - V_0)$$

左辺を $\left(\frac{\partial F}{\partial T}\right)_V = -S$ を使ってテイラー展開：

$$F(T+\Delta T, V_i) \approx F(T, V_i) - S(T, V_i)\Delta T$$

代入して $(*)$ を差し引くと：

$$-S(T,V_1)\Delta T + S(T,V_0)\Delta T = -\frac{dP_S}{dT}\Delta T(V_1 - V_0)$$ $$\therefore\quad \frac{dP_S}{dT} = \frac{S(T,V_1) - S(T,V_0)}{V_1 - V_0}$$

等温過程で吸収する熱：$Q_{0\to1} = T[S(T,V_1) - S(T,V_0)]$（可逆等温加熱）より：

$$\frac{dP_S}{dT} = \frac{Q_{0\to1}}{T(V_1 - V_0)}$$

Clausius-Clapeyron の近似形と沸点の圧力依存性

液体–気体の相転移では気体の体積が液体より遥かに大きい（$V_1 \gg V_0$）。 さらに気体を理想気体と近似すると，Clausius-Clapeyron 式を積分できる。

定理Clausius-Clapeyron の近似積分形

近似：$V_1 - V_0 \approx V_1 = RT/P_S$（気体を理想気体，$R$ はモル気体定数）を代入：

$$\frac{dP_S}{dT} = \frac{L}{T(V_1 - V_0)} \approx \frac{L P_S}{RT^2}$$

$\dfrac{d\ln P_S}{dT} = \dfrac{L}{RT^2}$ と書き直して積分（$L$ を定数と近似）：

$$\boxed{\ln P_S = -\frac{L}{RT} + \text{const} \implies P_S(T) = P_0\,e^{-L/(RT)}}$$

$L > 0$（気化熱）なので $P_S$ は温度の増加関数。温度が高いほど蒸気圧が指数的に高くなる。

応用高地での沸点の計算

標高 $h$ での大気圧 $P(h) \approx P_0 e^{-mgh/(k_BT)}$（$m$：空気分子の平均質量， $k_B$：Boltzmann 定数）。これが飽和蒸気圧と等しくなる温度が沸点 $T_b(h)$：

$$P_0 e^{-mgh/(k_BT)} = P_0\,e^{-L/(RT_b)} \implies T_b(h) = \frac{L}{mgh/(k_B) + L/T_{b,0}}$$

富士山頂（$h \approx 3776\,\mathrm{m}$）では大気圧 $\approx 63\,\mathrm{kPa}$， 水の沸点 $\approx 87\,^\circ\mathrm{C}$（Clausius-Clapeyron 式から $dT_b/dP_S \approx 0.037\,\mathrm{K/hPa}$）。 インスタントラーメンが山頂でうまく調理できないのはこのため。

Maxwell の等面積則（van der Waals 気体）

van der Waals 気体の $P$–$V$ 等温線には，安定性条件 $(\partial P/\partial V)_T < 0$ を破る 非物理的な領域が生じる。実際には相転移（液化）が起きてこの領域を避けるが， 飽和蒸気圧 $P_S(T)$ の値はMaxwell の等面積則で決まる。

定理Maxwell の等面積則

温度 $T$（臨界温度以下）における液体の体積 $V_0$，気体の体積 $V_1$， 飽和蒸気圧 $P_S$ は次の等面積条件を満たす：

$$\int_{V_0}^{V_1} P_{\rm vdW}(T, V)\,dV = P_S(T)\,(V_1 - V_0)$$

これは「P-V 図で van der Waals 等温曲線と水平線 $P = P_S$ で囲まれた 上側の面積と下側の面積が等しい」という条件で，図的には： $$S_{\rm 上} = S_{\rm 下}$$

導出等面積則の熱力学的根拠

等温・等圧条件での相平衡の条件は，液体と気体のGibbs 自由エネルギーが等しい： $G_{\rm liquid}(T, P_S) = G_{\rm gas}(T, P_S)$。

$G$ の圧力依存性：$(\partial G/\partial P)_T = V$ を使って，基準点 $P^*$ からの積分：

$$G(T, P_S) = G(T, P^*) + \int_{P^*}^{P_S} V(T, P)\,dP$$

等温曲線上では $P = P_{\rm vdW}(T, V)$ なので $dP = (\partial P/\partial V)_T dV$， 変数を $P \to V$ に変換して $G_{\rm liquid} = G_{\rm gas}$ を書き直すと：

$$\int_{V_0}^{V_1}\left[P_{\rm vdW}(T,V) - P_S\right]dV = 0$$

これを整理すると等面積則が得られる。

臨界点の条件と van der Waals 気体の臨界定数

定理臨界点の条件

臨界点では等温曲線の変曲点が安定性の限界点と一致する：

$$\left(\frac{\partial P}{\partial V}\right)_T = 0, \qquad \left(\frac{\partial^2 P}{\partial V^2}\right)_T = 0$$

（臨界点より高温では液体と気体の区別がなくなる。）

計算van der Waals 気体の臨界定数

van der Waals 方程式（1モル）$P = RT/(V-b) - a/V^2$ を使い，臨界点の条件を解く：

$$\left(\frac{\partial P}{\partial V}\right)_T = -\frac{RT}{(V-b)^2} + \frac{2a}{V^3} = 0 \quad \cdots(1)$$ $$\left(\frac{\partial^2 P}{\partial V^2}\right)_T = \frac{2RT}{(V-b)^3} - \frac{6a}{V^4} = 0 \quad \cdots(2)$$

$(2)/(1)$ から：$\dfrac{2}{V-b} = \dfrac{3}{V} \implies V_c = 3b$。

$(1)$ に代入：$T_c = \dfrac{8a}{27Rb}$，$P_c = \dfrac{a}{27b^2}$。

普遍的な比：

$$\frac{P_c V_c}{RT_c} = \frac{(a/27b^2)\cdot 3b}{R \cdot 8a/(27Rb)} = \frac{3}{8} = 0.375$$

実在気体ではこの値が $0.2 \sim 0.3$ 程度となり van der Waals の予測とずれるが， 気体と液体を統一的に記述する現象論的モデルとして有用。

5.2 ゴムの熱力学

問題設定とゴムの特徴

特徴ゴム金属ばね（比較）
弾性率非常に小さい（金属の $\sim 10^{-5}$ 倍）大きい
弾性の温度依存性あり（温めると固くなる）ほぼなし
断熱伸長での温度変化温度が上がる（気体と逆）ほぼなし
弾性の起源エントロピー弾性が主エネルギー弾性が主
定義ゴムの仕事

片側を固定したゴムを伸ばす状況を考える（$dV = 0$ と仮定）：

  • 張力：$X$（スカラー）
  • 自然長からの伸び量：$l$（$l > l_0$）

準静的過程での仕事（気体の $-PdV$ に対応）：

$$\delta W = X\,dl$$

第一法則・第二法則（$dV = 0$ なので $-PdV = 0$）：

$$dU = X\,dl + T\,dS$$
ミクロな理解なぜエントロピー弾性か

ゴムはポリマー（長鎖高分子）が絡み合った構造をもつ。 自然長では鎖がランダムに曲がりくねって「乱雑」な状態（高エントロピー）。 伸ばすと鎖が引き伸ばされ配向し「整列した」状態（低エントロピー）になる。 系はエントロピーを増やす方向（縮む方向）に力を及ぼす → これが弾性力の起源。

これは気体の圧力（自由膨張によりエントロピーが増大する方向に力）と同じ論理。 $PV^\gamma = \text{const}$ の気体断熱曲線に対応するのが ゴムの断熱曲線（後述）。

Helmholtz自由エネルギーとMaxwell関係式

定理ゴムのHelmholtz自由エネルギー
$F(T,l) = U - TS$ の微分：$dF = X\,dl - S\,dT$ より $$X = \left(\frac{\partial F}{\partial l}\right)_T, \quad S = -\left(\frac{\partial F}{\partial T}\right)_l$$

Maxwell 関係式（混合偏微分の可換性）：

$$\left(\frac{\partial X}{\partial T}\right)_l = -\left(\frac{\partial S}{\partial l}\right)_T$$
定理ゴムのエネルギー方程式
$$\left(\frac{\partial U}{\partial l}\right)_T = X - T\left(\frac{\partial X}{\partial T}\right)_l$$

（気体のエネルギー方程式 $(\partial U/\partial V)_T = T(\partial P/\partial T)_V - P$ と対応している。）

導出ゴムのエネルギー方程式

$F = U - TS$ を $l$ で偏微分（$T$ 固定）：

$$\left(\frac{\partial F}{\partial l}\right)_T = \left(\frac{\partial U}{\partial l}\right)_T - T\left(\frac{\partial S}{\partial l}\right)_T$$

左辺 $= X$（定義）。Maxwell 関係式 $(\partial S/\partial l)_T = -(\partial X/\partial T)_l$ を代入：

$$X = \left(\frac{\partial U}{\partial l}\right)_T + T\left(\frac{\partial X}{\partial T}\right)_l$$ $$\therefore\quad \left(\frac{\partial U}{\partial l}\right)_T = X - T\left(\frac{\partial X}{\partial T}\right)_l$$

エネルギー弾性とエントロピー弾性

定義張力の2成分への分解
$$X = \underbrace{\left(\frac{\partial U}{\partial l}\right)_T}_{\text{エネルギー弾性}} + \underbrace{T\left(\frac{\partial X}{\partial T}\right)_l}_{\text{エントロピー弾性}}$$
  • エネルギー弾性：伸ばすと原子間ポテンシャルが増大する成分（金属で支配的）
  • エントロピー弾性：伸ばすとエントロピーが減少するため元に戻ろうとする成分（ゴムで支配的）

$(\partial X/\partial T)_l > 0$（後の例から確認）なので、 温度が上がるほどエントロピー弾性（= 張力）が増す。 これが Gough-Joule 効果の熱力学的起源。

Gough-Joule効果

現象Gough-Joule効果
  • 観測 1（Gough，1805年発見）：一定の荷重（張力一定）をかけたゴムを 加熱すると縮む（冷やすと伸びる）。金属ばねとは逆。
  • 観測 2（Joule，1850年代）：ゴムを断熱状態で急激に伸ばすと温度が上がる （断熱圧縮する気体と逆）。
観測 1 ⟺ 観測 2 は熱力学的に等価であることが示される。
理論観測 1 ⟺ 観測 2 の証明

張力一定でのエントロピー $S(T,l)$ の全微分：

$$dS = \left(\frac{\partial S}{\partial l}\right)_T dl + \left(\frac{\partial S}{\partial T}\right)_l dT = -\left(\frac{\partial X}{\partial T}\right)_l dl + \frac{C_l}{T}dT$$

ここで $C_l = T(\partial S/\partial T)_l$ は長さ一定の熱容量。

断熱過程 $dS = 0$ より：

$$\left(\frac{\partial T}{\partial l}\right)_S = \frac{T}{C_l}\left(\frac{\partial X}{\partial T}\right)_l$$

$T > 0$，$C_l > 0$ なので：

$$\left(\frac{\partial T}{\partial l}\right)_S \gtrless 0 \iff \left(\frac{\partial X}{\partial T}\right)_l \gtrless 0$$
  • 「断熱伸長で温度が上昇」⟺「温度を上げると張力が増大（長さ一定）」
  • ゴムでは $(\partial X/\partial T)_l > 0$（後の例から確認）
  • → 断熱伸長で温度が上昇（観測 2）
  • → 一定の張力のもとで温度を上げるとエントロピー弾性が増し張力が増し、 張力一定なら長さが縮む（観測 1）

5.3 定期試験例題：ゴムの状態方程式

問題設定ゴムの状態方程式
以下の2条件を仮定する：
  1. 状態方程式： $$X(T,l) = AT\left(\frac{l}{l_0} - \left(\frac{l_0}{l}\right)^2\right)$$ （$A > 0$：正の定数，$l_0$：自然長，$l > l_0$）
  2. 長さ一定の熱容量 $C_l$ は温度に依存しない

注：この状態方程式は $X \propto T$（完全にエントロピー弾性）という仮定。 より精密な模型には Gaussian chain model や neo-Hookean model がある。

(1) 内部エネルギーの $l$ 依存性（依存しないことを示せ）

解答 (1)$(\partial U/\partial l)_T = 0$

エネルギー方程式を使う。状態方程式から $(\partial X/\partial T)_l$ を計算：

$$\left(\frac{\partial X}{\partial T}\right)_l = A\left(\frac{l}{l_0} - \frac{l_0^2}{l^2}\right) = \frac{X}{T}$$

エネルギー方程式に代入：

$$\left(\frac{\partial U}{\partial l}\right)_T = X - T\cdot\frac{X}{T} = X - X = 0$$

$U$ は $l$ に依存しない（理想気体で $(\partial U/\partial V)_T = 0$ になるのと同じ構造）。 状態方程式 $X \propto T$ という「エントロピー弾性のみ」という仮定がこれを保証する。

(2) 熱容量の比較 $C_X > C_l$

解答 (2)$C_X > C_l$

(1) より $(\partial U/\partial l)_T = 0$ なので第一法則から：

$$dU = C_l\,dT - X\cdot 0 \implies \delta Q = C_l\,dT - X\,dl$$

等張力条件 $dX = 0$（$X$ 一定）での $dT/dl$ を求める。$dX = (\partial X/\partial T)_l dT + (\partial X/\partial l)_T dl = 0$ から：

$$\left(\frac{\partial l}{\partial T}\right)_X = -\frac{(\partial X/\partial T)_l}{(\partial X/\partial l)_T}$$

$(\partial X/\partial T)_l = X/T > 0$，$(\partial X/\partial l)_T = AT(1/l_0 + 2l_0^2/l^3) > 0$ なので $(\partial l/\partial T)_X < 0$（温めると縮む；Gough-Joule効果の確認）。

$$C_X = \left(\frac{\delta Q}{dT}\right)_X = C_l - X\left(\frac{\partial l}{\partial T}\right)_X = C_l + \frac{X(\partial X/\partial T)_l}{(\partial X/\partial l)_T} > C_l$$

（$X > 0$，$(\partial X/\partial T)_l > 0$，$(\partial X/\partial l)_T > 0$ なので第2項が正。 直観的には定圧熱容量 $> $ 定積熱容量と同じ論理。）

(3) 等温過程 $l_1 \to l_2$（$l_1 < l_2$）での熱の出入り

解答 (3)発熱過程

等温 $dT = 0$ かつ $(\partial U/\partial l)_T = 0$ なので $dU = 0$，$\delta Q = -X\,dl$：

$$\int_{l_1}^{l_2}\delta Q = -\int_{l_1}^{l_2} X\,dl < 0 \quad (\because X > 0,\; l_2 > l_1)$$

→ 発熱過程（ゴムを伸ばすと熱が出る）。

エントロピー的に確認：$(\partial S/\partial l)_T = -(\partial X/\partial T)_l = -X/T < 0$ なので 等温伸長でエントロピーが減少 → 系から熱が出る（$\delta Q = T\,dS < 0$）。整合。

(5) 断熱曲線の方程式

解答 (5)断熱曲線

断熱 $\delta Q = 0$：$C_l\,dT = X\,dl = AT\left(\frac{l}{l_0} - \frac{l_0^2}{l^2}\right)dl$

変数分離：

$$\frac{C_l}{T}dT = A\left(\frac{l}{l_0} - \frac{l_0^2}{l^2}\right)dl$$

両辺を積分：

$$C_l\ln T = A\left(\frac{l^2}{2l_0} + \frac{l_0^2}{l}\right) + \mathrm{const}$$ $$T = T_0'\exp\left[\frac{A}{C_l}\left(\frac{l^2}{2l_0} + \frac{l_0^2}{l}\right)\right]$$

ここで $T_0'$ は積分定数。状態方程式 $X = AT(\cdots)$ に代入すると断熱曲線の $X$-$l$ 表示が得られる。

注：この指数関数型の断熱曲線は理想気体の $T V^{\gamma-1} = \mathrm{const}$（べき乗型）とは 形が異なる。作業物質によって断熱曲線の形は変わる。

全章のまとめ

総括熱力学の論理的構造
  1. 基礎（第1章）：状態量・状態方程式・偏微分の数学
    キー：三変数の偏微分値の関係 $(\partial P/\partial V)_T(\partial V/\partial T)_P(\partial T/\partial P)_V = -1$
  2. 第一法則（第2章）：内部エネルギーと熱・仕事の等価性
    キー：$dU = \delta Q + \delta W$，準静的 $\delta W = -PdV$，Mayer $C_P = C_V + nR$
  3. 第二法則（第3章）：熱と仕事の非等価性→エントロピー
    キー：$\oint \delta Q/T \le 0$，$dS \ge \delta Q/T$，$dU = TdS - PdV$，$\eta_\mathrm{max} = 1 - T_2/T_1$
