📌 数式の記法：不完全微分（経路依存量の微小変化）は $\delta Q$，$\delta W$ と表記。 完全微分（状態量の変化）は $dU$，$dS$ など。 偏微分は $\left(\partial V/\partial T\right)_P$（添字は固定変数）。 状態係数 $\beta$（定圧膨張率），$\kappa$（等温圧縮率），$\alpha_P$（圧力係数）は実験で測定可能な量。

重要公式クイックリファレンス
第一法則・基本関係
第一法則$dU = \delta Q + \delta W$
準静的仕事$\delta W = -P\,dV$
第一法則（準静的）$dU = TdS - PdV$
定積熱容量$C_V = \left(\dfrac{\partial U}{\partial T}\right)_V$
定圧熱容量$C_P = C_V + \left[\left(\dfrac{\partial U}{\partial V}\right)_T + P\right]\left(\dfrac{\partial V}{\partial T}\right)_P$
Mayerの関係（理想気体）$C_P = C_V + nR$
理想気体
状態方程式$PV = nRT$
内部エネルギー$U = C_V T + \mathrm{const}$
断熱曲線$TV^{\gamma-1} = \mathrm{const}$，$PV^\gamma = \mathrm{const}$
比熱比$\gamma = C_P/C_V > 1$
等温仕事$W = -nRT\ln(V_B/V_A)$
断熱仕事$W = \dfrac{nR}{\gamma-1}(T_B - T_A)$
第二法則・エントロピー
Clausiusの不等式$\displaystyle\oint \frac{\delta Q}{T} \le 0$
エントロピー定義$S_B - S_A = \displaystyle\int_{A \to B}^{\text{可逆}} \frac{\delta Q}{T}$
増大則（断熱）$\Delta S \ge 0$
Carnot効率$\eta = 1 - T_2/T_1$
理想気体のエントロピー$S = S_0 + C_V\ln\dfrac{T}{T_0} + nR\ln\dfrac{V}{V_0}$
$T$と$S$の関係$T = \left(\dfrac{\partial U}{\partial S}\right)_V$
熱力学関数
Helmholtz自由エネルギー$F = U - TS$，$dF = -SdT - PdV$
Gibbs自由エネルギー$G = F + PV$，$dG = -SdT + VdP$
エンタルピー$H = U + PV$，$dH = TdS + VdP$
エネルギー方程式$\left(\dfrac{\partial U}{\partial V}\right)_T = T\left(\dfrac{\partial P}{\partial T}\right)_V - P$
Fのmaxwell関係式$\left(\dfrac{\partial P}{\partial T}\right)_V = \left(\dfrac{\partial S}{\partial V}\right)_T$
Gのmaxwell関係式$\left(\dfrac{\partial S}{\partial P}\right)_T = -\left(\dfrac{\partial V}{\partial T}\right)_P$
偏微分の数学的関係
全微分$dz = \left(\dfrac{\partial z}{\partial x}\right)_y dx + \left(\dfrac{\partial z}{\partial y}\right)_x dy$
三変数の関係$\left(\dfrac{\partial x}{\partial y}\right)_z \left(\dfrac{\partial y}{\partial z}\right)_x \left(\dfrac{\partial z}{\partial x}\right)_y = -1$
定圧膨張率$\beta = \dfrac{1}{V}\left(\dfrac{\partial V}{\partial T}\right)_P$
等温圧縮率$K_T = -\dfrac{1}{V}\left(\dfrac{\partial V}{\partial P}\right)_T > 0$
安定性条件・応用
熱的安定性$C_V > 0$，$C_P \ge C_V$
力学的安定性$K_T > 0$ （等温圧縮率が正）
最大仕事の原理等温過程：外に取り出せる仕事 $\le -\Delta F$
Clausius-Clapeyron$\dfrac{dP_S}{dT} = \dfrac{Q_{0\to1}}{T(V_1 - V_0)}$
断熱曲線の一般的性質$\dfrac{C_P}{C_V} = \dfrac{(\partial P/\partial V)_{ad}}{(\partial P/\partial V)_T}$
