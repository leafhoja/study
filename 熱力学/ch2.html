第 2 章

熱力学第一法則

内部エネルギーという状態量を導入し、仕事と熱を合わせたエネルギー保存則（第一法則）を定式化する。 準静的過程に限定することで $\delta W = -PdV$ という仕事の表現が得られ、 定積・定圧熱容量や断熱曲線の理論が展開される。

2.1 状態変化と過程

操作の種類

平衡状態 A から B へ変化させる操作は3種類に分類される：

操作内容例
力学操作 すべて仕事で定量的に表現できる 体積変化（ピストン）、Jouleの羽根車（摩擦熱）、電気ヒーター（Joule熱）
熱的操作 系と熱源を接触させる
何 J 与えたかは直接測れない		 熱源との接触、炎を当てる
質的操作 粒子数を変える 成分の追加・除去（この講義では主に扱わない）

過程の種類

過程操作例
等温過程 力学操作＋熱的操作 熱源につないだままピストンを引く $A(T,V)\to B(T,V')$。
熱源の熱容量が非常に大きい（$\to\infty$）ので温度は一定。
断熱過程 力学操作のみ 断熱材で囲んでピストンを引く $A(T,V)\to B(T',V')$。
温度も変化する。

2.2 熱力学第一法則

力学的エネルギー保存則との類比

保存力のもとでは、その力のする仕事は始状態と終状態のみに依存し途中の経路に依存しない。 これにより位置エネルギーが定義できた：

$$W_{A\to B} = -U_A + U_B \quad (\text{位置エネルギーの定義})$$

熱力学でも同様に、断熱過程のみに限定すれば、 状態 A から B へ変化させる際に外からする仕事は経路によらない （これが第一法則の出発点）。

内部エネルギー

定義内部エネルギー $U$

基準状態 O から可逆断熱過程で状態 X に変化させるとき、外からされる仕事を内部エネルギーという：

$$U_X := W_{O\to X}^{\text{断熱}}$$

系の全エネルギー $U_\text{全} = U_\text{運動} + U_\text{位置} + U_\text{回転} + \cdots + U$ において、巨視的な運動で説明できない残余のエネルギーが $U$（内部エネルギー）である。 ミクロには分子の熱運動エネルギーや分子間ポテンシャルエネルギーなどに対応する。

第一法則の定式化

第一法則断熱過程の場合
断熱過程で状態 A から B へ変化するとき、外からする仕事は途中の経路によらない： $$W_{A\to B}^{\text{断熱}} = U_B - U_A$$

熱的操作も含む一般の場合：A から B へ移るとき、断熱仕事 $\widetilde{W}_{A\to B}$ に加えて 熱 $\widetilde{Q}_{A\to B}$ も加えると：

$$\widetilde{Q}_{A\to B} = (U_B - U_A) - \widetilde{W}_{A\to B}$$
第一法則一般の場合（積分形）
$$U_B - U_A = \widetilde{Q}_{A\to B} + \widetilde{W}_{A\to B}$$

左辺は状態量の差（経路によらない）。 右辺の $\widetilde{Q}, \widetilde{W}$ はそれぞれ経路に依存する過程量であり、 和としての $\widetilde{Q} + \widetilde{W}$ だけが経路によらない値（= $\Delta U$）を与える。

第一法則微小変化の形
A と B が無限小だけ離れているとき： $$dU = \delta Q + \delta W$$

$\delta Q$：系が受け取った熱（不完全微分；経路に依存）
$\delta W$：外から系にされた仕事（不完全微分；経路に依存）
$dU$：内部エネルギーの変化（完全微分；経路によらない）

符号の約束熱力学でよく使う符号規約
  • $\delta Q > 0$：系に熱が入る
  • $\delta W > 0$：外から系に仕事がされる（ピストンが押し込まれる ≒ 体積減少）
  • よって：$\delta W = -P\,dV$（体積増大 $dV>0$ → 系が外に仕事 → $\delta W < 0$）

教科書によっては「系が外にする仕事」を $W$ と定義し $dU = \delta Q - \delta W$ とするものもある。 符号規約が違うだけで物理内容は同じ。注意して使い分けること。

歴史エネルギー保存則の発見

R. Mayer（1814–1878）は 1842 年、「仕事と熱は相互変換可能なエネルギーの2形態だ」と 初めて提唱した（しかし数学的なフォローが不十分だとして最初は無視された）。 J. Joule（1818–1889）は 1843〜50 年代、精巧な羽根車実験で 熱の仕事当量（$1\,\mathrm{cal} \approx 4.186\,\mathrm{J}$）を精密測定し確立した。 Helmholtz（1821–1894）が 1847 年に数学的に一般化して第一法則を確立した。

2.3 準静的過程

定義準静的過程（quasi-static process）
系の状態が常に平衡状態を保ちながらゆっくり変化する理想的な過程。
  • 逆過程が存在する（可逆過程）
  • ゆっくりピストンを引く・押す
  • $T_A$ から $T_B$ に温めるとき、$T_A + n\varepsilon$ の熱源を順次くっつけながら温める
現実には「無限にゆっくり」は不可能だが、理論上の重要な概念。

準静的過程での仕事

公式準静的過程の仕事（気体）
$$\delta W = -P\,dV$$
導出$\delta W = -PdV$ の導出

断面積 $S$ のピストンを考える。外から圧力 $P_\text{ex}$ がかかっているとき、 外から系にかかる力は $F_\text{ex} = P_\text{ex}\cdot S$。 ピストンが微小距離 $dl$（外向きを正）だけ動くと、外からの仕事は：

$$\delta W = F_\text{ex}\cdot(-dl) = -P_\text{ex}\cdot S\cdot dl = -P_\text{ex}\,dV$$

ここで $dV = S\,dl > 0$（体積が増える）。 準静的過程では、常に系の圧力と外部の圧力が釣り合っているので $P_\text{ex} = P$。 よって：

$$\delta W = -P\,dV$$

（非準静的な「急激な膨張」では $P_\text{ex} \ne P$ となり、$\delta W \ne -P\,dV$。）

より一般には（気体以外の系）：

  • ゴム・針金：$\delta W = X\,dl$（$X$：張力，$l$：伸び）
  • 磁性体：$\delta W = \mu_0 H\,dM$（$H$：磁場，$M$：磁化）
  • 表面張力：$\delta W = \sigma\,dA$（$\sigma$：表面張力係数，$A$：表面積）
計算等温仕事（理想気体）

温度 $T$ 一定、体積 $V_A \to V_B$ の準静的等温過程：

$$W_{A\to B}^{\text{等温}} = -\int_{V_A}^{V_B} P\,dV \underset{PV=nRT}{=} -\int_{V_A}^{V_B}\frac{nRT}{V}dV = -nRT\left[\ln V\right]_{V_A}^{V_B} = -nRT\ln\frac{V_B}{V_A} = nRT\ln\frac{V_A}{V_B}$$

$V_B > V_A$（膨張）なら $W < 0$（系が外に仕事をする）。

サイクル過程と第一種永久機関の不在

A → B → ··· → A と元の状態に戻る過程（サイクル）では $\Delta U = 0$ より：

$$Q_\text{サイクル} + W_\text{サイクル} = 0 \implies -W_\text{サイクル} = Q_\text{サイクル}$$

「熱も仕事も与えずに永続的に仕事を取り出す機関（第一種永久機関）は存在しない」 ことを意味する。エネルギー保存則の熱力学的表現である。

2.4 熱容量

内部エネルギー $U(T,V)$ の全微分と第一法則 $dU = \delta Q - P\,dV$ から、 準静的過程での熱は：

$$\delta Q = dU + P\,dV = \left(\frac{\partial U}{\partial T}\right)_V dT + \left(\frac{\partial U}{\partial V}\right)_T dV + P\,dV = \left(\frac{\partial U}{\partial T}\right)_V dT + \left[\left(\frac{\partial U}{\partial V}\right)_T + P\right]dV \quad (\bigstar)$$

① 定積熱容量 $C_V$

定義・公式定積熱容量
$(\bigstar)$ で $dV = 0$（体積一定）を代入： $$C_V = \left(\frac{\delta Q}{dT}\right)_V = \left(\frac{\partial U}{\partial T}\right)_V$$

物理的意味：体積一定のまま温度を 1 K 上げるのに必要な熱量。 体積変化がないので仕事はゼロ、すべての熱が内部エネルギーの増加に使われる。

② 定圧熱容量 $C_P$

定義・公式定圧熱容量
$(\bigstar)$ で $dP = 0$（圧力一定）を適用。 状態方程式から $dV = (\partial V/\partial T)_P\,dT$（定圧での体積変化）を代入： $$C_P = \left(\frac{\delta Q}{dT}\right)_P = \left(\frac{\partial U}{\partial T}\right)_V + \left[\left(\frac{\partial U}{\partial V}\right)_T + P\right]\left(\frac{\partial V}{\partial T}\right)_P$$

第1項は $C_V$ に等しい：

$$\boxed{C_P = C_V + \left[\left(\frac{\partial U}{\partial V}\right)_T + P\right]\left(\frac{\partial V}{\partial T}\right)_P}$$

物理的意味：圧力一定のまま温度を 1 K 上げるのに必要な熱量。 体積変化が伴うため外への仕事 $P\,dV$ が含まれ、$C_P \ge C_V$（一般に成立；第4章で証明）。

③ $C_P - C_V$ の一般式

$(\bigstar)$ で定圧条件 $dV = (\partial V/\partial T)_P\,dT$ を代入すると， $C_P$ の定義から：

$$C_P - C_V = \left[\left(\frac{\partial U}{\partial V}\right)_T + P\right]\left(\frac{\partial V}{\partial T}\right)_P$$
公式$C_P - C_V$ の $\beta$ による表現

$(\partial V/\partial T)_P = V\beta$ を代入すると：

$$C_P - C_V = \left[\left(\frac{\partial U}{\partial V}\right)_T + P\right] V\beta$$

特に理想気体では $(\partial U/\partial V)_T = 0$ なので：

$$C_P - C_V = PV\beta = P \cdot V \cdot \frac{1}{T} = \frac{PV}{T} = nR$$

これがMayerの関係式。一般気体では $(\partial U/\partial V)_T \ne 0$ だが， 第4章のMaxwell関係式と組み合わせると： $$C_P - C_V = \frac{TV\beta^2}{\kappa}$$ となり（$\beta$，$\kappa$ は測定量なので実用的），常に $C_P - C_V \ge 0$ が成立する。

④ 断熱関係式

$(\bigstar)$ の係数 $[(\partial U/\partial V)_T + P]$ を $C_P - C_V$ を使って書き直すと： $$\left(\frac{\partial U}{\partial V}\right)_T + P = \frac{C_P - C_V}{(\partial V/\partial T)_P} = \frac{C_P - C_V}{V\beta}$$ これを $(\bigstar)$ に代入すると：

公式断熱関係式（準静的過程の一般形）
$$\delta Q = C_V\,dT + \frac{C_P - C_V}{V\beta}\,dV \quad (\bigstar\bigstar)$$

断熱過程 $\delta Q = 0$ に適用すると： $$C_V\,dT + \frac{C_P - C_V}{V\beta}\,dV = 0 \quad \text{（断熱関係式）}$$ これは $T$ と $V$ の微分方程式であり，状態方程式と組み合わせて解くと断熱曲線が得られる。

なぜ有用か： $(\partial U/\partial V)_T$ は直接測りにくいが，$C_P$，$C_V$，$\beta$ は実験で測定可能。 $(\bigstar\bigstar)$ はこれらの測定量だけで断熱変化を記述できる形。

⑤ 定圧加熱とエンタルピー

定積加熱（$dV = 0$）では $\delta Q = dU$，つまり熱がすべて内部エネルギーに変わる。 では定圧加熱（$dP = 0$）での熱はどう表されるか？

定義エンタルピー $H$
$$H := U + PV$$

準静的過程で：

$$dH = dU + P\,dV + V\,dP = (\delta Q - P\,dV) + P\,dV + V\,dP = \delta Q + V\,dP$$

定圧過程（$dP = 0$）では：

$$\delta Q\big|_{P=\text{const}} = dH$$

定積加熱での熱 = $dU$（内部エネルギーの変化）に対応するように， 定圧加熱での熱 = $dH$（エンタルピーの変化）。 化学反応・相転移（大気圧下）は多くが定圧条件で起きるため， エンタルピーは実用的に重要な量（第4章で詳述）。

$(\bigstar)$ で $dT = 0$（等温）を代入すると：

$$\delta Q = \left[\left(\frac{\partial U}{\partial V}\right)_T + P\right]dV$$

→ 体積変化は等温でも熱の出入りを伴う（理想気体では $(\partial U/\partial V)_T = 0$ なので別途見る）。

2.5 理想気体の例

理想気体の定義と性質

定義理想気体（ideal gas）
以下の2条件を満たす気体：
  1. 状態方程式：$PV = nRT$（ボイル–シャルルの法則）
  2. $C_V$ が温度に依存しない（定積熱容量が一定）

条件①から次の性質①'が導かれる（証明は下記）： $$\left(\frac{\partial U}{\partial V}\right)_T = 0 \quad \text{（内部エネルギーは体積によらない）}$$ ①と①'より $dU = C_V\,dT$ となり，$C_V$ が定数ならば $U = C_VT + \text{const}$。

実験根拠Joule膨張（断熱自由膨張）実験で $(\partial U/\partial V)_T = 0$ を確認

断熱容器を仕切りで2室に分け、片方に希薄気体を入れ他方を真空にする。 仕切りを取り去ると気体は真空に膨張するが、温度が変わらないことが観測される。

  • 断熱（$\delta Q = 0$）かつ仕事なし（$\delta W = 0$、真空への膨張なので抵抗力ゼロ）
  • よって $dU = 0$
  • また $dT = 0$（実験事実）
  • $dU = C_V\,dT + (\partial U/\partial V)_T\,dV = 0 + (\partial U/\partial V)_T\,dV = 0$ かつ $dV \ne 0$
  • ゆえに $(\partial U/\partial V)_T = 0$（$U$ は $V$ に依存しない）
厳密な証明第二法則から $(\partial U/\partial V)_T = 0$

第三章（第二法則）で示される一般式： $$\left(\frac{\partial U}{\partial V}\right)_T = T\left(\frac{\partial P}{\partial T}\right)_V - P$$ （エネルギー方程式。Maxwell 関係式 $(\partial S/\partial V)_T = (\partial P/\partial T)_V$ から導かれる。）

理想気体 $PV = nRT$ では：

$$\left(\frac{\partial P}{\partial T}\right)_V = \frac{nR}{V} = \frac{P}{T}$$

代入すると：

$$\left(\frac{\partial U}{\partial V}\right)_T = T \cdot \frac{P}{T} - P = P - P = 0 \quad \checkmark$$

これはJoule膨張実験（実験的確認）よりも厳密な理論的証明である。 ここに「第1条件 $PV = nRT$」と「第二法則」だけから①'が導かれ， Joule膨張の実験事実は理論の帰結として理解できる。

Mayerの関係式

定理Mayerの関係式（理想気体）
$$\boxed{C_P = C_V + nR}$$

$nR$ は定数なので $C_P > C_V$（理想気体では常に成立）。

導出Mayerの関係式

$C_P$ の定義式に $(\partial U/\partial V)_T = 0$ を代入：

$$C_P = C_V + \left[0 + P\right]\left(\frac{\partial V}{\partial T}\right)_P$$

$PV = nRT$ から定圧での体積変化：$V = nRT/P \implies (\partial V/\partial T)_P = nR/P$。代入：

$$C_P = C_V + P\cdot\frac{nR}{P} = C_V + nR$$
歴史Mayerの関係式と比熱の測定

R. Mayer（1814–1878）は 1842 年に、気体の等圧比熱と等積比熱の差が 気体定数 $R$ に比例することを理論的に導出した（測定値からも確認）。 これが熱と仕事の等価性（第一法則）の初期の定量的証拠の一つとなった。 Mayer は理論先行型だったのに対し、Joule は実験先行型だった。 両者はほぼ同時期に独立に同じ結論に達したが、Joule の実験精度が高く評価された。

断熱曲線

定義比熱比 $\gamma$
$$\gamma = \frac{C_P}{C_V} = 1 + \frac{nR}{C_V} > 1$$
  • 単原子理想気体（He, Ar など）：$C_V = \frac{3}{2}nR \implies \gamma = \frac{5}{3} \approx 1.67$
  • 二原子理想気体（$\mathrm{N_2}$, $\mathrm{O_2}$ など）：$C_V = \frac{5}{2}nR \implies \gamma = \frac{7}{5} = 1.4$
  • 空気（近似）：$\gamma \approx 1.4$
定理理想気体の断熱曲線
準静的断熱過程（$\delta Q = 0$）では： $$TV^{\gamma-1} = \text{const}, \qquad PV^\gamma = \text{const}$$
導出断熱曲線 $PV^\gamma = \text{const}$ の導出

断熱関係式 $(\bigstar\bigstar)$ を理想気体に適用する。 理想気体では $(\partial U/\partial V)_T = 0$ より $C_P - C_V = PV\beta = P \cdot V \cdot (1/T) = nR$， さらに $\beta = 1/T$（$V$ 一定で）。$(\bigstar\bigstar)$ は： $$0 = C_V\,dT + \frac{nR}{V\cdot(1/T)}\cdot\frac{1}{V}\cdot dV \cdot V = C_V\,dT + \frac{nRT}{V}\,dV$$

また直接でも：$\delta Q = 0$ に $(\partial U/\partial V)_T = 0$，$P = nRT/V$ を使うと同じ式。 変数分離して積分する：

$$C_V\,\frac{dT}{T} = -nR\,\frac{dV}{V}$$ $$C_V\ln T = -nR\ln V + \text{const} \implies T^{C_V} V^{nR} = \text{const}$$

$nR = C_P - C_V$ を代入し $1/C_V$ 乗：

$$T\cdot V^{(C_P - C_V)/C_V} = T\cdot V^{\gamma - 1} = \text{const}$$

$T = PV/(nR)$ を代入して整理：

$$PV^\gamma = \text{const}$$
比較断熱曲線と等温曲線のどちらが急か
  • 等温曲線：$P = nRT/V \implies P \propto 1/V$（傾きの絶対値 $|dP/dV|_T = nRT/V^2 = P/V$）
  • 断熱曲線：$P = \text{const}/V^\gamma \implies |dP/dV|_{ad} = \gamma P/V$
  • $\gamma > 1$ なので断熱曲線の方が急（同じ点から出発した場合）。

直観的理由：断熱膨張では外への仕事で温度も下がるため、等温膨張より圧力が早く落ちる。 グラフでは同じ点を通る断熱曲線が等温曲線の上・左から急激に落ちる形になる。

計算断熱仕事（理想気体）

$PV^\gamma = P_A V_A^\gamma = \text{const}$ を使い $V_A \to V_B$ の断熱仕事を計算：

$$W_{A\to B}^{\text{断熱}} = -\int_{V_A}^{V_B}P\,dV = -\int_{V_A}^{V_B}\frac{P_A V_A^\gamma}{V^\gamma}dV = -P_A V_A^\gamma\cdot\frac{V^{-\gamma+1}}{-\gamma+1}\Bigg|_{V_A}^{V_B}$$ $$= \frac{P_A V_A^\gamma}{{\gamma-1}}\left(V_A^{1-\gamma} - V_B^{1-\gamma}\right) = \frac{P_A V_A - P_B V_B}{\gamma - 1} = \frac{nR(T_A - T_B)}{\gamma - 1}$$

（$P_A V_A^\gamma V_A^{1-\gamma} = P_A V_A = nRT_A$、$P_A V_A^\gamma V_B^{-\gamma+1} = P_B V_B = nRT_B$ を使った。）

断熱曲線の一般的性質（理想気体に限らない）

定理断熱曲線と等温曲線の比の一般式

準静的過程のみを仮定（理想気体に限らない）：

$$\frac{C_P}{C_V} = \frac{\left(\frac{\partial P}{\partial V}\right)_{ad}}{\left(\frac{\partial P}{\partial V}\right)_T}$$

ここで $ad$ は断熱（adiabatic）を意味する。

導出一般の断熱曲線の傾き

$(\bigstar)$ で $\delta Q = 0$ と置き、$dT$ を消去するため $dT = (\partial T/\partial V)_P\,dV + (\partial T/\partial P)_V\,dP$ を代入：

$$0 = C_V\left[\left(\frac{\partial T}{\partial V}\right)_P dV + \left(\frac{\partial T}{\partial P}\right)_V dP\right] + \left[\left(\frac{\partial U}{\partial V}\right)_T + P\right]dV$$

$dP/dV$ 比（断熱での圧力変化率）として整理すると：

$$C_V\left(\frac{\partial P}{\partial V}\right)_{ad} = -C_V\left(\frac{\partial T}{\partial V}\right)_P \cdot\frac{1}{(\partial T/\partial P)_V} - \left[\left(\frac{\partial U}{\partial V}\right)_T + P\right]\cdot\frac{1}{(\partial T/\partial P)_V}$$

$CP - CV = [(\partial U/\partial V)_T + P](\partial V/\partial T)_P$ と三変数の関係式を使うと最終的に：

$$C_V\left(\frac{\partial P}{\partial V}\right)_{ad} = C_P\left(\frac{\partial P}{\partial V}\right)_T \implies \frac{C_P}{C_V} = \frac{(\partial P/\partial V)_{ad}}{(\partial P/\partial V)_T}$$

ポリトロープ過程

等温過程（$PV = \text{const}$）と断熱過程（$PV^\gamma = \text{const}$）を統一する 概念として，ポリトロープ過程（polytropic process）がある。

定義ポリトロープ過程
$$PV^n = \text{const}$$

$n$ をポリトロープ指数という。特殊値での対応：

$n$対応する過程成立する関係
$0$定圧過程（等圧）$P = \text{const}$
$1$等温過程（理想気体）$PV = nRT = \text{const}$
$\gamma$準静的断熱過程$PV^\gamma = \text{const}$，$Q = 0$
$\infty$定積過程（等積）$V = \text{const}$（$PV^n = \text{const}/V^{n-1} \to P = \text{const}$ は誤り）

$n \to \infty$ の場合：$PV^n = C$ と書き，両辺の $n$ 乗根 $(PV^n)^{1/n} = C^{1/n}$ で $V P^{1/n} = C^{1/n}$。$n \to \infty$ で $P^{1/n} \to 1$ なので $V \to \text{const}$。

公式ポリトロープ過程の仕事・熱・温度変化（理想気体）

仕事（外から系にされる仕事，$n \ne 1$ の場合）：

$$W_{A\to B}^{(n)} = -\int_{V_A}^{V_B}P\,dV = \frac{P_A V_A - P_B V_B}{n-1} = \frac{nR(T_A - T_B)}{n-1}$$

（$n = 1$ 等温の場合は $W = -nRT\ln(V_B/V_A)$ と既出の式。）

熱容量（ポリトロープ熱容量 $C_n$）：$\delta Q = C_n\,dT$ と定義すると

$$\boxed{C_n = C_V\cdot\frac{n - \gamma}{n - 1}}$$

従って熱 $Q_{A\to B} = C_n(T_B - T_A) = C_V\dfrac{n-\gamma}{n-1}(T_B - T_A)$。

導出ポリトロープ熱容量 $C_n$

第一法則 $Q = \Delta U - W_{ON}$（ここで $W_{ON}$ は外から系にされる仕事）を使う。

理想気体の $(\partial U/\partial V)_T = 0$ より $\Delta U = C_V(T_B - T_A)$。

ポリトロープ仕事：$W_{ON} = \dfrac{NR(T_A - T_B)}{n-1}$（$N$ はモル数）。

$$Q = C_V(T_B - T_A) - \frac{NR(T_A - T_B)}{n-1} = (T_B - T_A)\!\left(C_V + \frac{NR}{n-1}\right)$$

Mayerの関係式 $NR = C_P - C_V = C_V(\gamma - 1)$ を代入：

$$Q = (T_B - T_A)\cdot C_V\!\left(1 + \frac{\gamma-1}{n-1}\right) = (T_B - T_A)\cdot C_V\cdot\frac{n-\gamma}{n-1}$$

特殊値の確認：

  • $n = 0$（等圧）：$C_0 = C_V\cdot\dfrac{-\gamma}{-1} = \gamma C_V = C_P$ ✓
  • $n = \gamma$（断熱）：$C_\gamma = 0$（熱のやり取りなし） ✓
  • $n \to \infty$（等積）：$C_n \to C_V$ ✓
  • $n \to 1$（等温）：$C_n \to \pm\infty$（温度一定で無限大の熱が流れる） ✓
補足大気の断熱減率

上昇する空気塊は周囲と熱のやり取りが遅く，準断熱的（$n \approx \gamma$）に膨張して冷える。 温度低下率（乾燥断熱減率）を見積もると，状態方程式と $PV^\gamma = \text{const}$ から：

$$\frac{dT}{dz} = -\frac{g(\gamma - 1)}{\gamma}\cdot\frac{m}{k_B} \approx -9.8\,\mathrm{K/km}$$

（$g$：重力加速度，$m$：空気分子の平均質量，$k_B$：Boltzmann定数）

実際の大気の気温減率はこれより小さい（$\sim 6.5\,\mathrm{K/km}$）が、 ポリトロープ過程の近似として有用。
