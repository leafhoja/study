第 4 章

熱力学関数

基本関係式 $dU = TdS - PdV$ を出発点に、 独立変数を $(S,V) \to (T,V) \to (T,P)$ などに変換する ルジャンドル変換によって各種熱力学関数を導入する。 Helmholtz 自由エネルギー $F$・Gibbs 自由エネルギー $G$・エンタルピー $H$ から Maxwell 関係式・エネルギー方程式・最大仕事の原理・熱平衡の安定条件が得られる。

4.1 完全な熱力学関数

定義完全な熱力学関数
与えられた関数から状態方程式と熱容量が両方導き出せるものを 完全な熱力学関数という。
完全な例：$U(S,V)$，$F(T,V)$，$G(T,P)$，$H(S,P)$
不完全な例：$U(T,V)$（状態方程式を完全には出せない；定数項が不定）

$U(S,V)$：完全な熱力学関数

基本関係式と全微分を比較：

$$dU = T\,dS - P\,dV = \left(\frac{\partial U}{\partial S}\right)_V dS + \left(\frac{\partial U}{\partial V}\right)_S dV$$
結果$U(S,V)$ から得られる量
$$T = \left(\frac{\partial U}{\partial S}\right)_V = f(S,V), \quad -P = \left(\frac{\partial U}{\partial V}\right)_S = g(S,V)$$
  • これらから $S$ を消去 → 温度・圧力・体積の状態方程式
  • 定積熱容量：$C_V = \left(\dfrac{\partial U}{\partial T}\right)_V = \left(\dfrac{\partial U}{\partial S}\right)_V \bigg/ \left(\dfrac{\partial^2 U}{\partial S^2}\right)_V$（連鎖律）

$U(T,V)$：不完全な熱力学関数

$C_V = (\partial U/\partial T)_V$ は求まる。 しかし $(\partial U/\partial V)_T$ を状態方程式から求めるにはエネルギー方程式（後述）が必要で、 その際 $(\partial P/\partial T)_V$ が別途必要になり微分方程式に帰着する。 積分定数として定数項が残るため「不完全」とよばれる。

4.2 自由エネルギー

ルジャンドル変換のアイデア

数学ルジャンドル変換（Legendre transform）とは

関数 $f(x)$ のグラフで，$x$ の代わりに「接線の傾き $p = f'(x)$」を独立変数にする変換。 新しい関数 $g(p) = px - f(x)$（$x$ を $p$ で表す）を定義する。

幾何学的意味：$V$ を固定して $U$ を $S$ の関数としてグラフを描く。 点 $S^*$ での接線の傾きが温度 $T^* = (\partial U/\partial S)_V\big|_{S^*}$ である。 その接線を $U$ 軸まで延長したとき，$U$ 軸との切片が $F(T^*, V)$ に等しい：

$$\underbrace{U(S^*, V) - T^*\cdot S^*}_{= F(T^*,\,V)} \;=\; (\text{傾き }T^*\text{ の接線の }U\text{軸切片})$$
読み方：$U(S,V)$ の曲線は，点の集合として見ることも接線の集合として見ることもできる。 後者の見方が Legendre 変換。凸関数なら1対1対応が保証され情報は失われない。 Carnot サイクルで温度 $T$ を指定したとき，自然に現れるポテンシャルが $F(T,V)$。

物理的直観：$U$ は「エントロピー一定（断熱）」条件下での自然な状態関数。 実験では「温度一定（等温）」の条件が多いので，Legendre 変換で $S \to T$ に独立変数を変え $F(T,V)$ を作る。$V \to P$ の変換でさらに $G(T,P)$，$S \to T$ と $V \to P$ の両方で $G(T,P)$。

Helmholtz自由エネルギー $F$

定義Helmholtz自由エネルギー（Helmholtz free energy）
$$F(T,V) = U - TS$$

微分（$dU = TdS - PdV$ を使い $TdS$ が消える）：

$$dF = dU - T\,dS - S\,dT = (T\,dS - P\,dV) - T\,dS - S\,dT = -P\,dV - S\,dT$$

全微分と比較：

$$-S = \left(\frac{\partial F}{\partial T}\right)_V, \quad -P = \left(\frac{\partial F}{\partial V}\right)_T$$
歴史・物理的意味Helmholtz（1821–1894）と「自由」エネルギー

Hermann von Helmholtz はドイツの物理学者・生理学者。エネルギー保存則（1847 年）の数学的定式化、 神経伝達の速度測定、電磁気学への貢献など多方面で活躍した。

「自由」エネルギーとは等温過程で外に取り出せる仕事の最大量（= $-\Delta F$）、 すなわち「仕事として自由に使えるエネルギー」という意味。 内部エネルギー $U$ のうち $TS$（エントロピーに関連する部分）は熱として 外に出ていってしまい仕事にできない「束縛エネルギー」であり、残りの $F = U - TS$ が「自由」な分。

公式$F(T,V)$ から $U$（および $C_V$）を求める

$F = U - TS$ と $(\partial F/\partial T)_V = -S$ から $S$ を消去すると：

$$\boxed{U = F + TS = F - T\!\left(\frac{\partial F}{\partial T}\right)_V}$$

さらに整理すると（積の微分則を使って $\partial/\partial T$ を外に出す）：

$$U = -T^2\!\left(\frac{\partial}{\partial T}\frac{F}{T}\right)_V = \left(\frac{\partial (F/T)}{\partial (1/T)}\right)_V$$

この最後の形（$F/T$ を $1/T$ で偏微分）は統計力学の分配関数 $Z$ を使った $U = -\partial \ln Z/\partial \beta$（$\beta = 1/k_BT$）と対応する重要な表式。

熱容量は：

$$C_V = \left(\frac{\partial U}{\partial T}\right)_V = -T\left(\frac{\partial^2 F}{\partial T^2}\right)_V$$

Maxwell関係式

$F$ の混合偏微分の可換性（Schwarz の定理：$\partial^2 F/\partial T\partial V = \partial^2 F/\partial V\partial T$）より：

Maxwell関係式$F(T,V)$ から
$$-\left(\frac{\partial P}{\partial T}\right)_V = \frac{\partial^2 F}{\partial T\partial V} = -\left(\frac{\partial S}{\partial V}\right)_T$$ $$\therefore \quad \left(\frac{\partial P}{\partial T}\right)_V = \left(\frac{\partial S}{\partial V}\right)_T$$
Maxwell関係式$U(S,V)$ から
$$-\left(\frac{\partial P}{\partial S}\right)_V = \frac{\partial^2 U}{\partial S\partial V} = \left(\frac{\partial T}{\partial V}\right)_S$$
豆知識Maxwell関係式の実験的応用

Maxwell 関係式は「直接測りにくい量を測りやすい量で表す」ために非常に有用。 例えば $(\partial S/\partial V)_T$ は実験的に測りにくいが、 $(\partial P/\partial T)_V$ は等積での圧力温度係数として容易に測定できる。 同様に $(\partial S/\partial P)_T$ は $-(\partial V/\partial T)_P$（定圧膨張率）から求まる。

エネルギー方程式

定理エネルギー方程式
$$\left(\frac{\partial U}{\partial V}\right)_T = T\left(\frac{\partial P}{\partial T}\right)_V - P$$
導出エネルギー方程式の導出

$F = U - TS$ の $V$ による偏微分（$T$ 固定）：

$$\left(\frac{\partial F}{\partial V}\right)_T = \left(\frac{\partial U}{\partial V}\right)_T - T\left(\frac{\partial S}{\partial V}\right)_T$$

左辺 $= -P$（$F$ の定義より）。右辺の $(\partial S/\partial V)_T$ を Maxwell 関係式で置き換え：

$$-P = \left(\frac{\partial U}{\partial V}\right)_T - T\left(\frac{\partial P}{\partial T}\right)_V$$

整理すると：

$$\left(\frac{\partial U}{\partial V}\right)_T = T\left(\frac{\partial P}{\partial T}\right)_V - P$$
確認理想気体・van der Waals 気体への適用

理想気体 $PV = nRT \Rightarrow (\partial P/\partial T)_V = nR/V = P/T$：

$$\left(\frac{\partial U}{\partial V}\right)_T = T\cdot\frac{P}{T} - P = 0 \quad\checkmark$$

van der Waals 気体 $(P + a/V^2)(V - b) = nRT$： $P = nRT/(V-b) - a/V^2$ なので $(\partial P/\partial T)_V = nR/(V-b)$

$$\left(\frac{\partial U}{\partial V}\right)_T = T\cdot\frac{nR}{V-b} - \left(\frac{nRT}{V-b} - \frac{a}{V^2}\right) = \frac{a}{V^2}$$

$a > 0$ なので、van der Waals 気体の内部エネルギーは体積が増えると増加する （分子間引力に逆らって膨張するのでエネルギーが増える。直観的に正しい）。

Gibbs自由エネルギー $G$

定義Gibbs自由エネルギー（Gibbs free energy）
$$G(T,P) = F + PV = U - TS + PV$$

微分（$\pm PV$ の符号は $dV$ が消えるように選ぶ）：

$$dG = dF + P\,dV + V\,dP = (-P\,dV - S\,dT) + P\,dV + V\,dP = -S\,dT + V\,dP$$

よって：

$$-S = \left(\frac{\partial G}{\partial T}\right)_P, \quad V = \left(\frac{\partial G}{\partial P}\right)_T$$

Maxwell 関係式：$-(\partial S/\partial P)_T = (\partial V/\partial T)_P$

歴史・重要性Gibbs（1839–1903）と化学熱力学

Josiah Willard Gibbs はアメリカの物理学者・数学者。 1870〜80 年代にベクトル解析と熱力学（化学ポテンシャル・Gibbs 自由エネルギーなど）を確立した。 その論文はアメリカの地方誌に英語で書かれていたため当初は欧州では知られなかったが、 Maxwell が英国で紹介してから国際的に評価された。

なぜ $G$ が重要か：化学反応は通常「等温・等圧」条件で進む。 自発的に起こる反応では $\Delta G < 0$（Gibbs の自由エネルギーの減少則）。 化学平衡条件は $\Delta G = 0$。 電池の起電力 $\mathcal{E} = -\Delta G / (nF)$ など、化学・生化学・電気化学の基礎。

エンタルピー $H$

定義エンタルピー（enthalpy）
$$H(S, P) = U + PV$$ $$dH = dU + P\,dV + V\,dP = T\,dS + V\,dP$$

$T = (\partial H/\partial S)_P$，$V = (\partial H/\partial P)_S$

物理的意味：定圧過程での熱の出入りは $\delta Q = dH$。 化学反応の反応熱（燃焼熱・融解熱）はエンタルピー変化 $\Delta H$ で表される。

関数定義微分形自然変数Maxwell関係式
$U$（内部エネルギー）— $dU = TdS - PdV$$(S,V)$ $-(\partial P/\partial S)_V = (\partial T/\partial V)_S$
$F = U - TS$（Helmholtz）Legendre（$S \to T$） $dF = -SdT - PdV$$(T,V)$ $(\partial P/\partial T)_V = (\partial S/\partial V)_T$
$G = F + PV$（Gibbs）Legendre（$V \to P$） $dG = -SdT + VdP$$(T,P)$ $-(\partial S/\partial P)_T = (\partial V/\partial T)_P$
$H = U + PV$（エンタルピー）Legendre（$V \to P$） $dH = TdS + VdP$$(S,P)$ $(\partial T/\partial P)_S = (\partial V/\partial S)_P$

Gibbs-Helmholtz 方程式

定理Gibbs-Helmholtz方程式
$$\left(\frac{\partial (G/T)}{\partial T}\right)_P = -\frac{H}{T^2}$$

あるいは同値な形として：

$$H = -T^2\left(\frac{\partial (G/T)}{\partial T}\right)_P = G - T\left(\frac{\partial G}{\partial T}\right)_P$$
導出Gibbs-Helmholtz方程式

$G/T$ を $T$ で偏微分（$P$ 固定）する。積の微分則を使う：

$$\left(\frac{\partial (G/T)}{\partial T}\right)_P = \frac{1}{T}\left(\frac{\partial G}{\partial T}\right)_P + G\cdot\left(-\frac{1}{T^2}\right) = \frac{1}{T}\left(\frac{\partial G}{\partial T}\right)_P - \frac{G}{T^2}$$

$(\partial G/\partial T)_P = -S$ を代入：

$$= -\frac{S}{T} - \frac{G}{T^2} = -\frac{G + TS}{T^2}$$

$G = H - TS \implies G + TS = H$ を代入：

$$\left(\frac{\partial (G/T)}{\partial T}\right)_P = -\frac{H}{T^2}$$
応用化学平衡の温度依存性（van't Hoff の式）

化学反応の平衡定数 $K$ は Gibbs 自由エネルギー変化 $\Delta G$ と $K = e^{-\Delta G/RT}$（$R$：気体定数）で関係づけられる。 Gibbs-Helmholtz 方程式を $\Delta G$ に適用すると：

$$\frac{d\ln K}{d(1/T)} = -\frac{\Delta H}{R}$$

（van't Hoff の式）。発熱反応（$\Delta H < 0$）では温度を上げると $K$ が減少（平衡が逆方向にシフト）。 これはルシャトリエの原理の定量的表現。

4.3 熱力学的変化の方向

Clausius の不等式（微小形）$\delta Q / T \le dS$ と第一法則 $dU = \delta Q + \delta W$ を合わせると：

$$\delta Q = dU - \delta W \le T\,dS \implies dU - T\,dS \le \delta W$$
統一的理解3つの「方向則」— 断熱 / 等温定積 / 等温定圧

上の不等式 $dU - TdS \le \delta W$（$T$ は熱浴の温度）に各条件を代入すると， 自発変化の方向を決める三種類の減少則が得られる：

条件$\delta W$ の形得られる不等式「方向」の意味
断熱（$\delta Q = 0$） 任意 $\Delta S \ge 0$ 孤立系はエントロピーが増える方向へ向かう
等温 + 定積（仕事ゼロ） $\delta W = 0$ $dF \le 0$ $F$ が最小の状態が安定平衡
等温 + 定圧（$P$ 外部圧のみ） $\delta W = -P\,dV$ $dG \le 0$ $G$ が最小の状態が安定平衡（最も実験的に重要）

等温・定積の例：温度 $T$ の熱浴に接した固定容器（2区画，仕切りを動かせる）。 仕切りを解放すると $F_{\rm total} = F(T,V_1) + F(T,V_2)$（$V_1+V_2=V_{\rm total}$ 一定）が 最小の体積分配 $V_1^*, V_2^*$ に落ち着く。

等温・定圧の例：圧力 $P$ のピストン付き容器（気液共存系）。 $G_{\rm 液}(T,P) < G_{\rm 気}(T,P)$ なら液相が熱力学的に安定（相転移の方向を決める；第5章）。

最大仕事の原理（等温過程）

定理最大仕事の原理（Maximum Work Theorem）
等温過程（$dT = 0$、系の温度 $=$ 熱源温度 $T$）では： $$d(U - TS) \le \delta W \implies dF \le \delta W$$ 外にする仕事（$= -\delta W$）： $$-\delta W \le -dF$$

等温過程で外に取り出せる仕事の最大量は Helmholtz 自由エネルギーの減少分 $-\Delta F$ である。 等号成立（最大値）：可逆過程。

注：熱源から熱をもらって仕事をするが、Kelvin の原理に反しない。 なぜなら Kelvin の原理は「1つの熱源から 100% 仕事変換」を禁じているが、 等温過程では低温熱源に $T\Delta S > 0$ の熱を捨てているため（一般に $\Delta F < \Delta U$）。

各条件での減少則

条件成立する不等式物理的意味
断熱（$\delta Q = 0$） $\Delta S \ge 0$ エントロピー増大則（孤立系が向かう方向）
等温＋定積（$\delta W = 0$） $dF \le 0$ Helmholtz 自由エネルギーの減少則（化学反応の自発性）
等温＋定圧（$\delta W = -P\,dV$） $dG \le 0$ Gibbs 自由エネルギーの減少則（最も実験的に重要）
導出等温＋定圧での $dG \le 0$

等温定圧では $\delta W = -P\,dV$（体積変化のみが仕事）なので：

$$dF \le \delta W = -P\,dV$$ $$d(F + PV) = dF + P\,dV + V\,dP \le 0 + V\,dP$$

定圧（$dP = 0$）なので右辺 $= 0$：

$$dG \le 0$$

4.4 熱平衡の安定条件

安定性の2次条件

温度 $T^*$ の熱源に接し、圧力 $P^*$ で押されている系の平衡状態 A を考える。 A からわずかにずれた仮想状態 A' を取り、$\delta U - T^*\delta S + P^*\delta V > 0$ （A に戻ろうとする力が働く条件）を展開すると、 一次の項から $T = T^*$，$P = P^*$（一次条件）が得られ、 二次の項から安定性の実質的な条件が出る。

導出安定性の2次条件の計算（4ステップ）

Step 1：$\delta U$ の2次までのテイラー展開（$(\delta S, \delta V)$ を独立変数）：

$$2A \;=\; \left(\frac{\partial^2 U}{\partial S^2}\right)_V\!(\delta S)^2 + 2\frac{\partial^2 U}{\partial S\partial V}\delta S\,\delta V + \left(\frac{\partial^2 U}{\partial V^2}\right)_S\!(\delta V)^2$$

Step 2：$T = (\partial U/\partial S)_V$，$P = -(\partial U/\partial V)_S$ なので， $\delta T$ と $\delta P$ を $(\delta S, \delta V)$ で表すと：

$$\delta T = \frac{\partial^2 U}{\partial S^2}\,\delta S + \frac{\partial^2 U}{\partial S\partial V}\,\delta V, \qquad \delta P = \frac{\partial^2 U}{\partial S\partial V}\,\delta S + \frac{\partial^2 U}{\partial V^2}\,\delta V$$

これより：

$$\delta T\,\delta S - \delta P\,\delta V \;=\; \frac{\partial^2 U}{\partial S^2}(\delta S)^2 + 2\frac{\partial^2 U}{\partial S\partial V}\delta S\,\delta V + \frac{\partial^2 U}{\partial V^2}(\delta V)^2 \;=\; 2A$$

つまり：

$$\boxed{2A = \delta T\,\delta S - \delta P\,\delta V}$$

Step 3：独立変数を $(\delta S,\delta V)$ から $(\delta T, \delta V)$ に変更する。 $\delta S = \left(\partial S/\partial T\right)_V\delta T + \left(\partial S/\partial V\right)_T \delta V$ および $\delta P = \left(\partial P/\partial T\right)_V\delta T + \left(\partial P/\partial V\right)_T \delta V$ を代入すると：

$$2A \;=\; \left(\frac{\partial S}{\partial T}\right)_V\!(\delta T)^2 + \left[\left(\frac{\partial S}{\partial V}\right)_T - \left(\frac{\partial P}{\partial T}\right)_V\right]\!\delta T\,\delta V - \left(\frac{\partial P}{\partial V}\right)_T\!(\delta V)^2$$

Step 4：Maxwell 関係式（$dF = -SdT - PdV$ より）：

$$\left(\frac{\partial S}{\partial V}\right)_T = \left(\frac{\partial P}{\partial T}\right)_V$$

これを代入すると交差項が消え：

$$\boxed{2A \;=\; \left(\frac{\partial S}{\partial T}\right)_V\!(\delta T)^2 - \left(\frac{\partial P}{\partial V}\right)_T\!(\delta V)^2}$$

$\delta T$ と $\delta V$ が独立で任意なので，$2A > 0$ の必要十分条件は：

$$\left(\frac{\partial S}{\partial T}\right)_V > 0 \quad\text{かつ}\quad -\left(\frac{\partial P}{\partial V}\right)_T > 0$$

安定性条件：$C_V > 0$ と $K_T > 0$

定理熱的・力学的安定性
  • 熱的安定性： $\left(\frac{\partial S}{\partial T}\right)_V > 0 \iff C_V = T\left(\frac{\partial S}{\partial T}\right)_V > 0$
    （温度を上げるとエントロピーが増える → 加熱で温度が上がる）
  • 力学的安定性： $-\left(\frac{\partial P}{\partial V}\right)_T > 0 \iff K_T = -\frac{1}{V}\left(\frac{\partial V}{\partial P}\right)_T > 0$
    （等温圧縮率 $K_T > 0$：圧力を増やせば体積が減る）

これらが破れるとき（例：$C_V < 0$ や $K_T < 0$ の領域）、その状態は不安定で実現しない。

$C_P \ge C_V$ の証明

定理$C_P \ge C_V$
$$C_P - C_V = -T\left(\frac{\partial V}{\partial T}\right)_P^2 \left(\frac{\partial P}{\partial V}\right)_T \ge 0$$

（安定性より $(\partial P/\partial V)_T < 0$ なので右辺 $\ge 0$；等号は $(\partial V/\partial T)_P = 0$ のとき）

導出$C_P - C_V$ の公式

第2章の結果：

$$C_P = C_V + \left[\left(\frac{\partial U}{\partial V}\right)_T + P\right]\left(\frac{\partial V}{\partial T}\right)_P$$

エネルギー方程式 $(\partial U/\partial V)_T = T(\partial P/\partial T)_V - P$ を代入：

$$C_P - C_V = T\left(\frac{\partial P}{\partial T}\right)_V\left(\frac{\partial V}{\partial T}\right)_P$$

三変数の関係式 $(\partial P/\partial T)_V = -(\partial P/\partial V)_T(\partial V/\partial T)_P$ を代入：

$$C_P - C_V = -T\left(\frac{\partial V}{\partial T}\right)_P^2\left(\frac{\partial P}{\partial V}\right)_T \ge 0$$

$C_P - C_V$ の $\beta$・$\kappa_T$ 表示

定理$C_P - C_V = TV\beta^2/\kappa_T$
$$\boxed{C_P - C_V = \frac{TV\beta^2}{\kappa_T}}$$

$\beta$（定圧膨張率），$\kappa_T$（等温圧縮率），$V$（体積），$T$（温度）で表される完全に測定可能な量。

導出$C_P - C_V$ の $\beta,\kappa_T$ 表示

直前の式から：

$$C_P - C_V = -T\left(\frac{\partial V}{\partial T}\right)_P^2\left(\frac{\partial P}{\partial V}\right)_T$$

定圧膨張率と等温圧縮率の定義（第1章）を使って各因子を置き換える：

$$\left(\frac{\partial V}{\partial T}\right)_P = V\beta, \qquad \left(\frac{\partial P}{\partial V}\right)_T = -\frac{1}{V\kappa_T}$$

代入すると：

$$C_P - C_V = -T(V\beta)^2\cdot\left(-\frac{1}{V\kappa_T}\right) = \frac{TV^2\beta^2}{V\kappa_T} = \frac{TV\beta^2}{\kappa_T}$$

安定性条件 $\kappa_T > 0$ より右辺 $\ge 0$（等号は $\beta = 0$ のとき，例：4℃ の水）。 これは $C_P \ge C_V$ の一般的な表現であり，理想気体への適用： $\beta = 1/T$，$\kappa_T = 1/P = V/(nRT)$ から $C_P - C_V = T \cdot V \cdot (1/T)^2 \cdot nRT/V = nR$ となり，Mayer の関係式と一致。

断熱・等温圧縮率の関係 $K_T/K_{ad} = C_P/C_V$

定理断熱・等温圧縮率の比較
$$\frac{K_T}{K_{ad}} = \frac{C_P}{C_V} = \gamma > 1$$

断熱圧縮率 $K_{ad} < K_T$：断熱では圧力を増やしても体積が縮みにくい （断熱圧縮で温度も上昇し、圧力が余分に上がるため）。

証明$K_T/K_{ad} = C_P/C_V$

定義より：

$$K_T = -\frac{1}{V}\left(\frac{\partial V}{\partial P}\right)_T = \frac{-1}{V(\partial P/\partial V)_T}, \qquad K_{ad} = -\frac{1}{V}\left(\frac{\partial V}{\partial P}\right)_S = \frac{-1}{V(\partial P/\partial V)_S}$$

比をとると：

$$\frac{K_T}{K_{ad}} = \frac{(\partial P/\partial V)_S}{(\partial P/\partial V)_T}$$

第2章の断熱曲線の一般式（一般の物質に対して成立）では：

$$\frac{C_P}{C_V} = \frac{(\partial P/\partial V)_{ad}}{(\partial P/\partial V)_T} = \frac{(\partial P/\partial V)_S}{(\partial P/\partial V)_T}$$

（準静的断熱 = 等エントロピー（$dS = 0$），よって subscript $ad$ と $S$ は同じ。）よって：

$$\frac{K_T}{K_{ad}} = \frac{C_P}{C_V}$$

理想気体での確認：等温圧縮率 $K_T = 1/P$，断熱曲線 $PV^\gamma = C$ から $(\partial P/\partial V)_S = -\gamma P/V$ なので $K_{ad} = 1/(\gamma P) = K_T/\gamma$。 よって $K_T/K_{ad} = \gamma = C_P/C_V$。$\checkmark$

van der Waals 状態方程式と安定性

例van der Waals 気体
$$\left(P + \frac{a}{V^2}\right)(V - b) = nRT$$

$a > 0$：分子間引力の補正（分子が互いに引き合うので有効圧力が上昇）
$b > 0$：分子自身の体積の補正（分子が占めるので有効体積が減少）

これは J. van der Waals（1837–1923）が 1873 年に提案した現象論的モデル。 気体と液体の連続性を初めて説明し、ノーベル物理学賞（1910 年）を受賞。

低温でのvan der Waals 気体の $P$-$V$ 曲線には $(\partial P/\partial V)_T > 0$ の領域が現れる。 これは安定性条件 $K_T > 0$ に反するため熱力学的に実現不可能な状態。

重要現実の気体では相転移が起きる

不安定な領域に対応する温度・圧力では気体が液化し2相共存状態になる（第5章）。 この領域では $P = P_S(T)$（飽和蒸気圧）で水平線になり $(\partial P/\partial V)_T = 0$ となる。

気体と「気体＆液体の共存」の境界では微分値が不連続になるため、 Maxwell 関係式（$\partial^2 F / \partial T\partial V$ の連続性を仮定）は成立しない点に注意。
