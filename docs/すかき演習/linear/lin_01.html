線形代数学演習　第1回

線形写像の定義・行列表現・写像の合成・単射・全射

📋 この回のテーマ

線形代数の核心は「線形写像」だ。行列の計算ではなく，写像としての構造を理解することが目標。行列は線形写像を記述するための道具に過ぎない。

定義：線形写像

$\mathbb{R}$-ベクトル空間 $V, W$ の間の写像 $f : V \to W$ が線形写像（$\mathbb{R}$-線形写像）であるとは：

  • 加法性：任意の $v_1, v_2 \in V$ に対して $f(v_1 + v_2) = f(v_1) + f(v_2)$
  • スカラー倍性：任意の $v \in V$，$\alpha \in \mathbb{R}$ に対して $f(\alpha v) = \alpha f(v)$

2条件は $f(av_1 + bv_2) = af(v_1) + bf(v_2)$（線形結合を保存する）という1条件にまとめられる。

💡 線形写像の基本性質（必ず成立）

$f$ が線形写像ならば $f(\mathbf{0}) = \mathbf{0}$（ゼロベクトルはゼロベクトルに写る）。

証明：$f(\mathbf{0}) = f(0 \cdot \mathbf{0}) = 0 \cdot f(\mathbf{0}) = \mathbf{0}$。

この性質を使うと「$f(\mathbf{0}) \ne \mathbf{0}$ ならば $f$ は線形写像でない」という反例の作り方がわかる。

1 $f_A$ が線形写像であること，写像の等しさ

問題文

$A = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$，$f_A : \mathbb{R}^2 \to \mathbb{R}^2$ を $f_A(x) = Ax$ で定義する。

  1. $f_A$ が線形写像であることを示せ。
  2. 線形写像 $f : \mathbb{R}^2 \to \mathbb{R}^2$ が $f((1,0)^T) = (a, c)^T$，$f((0,1)^T) = (b,d)^T$ を満たすとき，$f = f_A$ であることを示せ。
📝 模範解答

(1) $f_A$ が線形写像であることの証明

任意の $x = (i,j)^T$，$y = (k,l)^T \in \mathbb{R}^2$，$\alpha \in \mathbb{R}$ に対して：

加法性：$x + y = (i+k, j+l)^T$ なので $$f_A(x+y) = A(x+y) = \begin{pmatrix}a(i+k)+b(j+l)\\c(i+k)+d(j+l)\end{pmatrix} = \begin{pmatrix}ai+bj\\ci+dj\end{pmatrix} + \begin{pmatrix}ak+bl\\ck+dl\end{pmatrix} = f_A(x) + f_A(y)$$

スカラー倍性：$\alpha x = (\alpha i, \alpha j)^T$ なので $f_A(\alpha x) = A(\alpha x) = \alpha Ax = \alpha f_A(x)$。

よって $f_A$ は線形写像。$\blacksquare$

(2) $f = f_A$ の証明

任意の $(x,y)^T \in \mathbb{R}^2$ を固定する。$(x,y)^T = x(1,0)^T + y(0,1)^T$ と書けるから $f$ の線形性より

$$f\binom{x}{y} = xf\binom{1}{0} + yf\binom{0}{1} = x\binom{a}{c} + y\binom{b}{d} = \binom{ax+by}{cx+dy} = A\binom{x}{y} = f_A\binom{x}{y}$$

$(x,y)^T$ は任意だったから $f = f_A$。$\blacksquare$

🧠 思考法・解説

(1) について：行列とベクトルの積の定義から直接確認する。行列積は分配則・スカラー倍を満たすように定義されているので，ほぼ計算するだけで成立する。

🔄 線形写像は基底での値で決まる

(2) の着想：任意の $(x, y)^T \in \mathbb{R}^2$ は標準基底で分解できる：$(x, y)^T = x(1,0)^T + y(0,1)^T$。

$f$ の線形性を使うと：$f((x,y)^T) = xf((1,0)^T) + yf((0,1)^T)$。

仮定より $f((1,0)^T) = (a,c)^T$，$f((0,1)^T) = (b,d)^T$ だから値が完全に決まる。

💡 写像の等しさの定義

写像 $f, g : V \to W$ が等しい（$f = g$）とは： 定義域と値域が等しく，かつ任意の $v \in V$ に対して $f(v) = g(v)$。

$\mathbb{R}^2$ の場合，任意の $(x,y)^T$ で値が一致すること。

💡 重要な一般原理：線形写像は基底での値で一意に決まる

$\mathbb{R}^n$ から $\mathbb{R}^m$ への任意の線形写像 $f$ は，$n$ 本の基底ベクトルの行き先を指定することで一意に決まる。この基底ベクトルの行き先を並べたものが表現行列だ。

$f(e_j) = $ （行列の第 $j$ 列）という対応関係が成立する。

2 線形写像の表現行列を求める

問題文

線形写像 $f : \mathbb{R}^2 \to \mathbb{R}^2$ が $f((2,1)^T) = (1,2)^T$，$f((3,4)^T) = (1,1)^T$ を満たすとき，$f(x) = Ax$ となる $2\times 2$ 行列 $A$ を求めよ。

📝 模範解答

$A = \begin{pmatrix}a&b\\c&d\end{pmatrix}$ とおく。$f_A((2,1)^T) = (2a+b, 2c+d)^T = (1,2)^T$ より $2a+b=1, 2c+d=2$。$f_A((3,4)^T) = (3a+4b, 3c+4d)^T = (1,1)^T$ より $3a+4b=1, 3c+4d=1$。

第一組：$2a+b=1$, $3a+4b=1$。前者の 4 倍から後者を引くと $5a = 3$，$a = 3/5$，$b = 1 - 2(3/5) = -1/5$。

第二組：$2c+d=2$，$3c+4d=1$。同様に $5c = 7$，$c = 7/5$，$d = 2 - 2(7/5) = -4/5$。

$$A = \frac{1}{5}\begin{pmatrix}3 & -1 \\ 7 & -4\end{pmatrix} \quad \blacksquare$$
🧠 思考法・解説
🔄 解法の着想

問題1(2) より，$A$ の第 $j$ 列は $f$ の標準基底での値。$(2,1)^T$ と $(3,4)^T$ で値がわかっているので，まず $f(e_1)$ と $f(e_2)$ を求める。

$(2,1)^T$ と $(3,4)^T$ を標準基底で表す連立方程式：$\alpha(2,1)^T + \beta(3,4)^T = (1,0)^T$ を解いて $f(1,0)^T = \alpha f(2,1)^T + \beta f(3,4)^T$。

あるいは今回のように，$A$ の成分を未知数として条件式を立てて連立方程式を解く方法も素直で有効。

3 写像の合成と行列の積

問題文

$A, B$ がともに $2\times 2$ 行列で，$f_A, f_B : \mathbb{R}^2 \to \mathbb{R}^2$ を対応する線形写像とする。$f_A \circ f_B = f_{AB}$ を示せ。

📝 模範解答

任意の $x \in \mathbb{R}^2$ に対して

$$(f_A \circ f_B)(x) = f_A(f_B(x)) = f_A(Bx) = A(Bx) = (AB)x = f_{AB}(x)$$

定義域・値域ともに $\mathbb{R}^2$ で一致しており，任意の $x$ で値が等しいから $f_A \circ f_B = f_{AB}$。$\blacksquare$

🧠 思考法・解説
💡 写像の合成と行列の積の「一致」

$f_A \circ f_B$ は「先に $f_B$，次に $f_A$」という操作。行列の積 $AB$ も「先に $B$ を掛け，次に $A$ を掛ける」操作だ。これらが一致することを示す。

注意：順番！写像の合成では $(f_A \circ f_B)(x) = f_A(f_B(x))$。行列積は右から掛ける。$(AB)x = A(Bx)$。

証明の流れは非常にシンプルで，結合律 $A(Bx) = (AB)x$ を使うだけ。これは行列積の定義から従う。

4 単射・全射と $\det A \ne 0$

問題文

$f_A : \mathbb{R}^2 \to \mathbb{R}^2$ について次を示せ。

  1. $f_A$ が単射または全射であるための必要十分条件は $\det A \ne 0$（$= ad - bc \ne 0$）。
  2. $f_A : \mathbb{Z}^2 \to \mathbb{Z}^2$ が全単射（全射かつ単射）であるための必要十分条件。
📝 模範解答

(1) 単射 $\Leftrightarrow$ $ad - bc \ne 0$

（$\Leftarrow$）$ad-bc \ne 0$ ならば単射：

$f_A\binom{w}{x} = f_A\binom{y}{z}$ を仮定。$aw+bx = ay+bz$，$cw+dx = cy+dz$。

第1式を $d$ 倍し第2式を $b$ 倍して引くと $(ad-bc)w = (ad-bc)y$。$ad-bc \ne 0$ より $w = y$。同様に $x = z$。よって単射。

（$\Rightarrow$）$ad-bc = 0$ ならば単射でない：

$f_A$ が単射でないことを示す（$f_A(v) = 0$ を $v = 0$ 以外で解く）。$ad = bc$ のとき，$A\binom{-b}{a} = \binom{-ab+ba}{-cb+da} = \binom{0}{ad-bc} = \binom{0}{0}$（$ad=bc$ を用いた）。$a, b$ が両方 $0$ でなければ $(-b,a)^T \ne 0$ は核の元。一方 $a = b = 0$ なら $A\binom{d}{-c} = \binom{0 \cdot d + 0 \cdot(-c)}{cd + d(-c)} = \binom{0}{0}$ であり，$(c,d) \ne (0,0)$ のとき $(d,-c)^T \ne 0$ が核の元（$c = d = 0$ なら $A = O$ なので任意の非零ベクトルが核の元）。よって単射でない。

全射の証明も同様：任意の $(\alpha,\beta)^T$ に対して $ax+by = \alpha, cx+dy = \beta$ の解の存在性が $ad-bc \ne 0$ と等価（Cramer の公式）。

(2) $f_A : \mathbb{Z}^2 \to \mathbb{Z}^2$ が全単射の条件

まず $a, b, c, d \in \mathbb{Z}$ が必要（$(1,0)^T$ や $(0,1)^T$ の行き先が $\mathbb{Z}^2$ に入るため）。

全射 $\Leftrightarrow$ 任意の $(\alpha,\beta)^T \in \mathbb{Z}^2$ に対して整数解 $x, y$ が存在する $\Leftrightarrow$ Cramer の公式で $x, y \in \mathbb{Z}$ が保証される $\Leftrightarrow$ $\det A = \pm 1$。

まとめ：$f_A : \mathbb{Z}^2 \to \mathbb{Z}^2$ が全単射 $\Leftrightarrow$ $a,b,c,d \in \mathbb{Z}$ かつ $\det A = \pm 1$。$\blacksquare$

🧠 思考法・解説
定義：単射・全射・全単射
  • 単射（injective）：$f(v_1) = f(v_2) \Rightarrow v_1 = v_2$（異なる入力からは必ず異なる出力）
  • 全射（surjective）：任意の $w \in W$ に対してある $v \in V$ があって $f(v) = w$（値域が $W$ 全体）
  • 全単射（bijective）：単射かつ全射（逆写像が存在）
🔄 単射の証明の標準的手順

$f_A(v_1) = f_A(v_2)$ を仮定して $v_1 = v_2$ を示す（または $f_A(v) = 0 \Rightarrow v = 0$，核が自明であることを示す同値な方法）。

逆向き（$\det A = 0$ ならば単射でない）は，核に非自明な元を具体的に構成して見せる。

💡 整数係数行列の逆行列が整数係数になる条件

逆行列の公式 $A^{-1} = \frac{1}{\det A}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}$ において，$\det A$ が整数のとき逆行列の係数が整数になるには $\det A = \pm 1$ が必要十分。

これは整数論の言葉では $A \in \mathrm{GL}(2,\mathbb{Z})$ という条件に対応する。

定理（$n$ 次正方行列の場合）

$n$ 次正方行列 $A$ に対し，次の3条件は同値：

  1. $\det A \ne 0$
  2. $f_A : \mathbb{R}^n \to \mathbb{R}^n$ が全射
  3. $f_A : \mathbb{R}^n \to \mathbb{R}^n$ が単射（よって全単射）

有限次元の場合，単射 $\Leftrightarrow$ 全射（次元定理による）。
