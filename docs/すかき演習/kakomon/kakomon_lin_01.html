1 表現行列・核・像

問題文

線形写像 $f \colon \mathbb{R}^2 \to \mathbb{R}^2$ は

$$f\!\left(\begin{pmatrix}2\\1\end{pmatrix}\right)=\begin{pmatrix}3\\2\end{pmatrix},\quad f\!\left(\begin{pmatrix}1\\-2\end{pmatrix}\right)=\begin{pmatrix}6\\4\end{pmatrix}$$

を満たすとする。

  1. $f = f_A$（左から行列 $A$ を掛ける線形写像）と表すときの $2 \times 2$ 行列 $A$ を求めよ。
  2. $\mathrm{Ker}(f)$ と $\mathrm{Im}(f)$ の基底をそれぞれ求めよ。
📝 解答 (1) — 表現行列 A

$f_A(\mathbf{v}) = A\mathbf{v}$ なので，与えられた条件は行列方程式

$$A\underbrace{\begin{pmatrix}2&1\\1&-2\end{pmatrix}}_{=:P} = \underbrace{\begin{pmatrix}3&6\\2&4\end{pmatrix}}_{=:Q}$$

と書ける（$P$ の列が入力ベクトル，$Q$ の列が対応する出力ベクトル）。

$\det(P) = 2\cdot(-2) - 1\cdot 1 = -5 \ne 0$ なので $P$ は正則であり，

$$P^{-1} = \frac{1}{-5}\begin{pmatrix}-2&-1\\-1&2\end{pmatrix} = \begin{pmatrix}\tfrac{2}{5}&\tfrac{1}{5}\\[4pt]\tfrac{1}{5}&-\tfrac{2}{5}\end{pmatrix}$$ $$A = QP^{-1} = \begin{pmatrix}3&6\\2&4\end{pmatrix}\begin{pmatrix}\tfrac{2}{5}&\tfrac{1}{5}\\[4pt]\tfrac{1}{5}&-\tfrac{2}{5}\end{pmatrix} = \begin{pmatrix}\tfrac{6}{5}+\tfrac{6}{5}&\tfrac{3}{5}-\tfrac{12}{5}\\[4pt]\tfrac{4}{5}+\tfrac{4}{5}&\tfrac{2}{5}-\tfrac{8}{5}\end{pmatrix} = \frac{1}{5}\begin{pmatrix}12&-9\\8&-6\end{pmatrix}$$

確認：$A\begin{pmatrix}2\\1\end{pmatrix}=\frac{1}{5}\begin{pmatrix}24-9\\16-6\end{pmatrix}=\begin{pmatrix}3\\2\end{pmatrix}$ ✓，　$A\begin{pmatrix}1\\-2\end{pmatrix}=\frac{1}{5}\begin{pmatrix}12+18\\8+12\end{pmatrix}=\begin{pmatrix}6\\4\end{pmatrix}$ ✓

📝 解答 (2) — 核と像の基底

まず $\det(A) = \frac{1}{25}(12\cdot(-6) - (-9)\cdot 8) = \frac{-72+72}{25} = 0$ なので $A$ は退化行列（階数 1）。

核 $\mathrm{Ker}(f)$：$A\mathbf{v} = \mathbf{0}$ を解く。

$$\frac{1}{5}\begin{pmatrix}12&-9\\8&-6\end{pmatrix}\begin{pmatrix}x\\y\end{pmatrix}=\begin{pmatrix}0\\0\end{pmatrix}\;\Longleftrightarrow\; 4x = 3y$$ $$\mathrm{Ker}(f) = \mathrm{span}\!\left\{\begin{pmatrix}3\\4\end{pmatrix}\right\}$$

確認：$A\begin{pmatrix}3\\4\end{pmatrix}=\frac{1}{5}\begin{pmatrix}36-36\\24-24\end{pmatrix}=\mathbf{0}$ ✓

像 $\mathrm{Im}(f)$：$A$ の列ベクトルをみると，第1列 $\frac{1}{5}(12,8)^\top \propto (3,2)^\top$，第2列 $\frac{1}{5}(-9,-6)^\top \propto (3,2)^\top$（同じ方向）。

$$\mathrm{Im}(f) = \mathrm{span}\!\left\{\begin{pmatrix}3\\2\end{pmatrix}\right\}$$

（次元定理の確認：$\dim\mathrm{Ker}(f) + \dim\mathrm{Im}(f) = 1 + 1 = 2 = \dim\mathbb{R}^2$ ✓）

🧠 思考の出発点

「行列を求めよ」は方程式で立てる。$f_A(\mathbf{v}) = A\mathbf{v}$ なので，$n$ 個の入出力ペアがあれば，列を横並びにして $A \cdot [\mathbf{v}_1 \cdots \mathbf{v}_n] = [\mathbf{w}_1 \cdots \mathbf{w}_n]$ という行列方程式が作れる。入力行列が正則なら $A = QP^{-1}$ で一意に決まる。

$\det(A)=0$ のとき，核は自明でない。$\mathrm{rank}(A)=1$ なら次元定理より $\dim\mathrm{Ker}=1$。Im は A の列空間（全列が同じ方向を向いているので1次元）。

2 直交行列と冪 $A^n = E$

問題文

$n$ を正の整数とする。実数係数 $2\times 2$ 行列 $A$ が

$${}^t\!A\,A = A\,{}^t\!A = E,\qquad A^n = E$$

を満たすとする（ここで $E$ は単位行列）。

  1. $n$ が奇数のとき，$A$ は回転行列であることを示せ。
  2. 各 $n$ に対し，条件を満たす行列 $A$ を全て求めよ。
2×2 実直交行列の分類

${}^t\!A\,A = E$ を満たす実 $2\times 2$ 行列（直交行列）は $\det A = \pm 1$ を満たし，次の2種類のみ存在する。

  • 回転行列（$\det A = 1$）：$R(\theta) = \begin{pmatrix}\cos\theta & -\sin\theta \\ \sin\theta & \cos\theta\end{pmatrix}$，　$R(\theta)^n = R(n\theta)$
  • 鏡映行列（$\det A = -1$）：$S(\alpha) = \begin{pmatrix}\cos\alpha & \sin\alpha \\ \sin\alpha & -\cos\alpha\end{pmatrix}$，　$S(\alpha)^2 = E$（対合）
📝 解答 (1) — n が奇数のとき A は回転行列

$A$ が直交行列なので $\det A = \pm 1$。$A^n = E$ の行列式をとると，

$$(\det A)^n = \det(A^n) = \det E = 1$$

$n$ が奇数のとき，実数 $d$ が $d^n = 1$ を満たすのは $d = 1$ のみ（奇数乗は単調）。よって $\det A = 1$。

直交行列かつ $\det A = 1$ の $2\times 2$ 行列は回転行列に限る。$\square$

📝 解答 (2) — 全ての解

（i）回転行列の場合（任意の $n$）：

$A = R(\theta)$ とおくと $A^n = R(n\theta) = E \Longleftrightarrow n\theta \equiv 0 \pmod{2\pi}$，すなわち $\theta = \dfrac{2\pi k}{n}$（$k \in \mathbb{Z}$）。$[0,2\pi)$ 内の互いに異なる角は $k = 0, 1, \ldots, n-1$ の $n$ 個。

$$\boxed{A = R\!\left(\frac{2\pi k}{n}\right) = \begin{pmatrix}\cos\tfrac{2\pi k}{n} & -\sin\tfrac{2\pi k}{n} \\[4pt] \sin\tfrac{2\pi k}{n} & \cos\tfrac{2\pi k}{n}\end{pmatrix},\quad k = 0, 1, \ldots, n-1}$$

（ii）鏡映行列の場合：

$A = S(\alpha)$ は $S(\alpha)^2 = E$ を満たす（対合）。よって

  • $n$ が偶数のとき：$A^n = (S(\alpha)^2)^{n/2} = E^{n/2} = E$ ✓ → 全ての鏡映行列が解。
  • $n$ が奇数のとき：$A^n = S(\alpha)^{2\lfloor n/2\rfloor}\cdot S(\alpha) = S(\alpha) \ne E$ → 解なし（(1) と整合）。

まとめ：

  • $n$ が奇数：$R\!\left(\dfrac{2\pi k}{n}\right)$，$k=0,\ldots,n-1$（計 $n$ 個）
  • $n$ が偶数：上記 $n$ 個の回転行列，および $S(\alpha) = \begin{pmatrix}\cos\alpha&\sin\alpha\\\sin\alpha&-\cos\alpha\end{pmatrix}$（$\alpha \in \mathbb{R}$ 任意）の全て
🧠 思考の出発点

(1) の核心：「det は積の準同型」$\det(A^n) = (\det A)^n$。奇数乗が 1 になる実数は 1 のみ（偶数乗だと $-1$ も候補になる）。これを使うと det の値が決まり，分類定理から型が決まる。

(2) の核心：直交行列は回転か鏡映の2種類しかない。鏡映の対合性 $S^2=E$ が偶数乗で効く。奇数のとき $S^n = S \ne E$ なので鏡映は排除される。

3 固有値・固有ベクトル・対角化・行列の冪

問題文

行列 $A = \begin{pmatrix}13&-30\\5&-12\end{pmatrix}$ に対し，次を求めよ。

  1. $\lambda$ に関する2次式 $\det(A - \lambda E) = 0$ の解 $\lambda_1,\lambda_2$ を求めよ。
  2. (1) で求めた $\lambda_i$ に対し，$A\mathbf{v}_i = \lambda_i \mathbf{v}_i$ を満たす $\mathbf{0}$ でないベクトル $\mathbf{v}_i$ をそれぞれ一つずつ求めよ。
  3. (2) で求めたベクトルを横に並べた $2\times 2$ 行列 $P := (\mathbf{v}_1,\mathbf{v}_2)$ を考える。このとき $P^{-1}AP$ を計算せよ。
  4. 自然数 $n$ に対し $A^n$ を求めよ。
📝 解答 (1) — 固有値
$$\det(A - \lambda E) = \det\begin{pmatrix}13-\lambda&-30\\5&-12-\lambda\end{pmatrix}$$ $$= (13-\lambda)(-12-\lambda) - (-30)\cdot 5$$ $$= \lambda^2 - \lambda - 156 + 150 = \lambda^2 - \lambda - 6 = (\lambda-3)(\lambda+2)$$

よって $\boxed{\lambda_1 = 3,\quad \lambda_2 = -2}$

📝 解答 (2) — 固有ベクトル

$\lambda_1 = 3$ のとき：$(A - 3E)\mathbf{v} = \mathbf{0}$

$$\begin{pmatrix}10&-30\\5&-15\end{pmatrix}\mathbf{v}=\mathbf{0}\;\Longrightarrow\; x = 3y\;\Longrightarrow\; \mathbf{v}_1 = \begin{pmatrix}3\\1\end{pmatrix}$$

確認：$A\mathbf{v}_1 = \begin{pmatrix}39-30\\15-12\end{pmatrix}=\begin{pmatrix}9\\3\end{pmatrix}=3\mathbf{v}_1$ ✓

$\lambda_2 = -2$ のとき：$(A + 2E)\mathbf{v} = \mathbf{0}$

$$\begin{pmatrix}15&-30\\5&-10\end{pmatrix}\mathbf{v}=\mathbf{0}\;\Longrightarrow\; x = 2y\;\Longrightarrow\; \mathbf{v}_2 = \begin{pmatrix}2\\1\end{pmatrix}$$

確認：$A\mathbf{v}_2 = \begin{pmatrix}26-30\\10-12\end{pmatrix}=\begin{pmatrix}-4\\-2\end{pmatrix}=-2\mathbf{v}_2$ ✓

📝 解答 (3) — $P^{-1}AP$
$$P = \begin{pmatrix}3&2\\1&1\end{pmatrix},\quad \det(P) = 3-2 = 1,\quad P^{-1} = \begin{pmatrix}1&-2\\-1&3\end{pmatrix}$$

対角化の原理：$A P = P\,\mathrm{diag}(\lambda_1,\lambda_2)$ なので $P^{-1}AP = \mathrm{diag}(\lambda_1,\lambda_2)$。

計算で確認：

$$AP = \begin{pmatrix}13&-30\\5&-12\end{pmatrix}\begin{pmatrix}3&2\\1&1\end{pmatrix}=\begin{pmatrix}9&-4\\3&-2\end{pmatrix}$$ $$P^{-1}AP = \begin{pmatrix}1&-2\\-1&3\end{pmatrix}\begin{pmatrix}9&-4\\3&-2\end{pmatrix}=\begin{pmatrix}9-6&-4+4\\-9+9&4-6\end{pmatrix}=\boxed{\begin{pmatrix}3&0\\0&-2\end{pmatrix}}$$
📝 解答 (4) — $A^n$

$A = P\,\mathrm{diag}(3,-2)\,P^{-1}$ より

$$A^n = P\begin{pmatrix}3^n&0\\0&(-2)^n\end{pmatrix}P^{-1}$$ $$= \begin{pmatrix}3&2\\1&1\end{pmatrix}\begin{pmatrix}3^n&0\\0&(-2)^n\end{pmatrix}\begin{pmatrix}1&-2\\-1&3\end{pmatrix}$$ $$= \begin{pmatrix}3\cdot 3^n&2\cdot(-2)^n\\3^n&(-2)^n\end{pmatrix}\begin{pmatrix}1&-2\\-1&3\end{pmatrix}$$ $$= \boxed{\begin{pmatrix}3^{n+1}-2\cdot(-2)^n & 6\bigl((-2)^n - 3^n\bigr)\\[4pt] 3^n - (-2)^n & 3\cdot(-2)^n - 2\cdot 3^n\end{pmatrix}}$$

確認（$n=1$）：$\begin{pmatrix}9+4&6(-2-3)\\3+2&-6-6\end{pmatrix}=\begin{pmatrix}13&-30\\5&-12\end{pmatrix}=A$ ✓

🧠 思考の出発点

流れをつかむ：固有方程式 → 固有値 → 固有ベクトル → 対角化行列 $P$ → $A = P\Lambda P^{-1}$ → $A^n = P\Lambda^n P^{-1}$。この一連の流れは「行列の冪を求めよ」という問題の標準戦略。

固有ベクトルの求め方：$(A-\lambda_i E)\mathbf{v}=\mathbf{0}$ の非自明解を探す。行基本変形して自由変数から書く。スカラー倍の自由度があるので簡単な整数にとることが多い。

$\det(P)=1$ は計算のラッキー：$P^{-1}$ が整数行列になり，後の計算が楽になる。

4 $\mathrm{Hom}(V,W)$ のベクトル空間構造と同型

問題文

$V$，$W$ を $\mathbb{R}$-ベクトル空間とする。

  1. $\mathrm{Hom}(V,W) := \{f : V \to W \mid f \text{ は線形写像}\}$ に，加法とスカラー倍を $$(f+g)(v) := f(v)+g(v),\qquad (af)(v) := a\cdot f(v)$$ $（f,g\in\mathrm{Hom}(V,W),\;a\in\mathbb{R},\;v\in V）$ で定める。これにより $\mathrm{Hom}(V,W)$ は $\mathbb{R}$-ベクトル空間になることを示せ。
  2. $V = \mathbb{R}$ のとき，写像 $\varphi \colon \mathrm{Hom}(\mathbb{R},W) \to W;\; f \mapsto f(1)$ は同型な線形写像であることを示せ。
📝 解答 (1) — $\mathrm{Hom}(V,W)$ がベクトル空間

ベクトル空間の公理（閉鎖性・零元・逆元・スカラー倍の規則）を順に確認する。

Step 1：演算が $\mathrm{Hom}(V,W)$ に閉じる（$f+g$，$af$ が線形写像）。

$f+g$ が線形：$a,b\in\mathbb{R}$，$v,w\in V$ に対し

$$(f+g)(av+bw) = f(av+bw)+g(av+bw) = af(v)+bf(w)+ag(v)+bg(w) = a(f+g)(v)+b(f+g)(w)$$

$af$ が線形：$(af)(bv+cw) = a\cdot f(bv+cw) = a(bf(v)+cf(w)) = b(af)(v)+c(af)(w)$

Step 2：零元（零写像 $\mathbf{0}: v \mapsto 0_W$）の存在。

$\mathbf{0}$ は線形（$\mathbf{0}(av+bw)=0=a\cdot 0+b\cdot 0$）かつ $(f+\mathbf{0})(v)=f(v)+0=f(v)$。

Step 3：逆元（$-f: v\mapsto -f(v)$）の存在。

$-f$ は線形（$(-f)$ は $(-1)f$ に同じ）かつ $(f+(-f))(v)=f(v)-f(v)=0=\mathbf{0}(v)$。

Step 4：残りの公理（加法の交換・結合，スカラー倍の規則）は $W$ での等式から直接従う。

例：$(f+g)(v) = f(v)+g(v) = g(v)+f(v) = (g+f)(v)$ （$W$ での加法の交換則を使用）。

以上より $\mathrm{Hom}(V,W)$ は $\mathbb{R}$-ベクトル空間である。$\square$

📝 解答 (2) — $\varphi: \mathrm{Hom}(\mathbb{R},W)\to W$ は同型

同型 ⟺ 線形 かつ 全単射，を示す。

線形性：

$$\varphi(af+bg) = (af+bg)(1) = af(1)+bg(1) = a\varphi(f)+b\varphi(g)$$

単射（$\ker\varphi = \{\mathbf{0}\}$）：

$\varphi(f)=0_W$，すなわち $f(1)=0_W$ とする。任意の $x\in\mathbb{R}$ に対し，$f$ の線形性より

$$f(x) = f(x\cdot 1) = x\cdot f(1) = x\cdot 0_W = 0_W$$

よって $f = \mathbf{0}$。ゆえに $\ker\varphi = \{\mathbf{0}\}$，$\varphi$ は単射。

全射：

任意の $w\in W$ に対し，$f_w\colon \mathbb{R}\to W;\; x\mapsto xw$ を定める。

$f_w$ は線形（$f_w(ax+by) = (ax+by)w = a(xw)+b(yw) = af_w(x)+bf_w(y)$）かつ $\varphi(f_w)=f_w(1)=w$。よって $\varphi$ は全射。

以上より $\varphi$ は同型な線形写像（$\mathbb{R}$-ベクトル空間の同型）である。$\square$

※ この同型は $\mathrm{Hom}(\mathbb{R},W)\cong W$ を示す。直感的には「$\mathbb{R}\to W$ の線形写像は $f(1)\in W$ で完全に決まる」ということ。

🧠 思考の出発点

(1) ベクトル空間の公理確認の戦略：「各演算が閉じているか」と「公理の各項目が成り立つか」を分けて考える。閉鎖性（$f+g,af\in\mathrm{Hom}$）は証明が必要。残りの公理は $W$ での性質を「点ごと（pointwise）に」引き継ぐだけなので比較的簡単。

(2) 同型を示すには線形性・単射性・全射性の3つを示す。単射性を示すとき「$\varphi(f)=0 \Rightarrow f=\mathbf{0}$（零写像）」を示すのが定石（$\ker\varphi$ を調べる）。全射性では，$w\in W$ が与えられたとき「原像になる元を具体的に構成」するのが素直な方針。$V=\mathbb{R}$ のとき線形写像は $x\mapsto xw$ の形に書ける，という事実がそのまま全射の証明になる。
