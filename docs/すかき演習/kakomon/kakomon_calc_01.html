1 ε-N 論法：$e^{a_n} \to e^a$

問題文

数列 $\{a_n\}_n$ が実数 $a$ に収束するとする。このとき数列 $\{e^{a_n}\}_n$ が実数 $e^a$ に収束することを $\varepsilon$-$N$ 論法を用いて示せ。

🧠 方針を立てる前に

何を示すか：「$e^{a_n} \to e^a$」$= $ 「任意の $\varepsilon > 0$ に対し，ある $N$ が存在して $n \geq N \Rightarrow |e^{a_n} - e^a| < \varepsilon$」。

何を使えるか：$a_n \to a$（仮定），平均値定理（または指数関数の性質）。

🔄 ゴールから逆算する

$|e^{a_n} - e^a|$ を $|a_n - a|$ で上から抑えたい。平均値定理を使うと

$$|e^{a_n} - e^a| = e^c |a_n - a| \quad (c \text{ は } a_n \text{ と } a \text{ の間の点})$$

$c$ は $a_n$ と $a$ の間なので，$|a_n - a| < 1$ が言えていれば $c \in (a-1,\, a+1)$ の範囲に収まり，$e^c \leq e^{a+1}$（有界な定数）が使える。だから「まず $|a_n - a| < 1$ を確保してから，$\varepsilon$ のサイズを調整する」という2段階の取り方になる。

📝 解答

$\varepsilon > 0$ を任意に与える。以下の2条件を満たす $N$ を構成する。

Step 1（$\varepsilon_1$ の設定）.　$\varepsilon_1 := \min\!\left(1,\; \dfrac{\varepsilon}{e^{a+1}}\right) > 0$ とおく。

$a_n \to a$ の仮定より，ある $N \in \mathbb{N}$ が存在して

$$n \geq N \implies |a_n - a| < \varepsilon_1.$$

Step 2（上界の評価）.　$n \geq N$ を固定する。$|a_n - a| < \varepsilon_1 \leq 1$ なので，$a_n$ と $a$ の間の任意の点 $c$ に対して

$$a - 1 < c < a + 1 \implies e^c < e^{a+1}.$$

平均値定理を $e^x$ に適用すると，$a_n$ と $a$ の間のある $c$ に対して

$$|e^{a_n} - e^a| = e^c |a_n - a|.$$

したがって

$$|e^{a_n} - e^a| = e^c |a_n - a| < e^{a+1} \cdot \varepsilon_1 \leq e^{a+1} \cdot \frac{\varepsilon}{e^{a+1}} = \varepsilon.$$

以上より，任意の $\varepsilon > 0$ に対して上記の $N$ が存在し $n \geq N \Rightarrow |e^{a_n} - e^a| < \varepsilon$ が成立する。よって $e^{a_n} \to e^a$。$\square$

🔑 ポイントまとめ
  • 平均値定理：$f(b) - f(a) = f'(c)(b-a)$ は「差を微分で評価する」基本技。$f = e^x$ なら $f' = e^x$ なので差がそのまま評価できる。
  • 2段階の $\varepsilon$ 設定：$\min(1, \varepsilon / C)$ という形は頻出。「まず $|a_n - a| < 1$ で定数 $C = e^{a+1}$ を確保し，次に $\varepsilon/C$ で誤差を制御する」というセット。
  • 別解（$|e^t - 1| \leq |t|e^{|t|}$ を使う方法）：$|e^{a_n} - e^a| = e^a|e^{a_n - a} - 1|$，$|e^s - 1| \leq |s|e^{|s|}$（$|s| \leq 1$）を使っても同様に示せる。
2 Cesàro 平均

問題文

数列 $\{a_n\}_n$ に対し，新たな数列 $\{A_n\}_n$ を $A_n := \dfrac{\displaystyle\sum_{k=1}^{n} a_k}{n}$ と定める。

  1. $\displaystyle\lim_{n\to\infty} a_n = \alpha$ ならば $\displaystyle\lim_{n\to\infty} A_n = \alpha$ を証明せよ。
  2. $\displaystyle\lim_{n\to\infty} \frac{1 + \sqrt{2} + \sqrt[3]{3} + \cdots + \sqrt[n]{n}}{n}$ を求めよ。
Cesàro 平均定理

この問題の (1) は Cesàro（チェザロ）平均定理として知られる重要な定理。$a_n \to \alpha$ ならば算術平均 $A_n$ も同じ極限に収束する。逆は一般に成立しない（$A_n \to \alpha$ でも $a_n$ が収束するとは限らない）。

🧠 (1) の方針
🔄 ゴールから逆算する

$|A_n - \alpha| < \varepsilon$ を示したい。$A_n - \alpha = \frac{1}{n}\sum_{k=1}^n (a_k - \alpha)$ なので

$$|A_n - \alpha| \leq \frac{1}{n}\sum_{k=1}^n |a_k - \alpha|.$$

$a_k \to \alpha$ なので，大きい $k$（$k > N$ 以降）については $|a_k - \alpha| < \varepsilon/2$ が使える。小さい $k$（$k \leq N$）の部分は固定された有限和なので，$n$ を大きくすれば $1/n$ で潰せる。

→「後半は $\varepsilon/2$ で抑え，前半は $n \to \infty$ で消す」という分割が定石。

📝 解答 (1)

$\varepsilon > 0$ を任意に与える。$a_n \to \alpha$ の仮定より，ある $N_0 \in \mathbb{N}$ が存在して

$$k > N_0 \implies |a_k - \alpha| < \frac{\varepsilon}{2}.$$

$C := \displaystyle\sum_{k=1}^{N_0} |a_k - \alpha|$（固定された有限和）とおく。$n > N_0$ に対して

$$|A_n - \alpha| = \left|\frac{1}{n}\sum_{k=1}^n(a_k - \alpha)\right| \leq \frac{1}{n}\sum_{k=1}^{N_0}|a_k - \alpha| + \frac{1}{n}\sum_{k=N_0+1}^{n}|a_k - \alpha|$$ $$\leq \frac{C}{n} + \frac{n - N_0}{n} \cdot \frac{\varepsilon}{2} < \frac{C}{n} + \frac{\varepsilon}{2}.$$

さらに $\dfrac{C}{n} < \dfrac{\varepsilon}{2}$ となる $N_1 = \left\lceil\dfrac{2C}{\varepsilon}\right\rceil + 1$ をとり，$N := \max(N_0, N_1)$ とおくと

$$n \geq N \implies |A_n - \alpha| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon.$$

以上より $\displaystyle\lim_{n\to\infty} A_n = \alpha$。$\square$

📝 解答 (2)

$a_k := \sqrt[k]{k} = k^{1/k}$ とおくと，求める極限は $\displaystyle\lim_{n\to\infty} A_n = \lim_{n\to\infty}\frac{1}{n}\sum_{k=1}^n a_k$。

$a_k \to 1$ の確認：

$$a_k = k^{1/k} = e^{\frac{\ln k}{k}}.$$

$\displaystyle\lim_{k\to\infty} \frac{\ln k}{k} = 0$（$\ln k$ は $k$ より遥かに遅く増大する）ので，指数関数の連続性より $a_k \to e^0 = 1$。

(1) の適用：$a_k \to 1 = \alpha$ なので，Cesàro 平均定理（(1)）より

$$\lim_{n\to\infty} \frac{1 + \sqrt{2} + \sqrt[3]{3} + \cdots + \sqrt[n]{n}}{n} = \lim_{n\to\infty} A_n = \boxed{1}.$$
🔑 ポイントまとめ
  • (1) の核心：$\sum |a_k - \alpha|$ を「前半（有限個）」と「後半（$\varepsilon/2$ 以下）」に分割するのが定石。前半は $n \to \infty$ で $1/n$ に消え，後半はすでに小さい。
  • (2) の核心：$k^{1/k} = e^{(\ln k)/k}$ と書いて $(\ln k)/k \to 0$ を使う。この変形は $n^{1/n}$ 型の極限の標準手法。
  • Cesàro の逆は不成立：$a_n = (-1)^n$ は収束しないが $A_n \to 0$。定理の「仮定は $a_n$ 収束」が必要。
3 $f(x) = \log(x + \sqrt{x^2+1})$ の高階微分

問題文

$f(x) := \log(x + \sqrt{x^2 + 1})$ とおく。

  1. $(1 + x^2)f''(x) + xf'(x) = 0$ を示せ。
  2. 任意の正整数 $n > 0$ に対し $$(1+x^2)f^{(n+2)}(x) + (2n+1)xf^{(n+1)}(x) + n^2 f^{(n)}(x) = 0$$ を示せ。
$f(x) = \log(x+\sqrt{x^2+1})$ の正体

$f(x)$ は双曲線逆正弦関数 $\mathrm{arcsinh}(x) = \sinh^{-1}(x)$（$\mathrm{asinh}$ とも書く）に等しい。

$$f'(x) = \frac{1}{\sqrt{x^2+1}} = (1+x^2)^{-1/2}.$$

これは $(\sin\theta)$ の代わりに $\sinh$ を使った「双曲線版の逆三角関数」。対数で表せる理由は $\sinh^{-1}(x) = \ln(x + \sqrt{x^2+1})$（定義から直接導ける）。

📝 準備：$f'(x)$ の計算

$f(x) = \log(x + \sqrt{x^2+1})$ を微分する（$\log$ は自然対数）。

$$f'(x) = \frac{1}{x + \sqrt{x^2+1}} \cdot \left(1 + \frac{x}{\sqrt{x^2+1}}\right) = \frac{1}{x+\sqrt{x^2+1}} \cdot \frac{\sqrt{x^2+1}+x}{\sqrt{x^2+1}} = \frac{1}{\sqrt{x^2+1}}.$$

さらに

$$f''(x) = \frac{d}{dx}(1+x^2)^{-1/2} = -\frac{1}{2}(1+x^2)^{-3/2}\cdot 2x = \frac{-x}{(1+x^2)^{3/2}}.$$
📝 解答 (1)

$(1+x^2)f''(x) + xf'(x)$ を計算する。

$$(1+x^2) \cdot \frac{-x}{(1+x^2)^{3/2}} + x \cdot \frac{1}{(1+x^2)^{1/2}}$$ $$= \frac{-x}{(1+x^2)^{1/2}} + \frac{x}{(1+x^2)^{1/2}} = 0. \quad \square$$
🧠 (2) の方針
🔄 ゴールから逆算する

(2) で示したい等式を $P(n)$ と呼ぶ。$P(n) \Rightarrow P(n+1)$ を示すには「$P(n)$ の両辺を $x$ で微分する」のが自然。なぜなら $f^{(k)}$ を微分すると $f^{(k+1)}$ が出てくるからだ。

$P(n)$：$(1+x^2)f^{(n+2)} + (2n+1)xf^{(n+1)} + n^2 f^{(n)} = 0$ を $x$ で微分すると

$$2x \cdot f^{(n+2)} + (1+x^2)f^{(n+3)} + (2n+1)f^{(n+1)} + (2n+1)x f^{(n+2)} + n^2 f^{(n+1)} = 0$$ $$\Rightarrow\; (1+x^2)f^{(n+3)} + \underbrace{(2+(2n+1))}_{=2n+3}x f^{(n+2)} + \underbrace{(2n+1+n^2)}_{=(n+1)^2}f^{(n+1)} = 0.$$

これは $P(n+1)$（$n$ を $n+1$ に置き換えた式）にちょうど一致する！

📝 解答 (2) — 数学的帰納法

基底（$n = 1$）：(1) で示した $(1+x^2)f'' + xf' = 0$ の両辺を $x$ で微分する。

$$2x f'' + (1+x^2)f''' + f' + x f'' = 0 \;\Longrightarrow\; (1+x^2)f''' + 3xf'' + f' = 0.$$

これは $n=1$ の式（$(1+x^2)f^{(3)} + 3xf^{(2)} + 1^2 f^{(1)} = 0$）に一致する。✓

帰納段階：$n$ で成立，すなわち

$$(1+x^2)f^{(n+2)} + (2n+1)xf^{(n+1)} + n^2 f^{(n)} = 0 \tag{$P(n)$}$$

を仮定する。両辺を $x$ について微分すると

$$2x f^{(n+2)} + (1+x^2)f^{(n+3)} + (2n+1)f^{(n+1)} + (2n+1)x f^{(n+2)} + n^2 f^{(n+1)} = 0.$$

$f^{(n+2)}$ の係数を整理：$2x + (2n+1)x = (2n+3)x$。

$f^{(n+1)}$ の係数を整理：$(2n+1) + n^2 = n^2 + 2n + 1 = (n+1)^2$。

よって

$$(1+x^2)f^{(n+3)} + (2(n+1)+1)\,x f^{(n+2)} + (n+1)^2 f^{(n+1)} = 0.$$

これは $n+1$ の場合 $P(n+1)$ に他ならない。帰納法より全ての正整数 $n$ で成立。$\square$

🔑 ポイントまとめ
  • 「微分して帰納」パターン：高階微分に関する等式を帰納で示す際，「$P(n)$ を微分して $P(n+1)$ を得る」のが定石。係数が $n$ に依存する場合でも，積の微分で整理すれば係数の更新規則が見える。
  • 係数の更新：$x$ の係数 $2n+1 \to 2(n+1)+1 = 2n+3$（$+2$），定数係数 $n^2 \to (n+1)^2$（これは $(n+1)^2 = n^2+2n+1$ と分解して考えるとわかりやすい）。
  • この漸化式の意味：$\mathrm{arcsinh}$ の Taylor 展開の係数に関する漸化式になっており，Legendre 方程式の変形とも関連する。
4 $f_n(x) = x^n\sin\!\dfrac{1}{x}$ の微分可能性と連続性

問題文

$n$ を正の整数とし，関数

$$f_n(x) := \begin{cases} x^n \sin\dfrac{1}{x} & (x \ne 0) \\ 0 & (x = 0) \end{cases}$$

を考える。これが $x = 0$ で微分可能かどうかを判定せよ。また微分可能である場合，導関数 $f_n'(x)$ は $x = 0$ で連続かどうか判定せよ。

🧠 方針
🔄 ゴールから逆算する

微分可能性（$x=0$）：定義通り $f_n'(0) = \displaystyle\lim_{h\to 0} \dfrac{f_n(h) - f_n(0)}{h} = \lim_{h\to 0} h^{n-1}\sin\dfrac{1}{h}$ が存在するかを調べる。

  • $n = 1$：$\sin(1/h)$ は $h \to 0$ で振動して極限を持たない。
  • $n \geq 2$：$|h^{n-1}\sin(1/h)| \leq |h|^{n-1} \to 0$（はさみうち）。

$f_n'(0)$ の連続性（$n \geq 2$ の場合）：$x \ne 0$ での $f_n'(x)$ を積の微分で求め，$x \to 0$ の極限と $f_n'(0) = 0$ を比較する。

📝 解答：微分可能性

$x = 0$ での微分係数を定義から計算する：

$$f_n'(0) = \lim_{h \to 0} \frac{f_n(h) - 0}{h} = \lim_{h \to 0} h^{n-1} \sin\frac{1}{h}.$$

場合 $n = 1$：　$\displaystyle\lim_{h\to 0} \sin\dfrac{1}{h}$ は振動して存在しない。よって $f_1$ は $x = 0$ で微分不可能。

場合 $n \geq 2$：　$\left|h^{n-1}\sin\dfrac{1}{h}\right| \leq |h|^{n-1} \to 0$（$h \to 0$）。はさみうちの原理より

$$f_n'(0) = 0.$$

よって $f_n$ は $x = 0$ で微分可能であり，$f_n'(0) = 0$。

📝 解答：$f_n'(x)$ の $x=0$ での連続性（$n \geq 2$ の場合）

$x \ne 0$ での $f_n'(x)$ を積の微分則で求める：

$$f_n'(x) = nx^{n-1}\sin\frac{1}{x} + x^n \cdot \cos\frac{1}{x} \cdot \left(-\frac{1}{x^2}\right) = nx^{n-1}\sin\frac{1}{x} - x^{n-2}\cos\frac{1}{x}.$$

$x \to 0$ での各項の挙動：

  • 第1項 $nx^{n-1}\sin(1/x)$：$|nx^{n-1}\sin(1/x)| \leq n|x|^{n-1} \to 0$（$n \geq 2$ で $n-1 \geq 1$）。
  • 第2項 $x^{n-2}\cos(1/x)$：$|x^{n-2}\cos(1/x)| \leq |x|^{n-2}$。これが $0 \to 0$ になるには $n - 2 \geq 1$，すなわち $n \geq 3$ が必要。$n = 2$ では $|x^0 \cos(1/x)| = |\cos(1/x)|$ が振動する。

場合 $n = 2$：

$$f_2'(x) = 2x\sin\frac{1}{x} - \cos\frac{1}{x} \quad (x \ne 0).$$

$x \to 0$ のとき第1項 $\to 0$，第2項 $\cos(1/x)$ は振動して極限を持たない。よって $\displaystyle\lim_{x\to 0}f_2'(x)$ が存在せず，$f_2'$ は $x=0$ で連続でない。

場合 $n \geq 3$：

両項ともに $x \to 0$ で $0$ に収束するので $\displaystyle\lim_{x\to 0}f_n'(x) = 0 = f_n'(0)$。よって $f_n'$ は $x=0$ で連続。

📊 結果まとめ
$n$ $x=0$ で微分可能？ $f_n'(0)$ $f_n'$ は $x=0$ で連続？
$1$✗ 不可能——
$2$✓ 可能$0$✗ 不連続
$n \geq 3$✓ 可能$0$✓ 連続
🔑 ポイントまとめ
  • 「定義から計算」：$x=0$ での微分可能性は必ず極限の定義に戻る。$|h^{n-1}\sin(1/h)| \leq |h|^{n-1}$ のはさみうちが鍵。
  • 「連続性の判定」：$f_n'(0)$ が求まっても $\lim_{x \to 0} f_n'(x) \ne f_n'(0)$ のことがある（$n=2$ の例）。$f'$ の連続性は $f'$ の極限を別途調べる必要がある。
  • 「微分可能だが $f'$ が不連続」：$f_2$ はその例。「微分可能 $\Rightarrow$ $f'$ 連続」は一般に成り立たない（連続微分可能と微分可能は別の概念）。
5 指数関数の部分和 $g_n(x)$

問題文

関数 $g_n(x) := 1 + x + \dfrac{x^2}{2!} + \cdots + \dfrac{x^n}{n!}$ を考える。

  1. 任意の正整数 $n > 0$ に対し，$g_{2n}(x) > 0$（$\forall x$）かつ $g_{2n-1}(x) = 0$ は唯一つの解 $\alpha_n$ を持つことを示せ。
  2. $\displaystyle\lim_{n\to\infty} \alpha_n = -\infty$ を示せ。
$g_n(x)$ の基本性質
  • $g_n(x) = \displaystyle\sum_{k=0}^{n} \dfrac{x^k}{k!}$　（$e^x$ の Taylor 展開の $n$ 次部分和）
  • $g_n'(x) = g_{n-1}(x)$　（微分すると一つ次数が下がる）
  • $g_n(x) = g_{n-1}(x) + \dfrac{x^n}{n!}$　（最後の項の追加）
  • 固定された $x$ に対し $g_n(x) \to e^x$（$n \to \infty$）
🧠 (1) の方針

「$g_{2n} > 0$」と「$g_{2n-1}$ が唯一根を持つ」は互いに支え合う構造を持つ。数学的帰納法で同時に証明するのが自然。

🔄 帰納構造の確認

仮定：$g_{2n-2}(x) > 0$（全実数で正）。

ステップ1：$g_{2n-1}' = g_{2n-2} > 0$ より $g_{2n-1}$ は狭義単調増加。

ステップ2：$g_{2n-1}(-\infty) = -\infty$，$g_{2n-1}(+\infty) = +\infty$（奇数次の最高次係数が正），かつ $g_{2n-1}(0) = 1 > 0$。中間値定理と単調性から唯一の根 $\alpha_n < 0$ が存在。

ステップ3：$g_{2n}(x) = g_{2n-1}(x) + \dfrac{x^{2n}}{(2n)!}$。この最小値を評価する。

$g_{2n}' = g_{2n-1}$：$g_{2n-1}$ は単調増加で根 $\alpha_n$ を持つので，$g_{2n}$ は $x = \alpha_n$ で最小値をとる。

最小値：$g_{2n}(\alpha_n) = \underbrace{g_{2n-1}(\alpha_n)}_{=0} + \dfrac{\alpha_n^{2n}}{(2n)!} = \dfrac{\alpha_n^{2n}}{(2n)!} > 0$（$\alpha_n \ne 0$ より）。

よって $g_{2n}(x) \geq g_{2n}(\alpha_n) > 0$。

📝 解答 (1) — 数学的帰納法

基底（$n = 1$）：

$g_1(x) = 1 + x$：狭義単調増加（$g_1' = 1 > 0$），$g_1(0) = 1 > 0$ なので唯一の根は $\alpha_1 = -1 < 0$。

$g_2(x) = 1 + x + x^2/2$：$g_2'(x) = g_1(x) = 1+x$，根 $x = -1$ で最小値 $g_2(-1) = 1 - 1 + \frac{1}{2} = \frac{1}{2} > 0$。最高次係数 $\frac{1}{2} > 0$ なので $x \to \pm\infty$ で $+\infty$。よって $g_2(x) > 0$（$\forall x$）。✓

帰納段階：$g_{2n-2}(x) > 0$（$\forall x$）を仮定する。

  1. $g_{2n-1}'(x) = g_{2n-2}(x) > 0$ より $g_{2n-1}$ は $\mathbb{R}$ 上狭義単調増加。
  2. $g_{2n-1}(x) \to -\infty$（$x \to -\infty$），$g_{2n-1}(x) \to +\infty$（$x \to +\infty$）かつ単調なので，中間値定理より唯一の根 $\alpha_n$ が存在。
  3. $g_{2n-1}(0) = 1 > 0$ かつ $g_{2n-1}$ は増加だから $\alpha_n < 0$，特に $\alpha_n \ne 0$。
  4. $g_{2n}'(x) = g_{2n-1}(x)$ は $x < \alpha_n$ で負，$x > \alpha_n$ で正 → $g_{2n}$ は $x = \alpha_n$ で最小値をとる。
  5. 最小値の評価： $$g_{2n}(\alpha_n) = g_{2n-1}(\alpha_n) + \frac{\alpha_n^{2n}}{(2n)!} = 0 + \frac{\alpha_n^{2n}}{(2n)!} > 0.$$
  6. $x \to \pm\infty$ で $g_{2n}(x) \to +\infty$（最高次 $x^{2n}/(2n)!$ が正）かつ最小値 $> 0$ だから $g_{2n}(x) > 0$（$\forall x$）。

以上で $n$ での成立 → $n+1$ での成立が示された（$g_{2n} > 0$ が次の $n$ の「仮定」になる）。$\square$

📝 解答 (2) — $\alpha_n \to -\infty$

方針：任意の $M > 0$ に対して，十分大きい $n$ では $\alpha_n < -M$ を示す。

🔄 $\alpha_n < -M$ と $g_{2n-1}(-M)$ の関係

$g_{2n-1}$ は狭義単調増加で根が $\alpha_n$。したがって

$$\alpha_n < -M \iff -M > \alpha_n \iff g_{2n-1}(-M) > g_{2n-1}(\alpha_n) = 0.$$

つまり「$\alpha_n < -M$ を示す」$\Leftrightarrow$ 「$g_{2n-1}(-M) > 0$ を示す」。

固定された $x = -M$（$M > 0$）に対し，$n \to \infty$ のとき

$$g_{2n-1}(-M) = \sum_{k=0}^{2n-1} \frac{(-M)^k}{k!} \to e^{-M} > 0.$$

（$e^x$ の Taylor 展開は任意の実数で収束するので，部分和は $e^{-M}$ に収束する。）

よって十分大きい $n$（具体的には $g_{2n-1}(-M) > e^{-M}/2 > 0$ となる $n$）に対して $g_{2n-1}(-M) > 0$，すなわち $\alpha_n < -M$。

$M > 0$ は任意だったから $\displaystyle\lim_{n\to\infty} \alpha_n = -\infty$。$\square$

🔑 ポイントまとめ
  • (1) の核心：帰納法の「連鎖」。$g_{2n-2} > 0 \Rightarrow g_{2n-1}$ が唯一根 $\alpha_n$ を持つ $\Rightarrow g_{2n}$ の最小値が正 $\Rightarrow g_{2n} > 0$。この3段ループが帰納の骨格。
  • 最小値の等式 $g_{2n}(\alpha_n) = \alpha_n^{2n}/(2n)!$：$g_{2n} = g_{2n-1} + x^{2n}/(2n)!$ という最後の項の分解と，$g_{2n-1}(\alpha_n) = 0$ を組み合わせる。$\alpha_n \ne 0$ だから偶数乗 $\alpha_n^{2n} > 0$。
  • (2) の核心：単調性と Taylor 収束。$g_{2n-1}$ の単調性から「$\alpha_n < -M$ $\Leftrightarrow$ $g_{2n-1}(-M) > 0$」と同値変換し，後者を Taylor 収束（$g_n(x) \to e^x$）で示す。
  • 直感的な理解：$e^x$ の $2n-1$ 次部分和で $g_{2n-1}(x) = 0$ の根が $\alpha_n$。$e^{-M}$ の部分和は有限次では必ず $e^{-M} > 0$ に近づき，どんなに大きい $M$ でも結局正になる。だから根はどんどん左（$-\infty$）へ逃げていく。
