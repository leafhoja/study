微分積分学演習　第1回

ε-N論法と数列の収束

📋 この回のテーマと必要な知識

この回では数列の収束を厳密に扱うための道具である ε-N論法（イプシロン・エヌ論法） を学ぶ。 「$n \to \infty$ のとき $a_n \to \alpha$」という直感的な概念を、 $\varepsilon$（どれだけ近いか）と $N$（いつから）という2つの数で正確に定義する。

必要な予備知識：

  • 絶対値の定義と性質：$|x| \ge 0$, $|{-x}| = |x|$, $|x| \ge x$ など
  • 三角不等式：$|x + y| \le |x| + |y|$（最重要！）
  • 逆三角不等式：$\big||x| - |y|\big| \le |x - y|$
  • 背理法（間接証明法）
  • 自然数 $\mathbb{N} = \{1, 2, 3, \ldots\}$ の well-ordering（任意の非空部分集合に最小元がある）
📖 ε-N 論法：数列の収束の厳密な定義
定義：数列の収束（ε-N論法）

数列 $\{a_n\}_{n=1}^{\infty}$ が実数 $\alpha$ に収束する（$\lim_{n\to\infty} a_n = \alpha$ と書く）とは：

$$\forall \varepsilon > 0,\; \exists N \in \mathbb{N},\; \forall n \ge N : \; |a_n - \alpha| < \varepsilon$$

が成立すること。日本語で言えば：

「任意の正の数 $\varepsilon$ に対して、ある自然数 $N$ が存在して、$n \ge N$ ならすべて $|a_n - \alpha| < \varepsilon$ が成り立つ。」

💡 定義の「意味」を掴む

この定義は「$n$ が十分大きければ $a_n$ は $\alpha$ にいくらでも近い」を数式化したものだ。

  • $\varepsilon > 0$：「近さ」の基準。どんなに小さな正数を指定しても…
  • $\exists N$：…そこから先はずっと…
  • $|a_n - \alpha| < \varepsilon$：…$a_n$ と $\alpha$ の距離が $\varepsilon$ 未満になる

注意： $N$ は $\varepsilon$ に依存してよい（$\varepsilon$ が小さければ $N$ が大きくなりがち）。 重要なのは、どんな $\varepsilon > 0$ を持ってきても対応する $N$ が必ず存在すること。

💡 ε-N 論法の証明の書き方：「逆算してから順向きに書く」

ε-N 証明は次の2段階で考える：

  1. 草稿（逆算）： $|a_n - \alpha| < \varepsilon$ を達成するには $N$ をどう取ればよいかを計算する。 「$n \ge N$ ならば $|a_n - \alpha| \le \text{（}n \text{ の式）} < \varepsilon$ になってほしい」という観点で $N$ を求める。
  2. 清書（順向き）： 求めた $N$ を冒頭で宣言し、そこから $|a_n - \alpha| < \varepsilon$ を導く。

試験や答案では清書だけを書くが、草稿段階の逆算が本質的な作業だ。

1 極限の一意性

問題文

数列 $\{a_n\}_n$ が2つの実数 $a$, $b$ に収束するとき，$a = b$ を示せ。

📝 模範解答
証明（背理法）

仮定：$a \ne b$ と仮定する。このとき $|a - b| > 0$。

ε の定義：$\varepsilon := \dfrac{|a - b|}{2} > 0$ とおく。

$\lim_{n \to \infty} a_n = a$ より，このε に対して自然数 $N_1$ が存在して， $n \ge N_1$ ならすべて $$|a_n - a| < \varepsilon = \frac{|a-b|}{2}$$

$\lim_{n \to \infty} a_n = b$ より，同様に自然数 $N_2$ が存在して， $n \ge N_2$ ならすべて $$|a_n - b| < \varepsilon = \frac{|a-b|}{2}$$

$N := \max\{N_1, N_2\}$ とおく。$N \ge N_1$ かつ $N \ge N_2$ だから， $n = N$ に対して両方の不等式が成立する。三角不等式より

$$|a - b| = |(a - a_N) + (a_N - b)| \le |a - a_N| + |a_N - b| < \frac{|a-b|}{2} + \frac{|a-b|}{2} = |a - b|$$

$|a - b| < |a - b|$ は矛盾。よって $a = b$。

🧠 思考法・解説

必要な知識

  • ε-N 論法の定義（2回使う）
  • 三角不等式 $|x + y| \le |x| + |y|$
  • 背理法

なぜ背理法を使うのか？

💡 証明方針の選び方

「$a = b$ を示せ」という等式の証明は、直接証明が難しい場合が多い。 $a$ と $b$ はそれぞれ「$a_n$ の極限」として定義されているだけで、その値を直接計算できるわけではない。

一方、背理法では「$a \ne b$ と仮定して矛盾を導く」。$a \ne b$ を仮定すると $|a - b| > 0$ が使えて、 正の数として活用できる。直感的には「$a_n$ が $a$ にも $b$ にも同時に近づくなら $a$ と $b$ は離れていられない」という矛盾。

ε はなぜ $|a - b|/2$ と置くのか？

🔄 逆算：どんな矛盾を導きたいか

最終的に「$|a - b| < |a - b|$（矛盾）」か「$|a - b| \le 0$（矛盾）」の形を作りたい。

仮に $a_N$ が $a$ のε近傍かつ $b$ のε近傍にあるとしよう。すると三角不等式より：

$$|a - b| = |(a - a_N) + (a_N - b)| \le |a - a_N| + |a_N - b| < \varepsilon + \varepsilon = 2\varepsilon$$

これが矛盾になるためには $2\varepsilon \le |a - b|$、すなわち

$$\varepsilon \le \frac{|a - b|}{2}$$

が必要。最もシンプルな選択が $\varepsilon = \dfrac{|a-b|}{2}$ だ。

なぜ $\varepsilon = |a-b|$ ではだめか？ $|a - b| < 2\varepsilon = 2|a-b|$ は常に真（$|a-b| > 0$ なら当然）なので矛盾にならない。 ちょうど半分にすることで「$|a-b| < 2 \cdot \dfrac{|a-b|}{2} = |a - b|$」という矛盾が生まれる。

思考の流れを整理する

① 何を示すか：等式 $a = b$
↓ 直接証明が難しい → 背理法
② $a \ne b$ を仮定 → $|a-b| > 0$ が使える
↓ この正の数を ε に使う
③ 収束の定義を 2回 使う（$a$ への収束 + $b$ への収束）
↓ $N = \max\{N_1, N_2\}$ で同時に両方が成立
④ 三角不等式で $|a-b| < 2\varepsilon = |a-b|$ → 矛盾

自力で解くために

チェックリスト（展開して確認）
  • 「収束する」の定義を思い出し，εと N の量化子の順序（∀ε, ∃N, ∀n≥N）を正確に言えるか
  • 仮定「$a \ne b$」から正の数 $|a-b| > 0$ を取り出し，これをεに使う発想があるか
  • 三角不等式 $|x+y| \le |x|+|y|$ を使って2つの項に分ける操作ができるか
  • $\max\{N_1, N_2\}$ を取る意味：$N \ge N_1$ かつ $N \ge N_2$ を同時に満たすため
  • 矛盾の形「$|a-b| < |a-b|$」が確かに矛盾であることを確認できるか
⚠ よくある間違い
  • $\varepsilon = |a-b|$ と置いてしまう（矛盾にならない）
  • $N_1$ と $N_2$ を別々に使う（同一の $n$ に対して両方を成立させる必要がある）
  • $|a - a_N| < \varepsilon$ と $|a_N - b| < \varepsilon$ を足し算するとき，$|a - b| \le |a - a_N| + |a_N - b|$ の形にすること（$a_N$ が「橋渡し」になっている）
2 極限の演算（和・積・定数倍・逆数）

問題文

数列 $\{a_n\}$，$\{b_n\}$ は $\lim_{n\to\infty} a_n = a$，$\lim_{n\to\infty} b_n = b$ を満たすとする。このとき次を示せ。

  1. $\lim_{n\to\infty}(a_n + b_n) = a + b$
  2. $\lim_{n\to\infty} a_n b_n = ab$
  3. 実数 $c$ に対し，$\lim_{n\to\infty} c a_n = ca$
  4. $a \ne 0$ のとき，$\lim_{n\to\infty}(1/a_n) = 1/a$
📝 模範解答
証明 (1) — 和の極限

$\varepsilon > 0$ を任意に取る。$\lim a_n = a$ より，自然数 $N_1$ があって $n \ge N_1 \Rightarrow |a_n - a| < \varepsilon/2$。同様に自然数 $N_2$ があって $n \ge N_2 \Rightarrow |b_n - b| < \varepsilon/2$。

$N = \max\{N_1, N_2\}$ とおく。任意の $n \ge N$ に対して三角不等式より

$$|(a_n + b_n) - (a+b)| \le |a_n - a| + |b_n - b| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$$

よって $\lim(a_n + b_n) = a+b$。

証明 (2) — 積の極限

補題（有界性）：$\varepsilon_0 = 1$ を用いると，ある $N_1$ があって $n \ge N_1 \Rightarrow |a_n - a| < 1$。よって $n \ge N_1$ では $|a_n| < 1 + |a|$。

$M := \max\{|a_1|, \ldots, |a_{N_1-1}|, 1 + |a|\} > 0$ とおくと，任意の $n$ に対して $|a_n| \le M$。

次に任意の $\varepsilon > 0$ をとる。

$\lim a_n = a$ より，ある $N_2$ があって $n \ge N_2 \Rightarrow |a_n - a| < \dfrac{\varepsilon}{2(1+|b|)}$。

$\lim b_n = b$ より，ある $N_3$ があって $n \ge N_3 \Rightarrow |b_n - b| < \dfrac{\varepsilon}{2M}$。

$N = \max\{N_2, N_3\}$ とおく。任意の $n \ge N$ に対して

$$|a_n b_n - ab| \le |a_n||b_n - b| + |b||a_n - a| < M \cdot \frac{\varepsilon}{2M} + (1+|b|) \cdot \frac{\varepsilon}{2(1+|b|)} = \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$$
証明 (3) — 定数倍

$c = 0$ のとき $ca_n = 0 \to 0 = ca$ は自明。$c \ne 0$ のとき：任意の $\varepsilon > 0$ をとる。$\lim a_n = a$ より，ある $N$ があって $n \ge N \Rightarrow |a_n - a| < \varepsilon/|c|$。よって $|ca_n - ca| = |c||a_n - a| < |c| \cdot \varepsilon/|c| = \varepsilon$。

証明 (4) — 逆数の極限

$a \ne 0$ なので $|a|/2 > 0$。$\lim a_n = a$ より，ある $N_1$ があって $n \ge N_1 \Rightarrow |a_n - a| < |a|/2$。

よって $n \ge N_1$ では $|a_n| \ge |a| - |a_n - a| > |a|/2$。

任意の $\varepsilon > 0$ をとる。$\lim a_n = a$ より，ある $N_2$ があって $n \ge N_2 \Rightarrow |a_n - a| \le |a|^2 \varepsilon / 2$。

$N = \max\{N_1, N_2\}$ とおく。任意の $n \ge N$ に対して

$$\left|\frac{1}{a_n} - \frac{1}{a}\right| = \frac{|a - a_n|}{|a_n||a|} < \frac{|a|^2\varepsilon/2}{(|a|/2) \cdot |a|} = \varepsilon$$
🧠 思考法・解説

(1) 和の極限：ε/2 トリック

🔄 逆算：ε/2 はなぜ使うのか

示したいのは $|(a_n + b_n) - (a+b)| < \varepsilon$。三角不等式を使うと：

$$|(a_n + b_n) - (a+b)| = |(a_n - a) + (b_n - b)| \le |a_n - a| + |b_n - b|$$

右辺を $\varepsilon$ 以下にしたい。最もシンプルな方法は 「$|a_n - a| < \varepsilon/2$ かつ $|b_n - b| < \varepsilon/2$」とすれば和が $< \varepsilon$。

なぜ $\varepsilon/2$ か？ 2項の和が $\varepsilon$ 未満になればよい。各項を「半分ずつ」に割り当てれば自然に成立する。 $\varepsilon/3$，$\varepsilon/4$ でも構わないが，$\varepsilon/2$ が最もシンプル。

(2) 積の極限：有界性の利用（最も技術的）

💡 なぜ有界性が必要か

積の誤差を評価するために次の恒等式を使う：

$$a_n b_n - ab = a_n(b_n - b) + b(a_n - a)$$
🔄 この恒等式はどこから来るか

$a_nb$ を「加えて引く」補間トリック：

$$a_n b_n - ab = \underbrace{a_n b_n - a_n b}_{a_n(b_n - b)} + \underbrace{a_n b - ab}_{b(a_n - a)}$$

こうすると「$b_n - b$（$b_n$ の誤差）」と「$a_n - a$（$a_n$ の誤差）」という，それぞれ $\varepsilon$ で制御できる項に分離できる。$a_nb_n - ab$ を直接扱うより，2つの「差」に分けた方が操作しやすい。

三角不等式で：

$$|a_n b_n - ab| \le |a_n| \cdot |b_n - b| + |b| \cdot |a_n - a|$$

ここで $|a_n|$ は $n$ によって変化するので，直接 $\varepsilon$ で制御するには $|a_n|$ の上界 $M$ が必要だ。 「収束数列は有界」という補題を先に示し，$|a_n| \le M$ を確保してから議論を進める。

🔄 逆算：有界性の補題の証明

$\lim a_n = a$ の定義に $\varepsilon = 1$ を代入する。すると自然数 $N_1$ があって $n \ge N_1 \Rightarrow |a_n - a| < 1$。

逆三角不等式より $n \ge N_1$ に対して $|a_n| \le |a_n - a| + |a| < 1 + |a|$。

$N_1$ 未満の項（有限個！）については単純に最大値を取ればよい。

$$M := \max\{|a_1|, |a_2|, \ldots, |a_{N_1-1}|, 1 + |a|\}$$

このとき任意の $n \in \mathbb{N}$ に対して $|a_n| \le M$。

⚠ なぜ $\varepsilon = 1$ を使うのか

有界性は「極限値からある定数以内に収まること」を言いたい。$\varepsilon$ として何でも好きな正の数を代入できるが， $\varepsilon = 1$ が最もシンプル。結局 $M = 1 + |a| + (\text{有限個の例外の最大値})$ という形になる。

🔄 逆算：ε の配分

$|a_n b_n - ab| \le M|b_n - b| + |b||a_n - a| < \varepsilon$ を達成したい。

自然な配分：各項を $\varepsilon/2$ 以下にする。

  • $M|b_n - b| < \varepsilon/2 \Rightarrow |b_n - b| < \varepsilon/(2M)$
  • $|b||a_n - a| < \varepsilon/2 \Rightarrow |a_n - a| < \varepsilon/(2|b|)$（もし $|b| = 0$ なら自動成立）

$|b| = 0$ の場合に $\varepsilon/(2|b|)$ は定義できないので，$|b|$ の代わりに $1 + |b| > 0$ を使う。 $|b||a_n - a| \le (1+|b|)|a_n - a|$ なので，$|a_n - a| < \varepsilon/(2(1+|b|))$ としておけば安全。

(4) 逆数の極限：分母の下界確保

🔄 逆算：なぜ $|a|/2$ と $|a|^2\varepsilon/2$ が登場するか

$|1/a_n - 1/a| = \dfrac{|a - a_n|}{|a_n||a|}$ と変形できる。分母に $|a_n|$ があるため， $a_n \to 0$ に近づく（$|a_n|$ が小さくなる）と分母が 0 になりかねない。 しかし $a \ne 0$ なら十分大きな $n$ では $a_n$ も 0 にならない。それを保証するのが次の下界。

$\varepsilon_0 = |a|/2 > 0$ を定義に代入。ある $N_1$ があって $n \ge N_1 \Rightarrow |a_n - a| < |a|/2$。

逆三角不等式より：$|a_n| \ge |a| - |a_n - a| > |a| - |a|/2 = |a|/2$。

これで $n \ge N_1$ では $|a_n| \ge |a|/2 > 0$，つまり $1/a_n$ が定義できて分母の下界が確保された。

ここからさらに逆算：$\varepsilon$ の条件 $|a|^2\varepsilon/2$ の導出

達成したいのは $\left|\dfrac{1}{a_n} - \dfrac{1}{a}\right| < \varepsilon$。$|a_n| \ge |a|/2$ を使うと

$$\left|\frac{1}{a_n} - \frac{1}{a}\right| = \frac{|a_n - a|}{|a_n||a|} \le \frac{|a_n - a|}{(|a|/2) \cdot |a|} = \frac{2|a_n - a|}{|a|^2}$$

これを $< \varepsilon$ にするには $|a_n - a| < \dfrac{|a|^2\varepsilon}{2}$ が十分。だから $N_2$ を「$n \ge N_2 \Rightarrow |a_n - a| < \dfrac{|a|^2\varepsilon}{2}$」と選ぶ。

ε の決め方の一般原則

💡 「終わりから考えて、始めに宣言する」
  1. 達成したい不等式（例：$|a_n b_n - ab| < \varepsilon$）をゴールとして設定
  2. ゴールを達成するための条件（例：$|a_n - a| < \delta_1$ かつ $|b_n - b| < \delta_2$）を逆算
  3. $\delta_1$ と $\delta_2$ を $\varepsilon$ で表す（例：$\delta_1 = \varepsilon/(2(1+|b|))$）
  4. 対応する $N$ を宣言して順向きに証明を書く
3 0.999… = 1

問題文

$0.999\cdots = 1$ を示せ。

📝 模範解答

$a_n := \displaystyle\sum_{k=1}^{n} 9 \cdot 10^{-k}$ とおくと，$0.999\cdots = \lim_{n\to\infty} a_n$ と定義される。

等比数列の和の公式より：

$$a_n = 9 \cdot \frac{(1/10)(1 - (1/10)^n)}{1 - 1/10} = 1 - 10^{-n}$$
証明

上の計算より $a_n = 1 - 10^{-n}$。任意の $\varepsilon > 0$ をとる。

$10^N > 1/\varepsilon$ すなわち $N > \log_{10}(1/\varepsilon)$ となる自然数 $N$ を取る（Archimedes の原理）。

任意の $n \ge N$ に対して $10^{-n} \le 10^{-N} < \varepsilon$ より

$$|a_n - 1| = 10^{-n} < \varepsilon$$

よって $\lim_{n\to\infty} a_n = 1$，すなわち $0.999\cdots = 1$。

🧠 思考法・解説

解釈：無限小数を数列の極限として定義する

$0.999\cdots$ の意味をまず確定させる。無限小数とは数列の極限として定義される：

$$a_n = 0.\underbrace{99\cdots9}_{n\text{個}} = 9 \cdot 10^{-1} + 9 \cdot 10^{-2} + \cdots + 9 \cdot 10^{-n} = \sum_{k=1}^{n} 9 \cdot 10^{-k}$$

$0.999\cdots := \lim_{n\to\infty} a_n$ と定義するので，$\lim_{n\to\infty} a_n = 1$ を示せばよい。

等比数列の公式による $a_n$ の計算

等比数列の和の公式 $\sum_{k=1}^{n} r^k = \dfrac{r(1-r^n)}{1-r}$ を $r = 1/10$ に適用：

$$a_n = 9 \cdot \frac{(1/10)(1 - (1/10)^n)}{1 - 1/10} = 9 \cdot \frac{(1/10)(1 - 10^{-n})}{9/10} = 1 - 10^{-n}$$
💡 直感的な説明

$0.999\cdots$ と $1$ の差は $0.000\cdots$ であり，これは $\lim_{n\to\infty} 10^{-n} = 0$ により $0$ だ。 2つの実数の差が $0$ ならば等しい（問題1の一般化）。 「でも $0.999\cdots < 1$ だろう？」という直感は，実数の連続性（アルキメデスの原理）を無視した誤解だ。 実数では「無限に小さいが 0 でない数（無限小）」は存在しない。

別解：代数的アプローチ

別解（非厳密な代数的議論）

$x = 0.999\cdots$ とおくと $10x = 9.999\cdots = 9 + 0.999\cdots = 9 + x$。よって $9x = 9$，$x = 1$。

この証明は「$0.999\cdots$ という無限小数が実数として定義されており，実数の演算に従う」という前提が必要で，それ自体が非自明。ε-N 論法による証明の方が厳密だ。

4 部分和と収束

問題文

数列 $\{a_n\}_n$ に対し，新たな数列 $\{S_n\}_n$ を $S_n = \sum_{k=1}^{n} a_k$ と定義する。

  1. $\{S_n\}_n$ が収束するとき，$\{a_n\}_n$ は 0 に収束することを示せ。
  2. $\{a_n\}_n$ は 0 に収束するが $\{S_n\}_n$ は収束しないような数列 $\{a_n\}$ の例を挙げよ。
📝 模範解答
証明 (1)

$\{S_n\}$ が $S$ に収束するとする。$n \ge 2$ に対して $a_n = S_n - S_{n-1}$。

$\{S_n\}$ と $\{S_{n-1}\}$ はともに $S$ に収束するから（$\{S_{n-1}\}$ は $\{S_n\}$ のインデックスを 1 ずらした数列なので同じ極限を持つ），問題2の演算法則より

$$\lim_{n\to\infty} a_n = \lim_{n\to\infty} S_n - \lim_{n\to\infty} S_{n-1} = S - S = 0$$
答 (2) — 反例

$a_n = \sqrt{n} - \sqrt{n-1}$ とおく。

$a_n = \dfrac{1}{\sqrt{n} + \sqrt{n-1}} \to 0$（分母 $\to \infty$）。

一方 $S_n = \sqrt{n} \to \infty$（発散）。

🧠 思考法・解説

(1) 部分和の収束 → $a_n \to 0$

💡 $a_n = S_n - S_{n-1}$ という恒等式

$S_n = a_1 + a_2 + \cdots + a_n$ であり $S_{n-1} = a_1 + \cdots + a_{n-1}$ だから， $a_n = S_n - S_{n-1}$。 $\{S_n\}$ が $S$ に収束するなら $\{S_{n-1}\}$ も $S$ に収束する（同じ数列のインデックスをずらしただけ）。 問題2の(1)と(3)（$c = -1$ の場合）より $a_n = S_n + (-1) \cdot S_{n-1} \to S + (-1) \cdot S = 0$。

(2) 対偶：$a_n \not\to 0$ なら級数は発散

(1) の対偶を取ると「$a_n \not\to 0$ ならば $\{S_n\}$ は収束しない」。つまり $a_n \to 0$ は級数の収束の必要条件（十分条件ではない！）。

💡 反例：調和級数 $a_n = 1/n$

$a_n = 1/n$ は $0$ に収束するが，$S_n = \sum_{k=1}^{n} 1/k$（調和級数）は発散する（$S_n \to \infty$）。

発散の証明（グループ分け法）：

$$S_{2^m} = 1 + \frac{1}{2} + \underbrace{\frac{1}{3} + \frac{1}{4}}_{\ge 1/2} + \underbrace{\frac{1}{5}+\cdots+\frac{1}{8}}_{\ge 1/2} + \cdots \ge 1 + \frac{m}{2} \to \infty$$

この発散は有名で重要だが，$a_n \to 0$ だけでは不十分という直感に反した事実を示している。

問題文では $a_n = \sqrt{n} - \sqrt{n-1}$ という例が挙げられている。

$a_n = \sqrt{n} - \sqrt{n-1} = \dfrac{1}{\sqrt{n} + \sqrt{n-1}} \to 0$（分母 $\to \infty$）。

↓

$S_n = \sqrt{n} - \sqrt{0} = \sqrt{n} \to \infty$（発散）。

📌 第1回のまとめ・重要ポイント

ε-N 証明の共通戦略

  • 逆算が本質：$\varepsilon$ を任意に固定し，$|a_n - \alpha| < \varepsilon$ を成立させるための $N$ を逆算で求める
  • 背理法の活用：等式 $a = b$ の証明では，$a \ne b$ の仮定 $\Rightarrow |a-b| > 0$ をεとして使う
  • $\max\{N_1, N_2\}$：複数の条件を同時に成立させるときに使う
  • $\varepsilon/2$ トリック：三角不等式で2項が現れるとき，各項を $\varepsilon/2$ 以下に抑える
  • 有界性：積の極限には「収束列は有界」を先に示す；$\varepsilon=1$ を使って $M = 1+|\alpha|$ を確保
  • 逆数の分母：$a \ne 0$ なら $\varepsilon_0 = |a|/2$ を使って $|a_n| \ge |a|/2$ を確保
💡 ε-N 論法の先にあるもの

ε-N 論法は面倒に感じるかもしれないが，実数の「連続性」（完備性）を数学的に厳密に扱うための唯一の方法だ。 この論法を体得することで， 「連続関数とは何か」「微分可能とは何か」「積分とは何か」という問いにすべて厳密に答えられるようになる。 いわば解析学の共通言語だ。
