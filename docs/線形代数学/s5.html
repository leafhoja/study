§5 $\mathbb{K}^n$ を定義域とする $\mathbb{K}$-線形写像

講義ノート④ をもとに作成

設定 補題5.1 行き先による決定 まとめ — 同値な言い換え
Setup

設定

この節の目標

$n$ を正整数，$V$ を $\mathbb{K}$-ベクトル空間とする。 $T:\mathbb{K}^n\to V$ という $\mathbb{K}$-線形写像の構造を明らかにする。

鍵となる問い：$T:\mathbb{K}^n\to V$ を決めるには何を指定すればよいか？

$e_1,\ldots,e_n$ を $\mathbb{K}^n$ の基本単位ベクトルとする： $$e_i = \begin{pmatrix}0\\\vdots\\0\\1\\0\\\vdots\\0\end{pmatrix} \leftarrow i\text{ 番目}$$ 任意の $x = (x_1,\ldots,x_n)^\top\in\mathbb{K}^n$ は $$x = x_1e_1 + x_2e_2 + \cdots + x_ne_n$$ と一意に表せる。$T$ が線形写像ならば $$T(x) = x_1T(e_1) + x_2T(e_2) + \cdots + x_nT(e_n)$$ となる（線形性 (L1)(L2) の反復適用）。よって $T$ の値は $T(e_1),\ldots,T(e_n)$ だけで完全に決まる。

Lemma 5.1

補題 5.1 — 行き先 $v_1,\ldots,v_n$ による $T$ の決定

補題 5.1 $v_1,\ldots,v_n\in V$ とする。写像 $T_{v_1,\ldots,v_n}:\mathbb{K}^n\to V$ を $$T_{v_1,\ldots,v_n}\!\begin{pmatrix}x_1\\\vdots\\x_n\end{pmatrix} = x_1v_1 + x_2v_2 + \cdots + x_nv_n$$ で定める。すると：
  1. $T_{v_1,\ldots,v_n}$ は $T_{v_1,\ldots,v_n}(e_i)=v_i\ (1\le i\le n)$ をみたす $\mathbb{K}$-線形写像である。
  2. 逆に $T:\mathbb{K}^n\to V$ を任意の $\mathbb{K}$-線形写像とすると， $$T_{T(e_1),\ldots,T(e_n)} = T$$ が成り立つ。
補題 5.1 の意味（2 つのポイント）
  • $T$ は $e_1,\ldots,e_n$ の行き先で決まっている： 2 つの線形写像 $T,T':\mathbb{K}^n\to V$ が $T(e_i)=T'(e_i)$（$1\le i\le n$）を満たせば $T=T'$。
  • $T:\mathbb{K}^n\to V$ を与えることと，$v_1,\ldots,v_n\in V$ を与えることは等価： 対応 $T\mapsto(T(e_1),\ldots,T(e_n))$ は全単射。
補題 5.1 の証明（クリックで展開）
(1) $T_{v_1,\ldots,v_n}$ が線形写像であること：
$x,y\in\mathbb{K}^n$，$a\in\mathbb{K}$ に対して $$T_{v_1,\ldots,v_n}(x+y) = \sum_i(x_i+y_i)v_i = \sum_i x_iv_i + \sum_i y_iv_i = T_{v_1,\ldots,v_n}(x) + T_{v_1,\ldots,v_n}(y)$$ $$T_{v_1,\ldots,v_n}(ax) = \sum_i(ax_i)v_i = a\sum_i x_iv_i = aT_{v_1,\ldots,v_n}(x)$$ (L1)(L2) をみたす。また $T_{v_1,\ldots,v_n}(e_i) = v_i$ は定義より明らか。

(2) $T_{T(e_1),\ldots,T(e_n)} = T$ であること：
任意の $x=(x_1,\ldots,x_n)^\top\in\mathbb{K}^n$ に対して $$T_{T(e_1),\ldots,T(e_n)}(x) = \sum_i x_iT(e_i) = T\!\left(\sum_i x_ie_i\right) = T(x)$$ （$T$ の線形性を使った）。写像として一致するので $T_{T(e_1),\ldots,T(e_n)} = T$。
QED
例 — 行列表示との関係

$V = \mathbb{K}^m$，$v_j = $ $A$ の第 $j$ 列（$A\in M_{m,n}(\mathbb{K})$）とすると， $$T_{v_1,\ldots,v_n}(x) = x_1v_1+\cdots+x_nv_n = Ax$$ これはちょうど行列 $A$ による線形写像 $T_A$ に一致する。

補題 5.1(2) を $V=\mathbb{K}^m$ に適用すると： $$T_{T(e_1),\ldots,T(e_n)} = T \iff \text{$T$ の行列表示} = \bigl(T(e_1)\mid\cdots\mid T(e_n)\bigr)$$ つまり線形写像の行列表示は「基本単位ベクトルの行き先を並べたもの」（系 3.4 の設定と整合）。

例 — $V$ が一般のベクトル空間の場合

$V = M_2(\mathbb{K})$（$2\times 2$ 行列の空間），$n=2$ として $$v_1 = \begin{pmatrix}1&0\\0&0\end{pmatrix}, \quad v_2 = \begin{pmatrix}0&1\\1&0\end{pmatrix}$$ とおくと，$T_{v_1,v_2}:\mathbb{K}^2\to M_2(\mathbb{K})$ は $$T_{v_1,v_2}\!\begin{pmatrix}x_1\\x_2\end{pmatrix} = x_1\begin{pmatrix}1&0\\0&0\end{pmatrix} + x_2\begin{pmatrix}0&1\\1&0\end{pmatrix} = \begin{pmatrix}x_1&x_2\\x_2&0\end{pmatrix}$$ これは $\mathbb{K}^2$ から対称行列の部分空間への線形写像で，$T(e_1)=v_1$，$T(e_2)=v_2$。

Summary

まとめ — $T:\mathbb{K}^n\to V$ に関する同値な言い換え

§5 の核心
言い換え内容
$T:\mathbb{K}^n\to V$ を与える $v_1=T(e_1),\ldots,v_n=T(e_n)\in V$ を与えることと等価
$T$ の決定 $T$ は基本単位ベクトルの行き先 $(T(e_1),\ldots,T(e_n))$ で一意に決まる
$V=\mathbb{K}^m$ のとき 行き先を並べた行列 $A=(v_1\mid\cdots\mid v_n)$ が $T$ の行列表示

直感的な言葉： 線形写像 $T:\mathbb{K}^n\to V$ は「$n$ 個の出力 $T(e_1),\ldots,T(e_n)\in V$ だけ」で完全に記述できる。 他の入力値はすべてこの $n$ 個から一次結合で計算できる。
