📋 常微分方程式（ODE）の型と解法の地図

常微分方程式（Ordinary Differential Equation, ODE）とは，未知関数 $x(t)$ とその導関数の間の方程式だ。解法は方程式の「型」によって異なる。

型1：変数分離形
$x' = f(t) g(x)$ → 両辺を変数ごとに分けて積分
型2：一次線形微分方程式
$x' + p(t)x = q(t)$ → 積分因子 $\mu(t) = e^{\int p(t)\,dt}$ を掛ける
型3：同次形
$x' = f(x/t)$ → $u = x/t$ と置換して変数分離形に帰着
型4：連立線形ODE
$y' = y, z' = 2z$ などに因数分解で分解して解く
💡 一般解・特殊解・初期値問題
  • 一般解：パラメータ $C$（積分定数）を含む解の族。$x' = x$ の一般解は $x = Ce^t$
  • 特殊解：初期条件 $x(t_0) = x_0$ で $C$ を決めたもの
  • 恒等定数解：$x' = f(x)$ で $f(x_0) = 0$ ならば $x \equiv x_0$ は解
1 4種類のODEの解

(1) $x' = tx$（変数分離形）

問題文

$x' = tx$ を解け。

📝 模範解答

$y(t) = x(t)/e^{t^2/2}$ とおく。$y' = (x' \cdot e^{t^2/2} - x \cdot te^{t^2/2})/(e^{t^2/2})^2 = (x' - tx)/e^{t^2/2} = 0$。

よって $y$ は定数。$y(t) = y(0) = x(0)/1 = x(0)$。$C = x(0)$ とおくと

$$\boxed{x(t) = Ce^{t^2/2}} \quad (C \in \mathbb{R})$$
🧠 思考法・解説
🔄 なぜ $y(t) = x(t)/e^{t^2/2}$ とおくのか

方程式 $x' = tx$ を変形すると $x' - tx = 0$。一次線形方程式として積分因子 $\mu = e^{-\int t\,dt} = e^{-t^2/2}$ を掛けると左辺が完全微分になる発想。$y = xe^{-t^2/2}$ とおくと $y' = (x' - tx)e^{-t^2/2} = 0$ → $y = $ 定数。よって $x = Ce^{t^2/2}$。

変数分離でも解ける：$x \ne 0$ のとき $\dfrac{dx}{x} = t\,dt$，積分して $\ln|x| = \dfrac{t^2}{2} + C_0$，よって $|x| = e^{C_0}e^{t^2/2}$，$C = \pm e^{C_0}$ とおけば $x = Ce^{t^2/2}$。$x = 0$ は $C = 0$ に対応。

(2) ロジスティック方程式 $\dot{x} = x(1-x)$

問題文

$\dot{x} = x(1-x)$ を解け。

📝 模範解答

恒等定数解 $x \equiv 0$，$x \equiv 1$ を確認（代入すると両辺が 0）。$x \ne 0, 1$ のとき変数分離：

$$\frac{dx}{x(1-x)} = dt$$

部分分数分解：$\dfrac{1}{x(1-x)} = \dfrac{1}{x} + \dfrac{1}{1-x}$。積分：

$$\ln|x| - \ln|1-x| = t + C_0 \Rightarrow \ln\left|\frac{x}{1-x}\right| = t + C_0$$

指数化：$\dfrac{x}{1-x} = De^t$（$D = \pm e^{C_0}$）。$x$ について解くと $x = \dfrac{De^t}{1+De^t}$。$D = 0$ は $x \equiv 0$ に対応。

$$\boxed{x = \frac{De^t}{1 + De^t}} \quad (D \in \mathbb{R}), \quad \text{または} \quad x \equiv 1$$

$D$ が負のとき定義域は $t \ne \log(-1/D)$ に注意。

🧠 思考法・解説

自律系 $\dot{x} = F(x)$ ではまず平衡点（$F(x) = 0$ の根）を確認する。$F(x) = x(1-x)$ の根は $x = 0, 1$。これらが恒等定数解。

次に $x \ne 0, 1$ の範囲で変数分離。右辺が $x$ だけの関数・左辺が $t$ だけの関数に分けられる形 $x' = F(x)$ は変数分離形。

💡 ロジスティック方程式の背景

この方程式は人口動態や生態学で「環境収容力 $K = 1$ に近づく個体数変動」として現れる。$x = 0$ と $x = 1$ が平衡点で，$0 < x < 1$ では個体数が増加，$x > 1$ では減少する。

(3) 同次形 $t\tan(x/t) - x + tx' = 0$

問題文

$t\tan(x/t) - x + tx' = 0$ を解け。

📝 模範解答

$u = x/t$ とおくと $x = ut$，$x' = u + tu'$。方程式を整理して $x' = (x - t\tan(x/t))/t = x/t - \tan(x/t)$ に代入：

$$u + tu' = u - \tan u \Rightarrow tu' = -\tan u$$

変数分離：

$$\frac{du}{\tan u} = -\frac{dt}{t}$$

$\dfrac{du}{\tan u} = \dfrac{\cos u}{\sin u}\,du = d(\ln|\sin u|)$ を積分して

$$\ln|\sin u| = -\ln|t| + C \Rightarrow \ln|t \sin u| = C \Rightarrow t\sin u = D$$

$u = x/t$ を戻すと

$$\boxed{t\sin(x/t) = D} \quad (D \in \mathbb{R})$$
🧠 思考法・解説
🔄 同次形の見抜き方と置換の理由

$x' = (x - t\tan(x/t))/t = x/t - \tan(x/t)$。右辺は $x/t$ だけの関数（$x$ と $t$ が比の形にしか現れない）→ 同次形。$u = x/t$ と置換すると $x = ut$，$x' = u + tu'$。代入後に $t$ と $u$ が分離できる変数分離形になる。

同次形の判定：方程式を $x' = (\text{右辺})$ の形に整理したとき，右辺が $x/t$ のみの関数で書けるかを確認する。この問題では $\tan$ が含まれているが，引数が $x/t$ のみなので同次形。

(4) 積分因子法 $x' + x\cos t = \sin t \cos t$

問題文

$x' + x\cos t = \sin t \cos t$ を解け。

📝 模範解答

一次線形 $x' + p(t)x = q(t)$，$p(t) = \cos t$，$q(t) = \sin t\cos t$。積分因子 $\mu(t) = e^{\int \cos t\,dt} = e^{\sin t}$ を両辺に掛ける：

$$\frac{d}{dt}(xe^{\sin t}) = e^{\sin t}\sin t\cos t$$

右辺を積分：$u = \sin t$ と置換すると $\int e^u u\,du = (u-1)e^u + C$（部分積分）。よって

$$xe^{\sin t} = (\sin t - 1)e^{\sin t} + C$$ $$\boxed{x = \sin t - 1 + Ce^{-\sin t}} \quad (C \in \mathbb{R})$$
🧠 思考法・解説
💡 積分因子のアイデア

一次線形 $x' + p(t)x = q(t)$ の形。$p(t) = \cos t$ に対して積分因子 $\mu(t) = e^{\int \cos t\,dt} = e^{\sin t}$ を掛ける。なぜ？ 左辺が $\dfrac{d}{dt}(\mu x)$ の形になるから（積の微分の逆）：

$$\frac{d}{dt}(xe^{\sin t}) = x'e^{\sin t} + xe^{\sin t}\cos t = e^{\sin t}(x' + x\cos t) = e^{\sin t}\sin t\cos t$$

積分因子の公式：$\mu(t) = e^{\int p(t)\,dt}$。掛けると左辺は必ず $(\mu x)'$ に変換される。右辺の積分で部分積分が必要な場合は $\int ue^u\,du = (u-1)e^u + C$ を活用する。

2 平衡点と変数変換（自律型ODE）

問題文

Riccati 型の微分方程式

$$x' = x^2 - x - 2$$

を考える。

  1. この微分方程式の解を一つ見つけよ。
  2. (1) で求めた解を $x_0$ とおく。$x = x_0 + u$ と変換することにより，上の微分方程式を $u$ に関する微分方程式に変換せよ。
  3. 一般解 $x$ を求めよ。
📝 模範解答

(1) 平衡点の発見：$x^2 - x - 2 = (x-2)(x+1) = 0$ より平衡点は $x = 2$ と $x = -1$。定数関数 $x \equiv 2$，$x \equiv -1$ は解。

(2) 変数変換：$x = 2 + u$ とおくと $x' = u'$。方程式を $u$ で書き直す：

$$u' = (2+u)^2 - (2+u) - 2 = 4 + 4u + u^2 - 2 - u - 2 = 3u + u^2 = u(u+3)$$

(3) 変数分離で $u' = u(u+3)$ を解く：部分分数 $\dfrac{1}{u(u+3)} = \dfrac{1}{3} \cdot \dfrac{1}{u} - \dfrac{1}{3} \cdot \dfrac{1}{u+3}$。積分：

$$\frac{1}{3}\ln\left|\frac{u}{u+3}\right| = t + C \Rightarrow \frac{u}{u+3} = Ce^{3t}$$

$u$ について解いて $u = \dfrac{3Ce^{3t}}{1 - Ce^{3t}}$，$x = 2 + u$ を戻すと

$$\boxed{x(t) = 2 + \frac{3Ce^{3t}}{1 - Ce^{3t}}} \quad (C \in \mathbb{R})$$

$x = -1$ も解（$u = -3$ は恒等定数解 $u \equiv -3$）。

🧠 思考法・解説

自律系 $x' = F(x)$ ではまず因数分解して平衡点を求める。$F(x) = (x-2)(x+1)$ の根 $x = 2, -1$ が平衡点。

なぜ変数変換 $x = x_0 + u$（$x_0$ は平衡点）を行うか？ 平衡点付近で $u$ が小さいとき，$u' \approx F'(x_0) u$（線形化）で局所的な振る舞いが分かる。さらに非線形項を含めて厳密に解くことも，変換後の方程式が整理されて解きやすくなるため。

今回は $u' = u(u+3)$ という変数分離形に帰着し，部分分数展開 $\dfrac{1}{u(u+3)} = \dfrac{1}{3u} - \dfrac{1}{3(u+3)}$ を用いて積分した。

3 二階線形ODE の因数分解による解法

問題文

線形微分方程式

$$x'' - 3x' + 2x = 0$$

を考える。

  1. この微分方程式の解 $x$ は，$y' - y = 0$，$z' - 2z = 0$ を満たす関数 $y$，$z$ を用いて $x = y + z$ と表されることを示せ。
  2. 一般解 $x$ を求めよ。
📝 模範解答

$y := x' - x$ とおく。$y' - 2y = (x'' - x') - 2(x' - x) = x'' - 3x' + 2x = 0$（問題の方程式）。

$y' = 2y$ の一般解：$y = Ce^{2t}$（$C$ は定数）。

$x' - x = Ce^{2t}$（一次線形）。積分因子 $e^{-t}$：$(xe^{-t})' = Ce^t$，積分して $xe^{-t} = Ce^t + D$，よって

$$\boxed{x(t) = Ce^{2t} + De^t} \quad (C, D \in \mathbb{R})$$
🧠 思考法・解説
💡 因数分解のアイデア

微分演算子 $D = d/dt$ を使うと方程式は $(D^2 - 3D + 2)x = 0$。多項式と同じように因数分解：$D^2 - 3D + 2 = (D-1)(D-2)$（特性方程式 $\lambda^2 - 3\lambda + 2 = 0$ の根は 1 と 2）。

$(D-2)(D-1)x = 0$ と書き換え，$y := (D-1)x = x' - x$ とおく。

🔄 なぜ $y = x' - x$ と定義するのか

$(D-2)(D-1)x = 0$ は「まず $(D-1)$ を $x$ に作用させ，次に $(D-2)$ を作用させると $0$ になる」という構造。

だから $y := (D-1)x = x' - x$ と定義すると，方程式は $(D-2)y = 0$，すなわち $y' - 2y = 0$ という一階線形方程式に帰着する。これなら積分因子ですぐ解ける。

手順：

  1. $y' - 2y = 0$ を解く → $y = Ce^{2t}$（$C$ は定数）
  2. $x' - x = Ce^{2t}$ を解く（一次線形，積分因子 $e^{-t}$）

答案では「$y := x' - x$ とおく」という宣言から始まるが，その着想は「二階演算子 $(D-2)(D-1)$ を1つずつ処理する」という因数分解の論理から来ている。

一般解の構造：$y' - y = 0$ の一般解 $Ce^t$ と $z' - 2z = 0$ の一般解 $De^{2t}$ の和 $x = Ce^t + De^{2t}$ と一致する。

💡 特性方程式との対応

二階線形定係数 $ax'' + bx' + cx = 0$ は特性方程式 $a\lambda^2 + b\lambda + c = 0$ の根 $\lambda_1, \lambda_2$ を用いて $x = C_1 e^{\lambda_1 t} + C_2 e^{\lambda_2 t}$（$\lambda_1 \ne \lambda_2$ の場合）。今回は $\lambda^2 - 3\lambda + 2 = (\lambda-1)(\lambda-2) = 0$ より $\lambda = 1, 2$。

📌 第4回のまとめ：ODEの解法選択

ODEの解法は方程式の構造を見て決まる。

  • 変数分離：$x' = f(t)g(x)$ の形。両辺を $g(x)$ で割り，$t$ と $x$ を分けて積分
  • 積分因子：$x' + p(t)x = q(t)$ の形。$e^{\int p\,dt}$ を掛けると左辺が完全微分
  • 同次形：$x' = f(x/t)$ の形。$u = x/t$ で変数分離形に
  • 平衡点変換：自律系 $x' = F(x)$ で $F(x_0) = 0$ なら $u = x - x_0$ で簡単化
  • 因数分解：定係数線形 → 特性方程式の根が指数関数解
⚠ 積分定数の取り扱い

$\pm e^C$ などの任意定数は「$D$（任意実数）」と書き直す。恒等定数解（$x \equiv $ 定数）が別途存在することも確認する。
